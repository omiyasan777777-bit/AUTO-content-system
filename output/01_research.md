## 2. 売れている要素の逆引き

共通して使われている要素（市場が求めているもの）
1. 「顔出しなし」というキーワードは全競合が使用 → 鉄板の訴求
2. AIツール活用（AIで自動化・効率化）という方向性
3. 具体的な収益数字（月10万・月15万）の提示
4. 初心者OK・完全未経験という間口の広さ
5. 「最短」「最速」という時間短縮訴求

差別化に使われている要素
1. 声出しなし（Nox） ← 顔だけでなく声も不要という追加の安心感
2. ロードマップ形式（長谷川） ← 全体像が見えて不安が下がる
3. 具体的なツール数（3選）← 「何を買えばいいか」がシンプル

価格と内容の相関
- 500〜980円台: 入門・概要紹介型（スモールスタート）
- 980〜1,980円台: ロードマップ型・手法詳細解説
- 1,980〜3,000円円台: 実績訴求+完全手順型

タイトルパターン
- パターンA: 【年号最新】+ 顔出しなし + AIツール + 収益化期間
- パターンB: 実績数字 + AIツール数 + 顔出しなし + 最速・最短
- パターンC: AI×YouTube + 完全攻略 + 月○万円 + 自動化

無料部分の構成パターン

- パターンB: ノウハウ一部公開（ツール紹介）→「詳細は有料で」

## 6. 差別化ポイント

### 差別化1: 11Labsリスク開示
- 競合の現状: 11Labsを平然と推奨している記事が多い（リスク説明なし）
- 自分ならこう差別化: 「使ってはいけない理由」として章を設け、収益化停止リスクの実態を正直に解説
- 読者にとっての価値: 失敗を未然に防ぐ → 信頼感・安心感が一気に上がる
- これを使ったタイトル案: 「なぜ99%の解説が教えない『11Labs禁止』の理由があるのか」

### 差別化2: 月収61万円チャンネルの具体的再現
- 競合の現状: 「AIで動画を作れる」という概念説明どまりで、具体的な参照チャンネルの分析がない
- 自分ならこう差別化: 登録者13万人・動画12本・再生700万回という実存する成功事例を解剖し、同じものを20分で再現する手順を提供
- 読者にとっての価値: 「これを真似すれば再現できる」という具体的な設計図
- これを使ったタイトル案: 「月収61万円チャンネルの正体を解剖したら、20分で再現できた」

### 差別化3: Claude Code + 3ツールスタックの完全詳細
- 競合の現状: 「AIツール3選」と言いながら設定方法・連携手順が薄い
- 自分ならこう差別化: Claude Code・Hixfield・Turboscribeの3ツールを連携させる具体的なセットアップ手順とコマンドを公開
- 読者にとっての価値: 読んだ当日に動画1本を完成できる再現性
