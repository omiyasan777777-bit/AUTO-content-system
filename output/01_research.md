## 2. 売れている要素の逆引き

共通して使われている要素（市場が求めているもの）
1. 「在庫なし・自動配信」のデジタル商品メリット訴求
2. 「寝ている間に稼げる」不労所得イメージ
3. 初心者向けのステップバイステップ構成
4. 実際の売上・件数などの数字・実績
5. 「完全ガイド」「保存版」などの網羅性・信頼ワード

差別化に使われている要素（独自性のパターン）
1. 属性の差別化（50代・学生など）
2. 成功ジャンルの具体化（「8つのジャンル」など数字）
3. AI×Etsy の掛け合わせ訴求

価格と内容の相関
- 500〜980円台: 入門・概念解説系
- 980〜1,980円台: 手順書・ガイド系（最も多い）
- 1,980円〜: 実績付き・ノウハウ特化型

タイトルパターン
- パターンA: [数字] + [ジャンル] + [結果]（例: 売れる8つのジャンル）
- パターンB: [ターゲット属性] + [手段] + [仕組み]（例: 日本でEtsyを副業として）
- パターンC: [キャッチ疑問] + [リアル] + [成功例]

無料部分の構成パターン
- パターンA: 疑念先回り → 実例 → ジャンル紹介（競合2）
- パターンB: メリット説明 → 体験談 → 手順予告（競合1）
- パターンC: プラットフォーム紹介 → 概要 → 詳細は有料（競合3）

## 3. 市場の傾向

- 価格帯の相場: 500〜1,980円
- 最も売れている価格帯: 980〜1,480円
- よく使われる訴求: 不労所得・在庫なし・自動配信・英語不要・初心者OK
- 読者が求めていること TOP5:
  1. 「何を作ればいいかわからない」への具体的答え
  2. 英語ができなくても本当に運営できるのかの証明
  3. ショップ開設〜初売上までの具体的ステップ
  4. 稼げるジャンル・商品カテゴリのリスト
  5. 手数料・収益計算・確定申告などのリアルな情報
- 競合が扱っていない空白地帯: 「英語テンプレ完備+リサーチ手順+商品量産フロー」を一冊に体系化したコンテンツが少ない

## 6. 差別化ポイント

### 差別化1: 「英語ゼロ完全対応」のテンプレ・手順書付き
- 競合の現状: 「英語不要」と謳っていても、具体的な英語テンプレが付いていない
- 自分ならこう差別化: タイトル・タグ・商品説明・レビュー返信まですべてコピペ英語テンプレを同梱
- 読者にとっての価値: 「英語の壁」が完全に消え、当日から出品できる
- タイトル案: 「英語ゼロでも今日から出品できるEtsy完全マニュアル」

### 差別化2: 「リサーチファースト」の逆算型メソッド
- 競合の現状: 「こんな商品が売れる」と紹介するだけで、自分で調べる手順がない
- 自分ならこう差別化: eRankを使った売れ筋リサーチの手順を図解・ステップで完全解説
- 読者にとっての価値: 自分で継続的に「次に何を作るか」判断できるようになる
- タイトル案: 「売れているものを先に調べる。Etsy逆算式デジタル副業メソッド」

### 差別化3: 「ハンドメイド不要・デジタルデータ特化」の絞り込み
- 競合の現状: ハンドメイドとデジタルが混在しており、初心者が迷う
- 自分ならこう差別化: デジタルデータ販売のみに完全特化。CanvaだけでOKという入門ハードルの低さを全面に
- 読者にとっての価値: 「自分にも作れる」という確信が持てる
- タイトル案: 「Canvaだけ。ハンドメイド不要のEtsyデジタル副業ガイド」

## 7. コンセプト設計への引き継ぎメモ

- 推奨コンセプトの方向性: 「英語ゼロ・ハンドメイド不要・リサーチからCanva制作まで完全フロー」の一冊完結型
- ターゲットの絞り方の提案: 「英語コンプレックスがある・副業初心者・30〜40代会社員or主婦」に絞る
- 避けるべきアプローチ: 「なんとなく出品すれば稼げる」的な夢物語。リアルな手順・手数料計算まで誠実に書く
- 価格帯の提案: ローンチ980円 → 通常1,480円 → 最終1,980円（相場上位帯で価値を示す）
- コンテンツマージ元の候補3商品:
  1. 競合1（さと｜50代）← 「寝ている間に稼ぐ」感情訴求・不労所得の世界観
  2. 競合2（CTEE JP）← 売れるジャンル特定・成功事例の具体性
  3. 競合3（理系大学生）← 初心者向けの丁寧なステップ構成・網羅性
