# 競合リサーチレポート

生成日: 2026-06-22
テーマ: 彼氏との関係に悩む女性がChatGPTにするベストな質問方法
ターゲット: 彼氏と上手くいかない20代前半の女性・感情的になりがちな女性
モード: クイック

## 1. 競合分析

### 競合1: 「ChatGPTで恋愛相談をするときに使ってほしい心が軽くなる「おすすめプロンプト」」
- 販売者: そら｜恋愛で疲れやすい人の心の相談室
- 価格: 有料（推定300〜500円）
- 推定販売規模: スキ数・コメント数から中程度の反応あり
- メインの訴求: ChatGPTに恋愛相談するときのプロンプトを提供し「心が軽くなる」体験を約束
- ターゲット像: 恋愛に疲れやすい・誰かに相談したい女性全般
- タイトルの構造: [手段（ChatGPT）] + [用途（恋愛相談）] + [感情的ベネフィット（心が軽くなる）]
- 無料部分の構成: 「恋愛相談をAIにする需要」を認める共感から入り、プロンプトの一部を見せて有料へ誘導
- 強み: 感情的なコピー・「心が軽くなる」というベネフィットが刺さりやすい
- 弱み: ターゲットが広すぎる（彼氏特化でない）・感情的になりやすい女性への特化がない

### 競合2: 「💘ChatGPTで恋愛相談役を作るテンプレ3選」
- 販売者: きたよん
- 価格: 無料〜有料（一部有料）
- 推定販売規模: スキ数多め（人気コンテンツ）
- メインの訴求: ChatGPTを恋愛相談役として設定するテンプレを3種類提供
- ターゲット像: ChatGPTを使い始めた恋愛に悩む人全般（性別不問）
- タイトルの構造: [絵文字で親しみ] + [手段] + [内容（テンプレ3選）]
- 無料部分の構成: テンプレの概要紹介 → 1つ目を無料公開 → 残り2つは有料
- 強み: 具体的なテンプレが即使えるという実用性・絵文字タイトルで親しみやすい
- 弱み: 女性向け・彼氏との関係特化ではない・感情整理のプロセスがない

### 競合3: 「ChatGPTに恋愛相談する いい点と悪い点｜AI恋愛カウンセリングの真実と"心の副作用"」
- 販売者: あなたのトリセツ@カウンセラーのナミ
- 価格: 有料（推定500〜1,000円）
- 推定販売規模: カウンセラー肩書きで信頼性高め
- メインの訴求: AIへの恋愛相談のメリット・デメリットを専門家目線で解説
- ターゲット像: AIに相談することへの不安・疑問がある女性
- タイトルの構造: [手段] + [バランス訴求（いい点と悪い点）] + [専門的サブタイトル]
- 無料部分の構成: 「AIに相談していいの？」という読者の疑問に共感 → 専門家として信頼性を提示
- 強み: カウンセラー資格・専門性がある・副作用という切り口が独自
- 弱み: How-toではなく評論系・実践的な質問方法が学べない・彼氏との喧嘩特化でない

## 2. 売れている要素の逆引き

### 共通して使われている要素（=市場が求めているもの）
1. ChatGPTへの「具体的な質問の型・プロンプト」を求めている
2. 「恋愛相談の場所がない・誰にも言えない」という孤独感への共感
3. AIへの相談に対する罪悪感や不安の解消
4. 「すぐ使えるテンプレ」という実用性
5. 感情的に疲れている人へのやさしいトーン

### 差別化に使われている要素（=独自性のパターン）
1. 専門家資格（カウンセラー・心理士）の信頼性
2. 絵文字・親しみやすいビジュアルでの親近感
3. 「テンプレ何選」という具体的な数字訴求

### 価格と内容の相関
- 無料〜300円: プロンプト1〜2個公開、体験談記事
- 300〜800円: プロンプトテンプレ3〜10個セット
- 800〜1,500円: 専門家監修・体系的メソッド・複数章構成

### タイトルパターン
- パターンA: [手段（ChatGPT）] + [用途（恋愛相談）] + [数字（○選）]
- パターンB: [ターゲット感情（心が軽くなる）] + [手段] + [内容]
- パターンC: [専門性（カウンセラーが教える）] + [テーマ] + [ベネフィット]

### 無料部分の構成パターン
- パターンA: 悩み共感 → 「AIに相談していいの？」の疑問解消 → テンプレ一部公開 → 購入促進
- パターンB: テンプレ紹介 → 1つ無料公開 → 「続きは有料で」
- パターンC: 著者の専門性提示 → 問題定義 → 解決策の示唆 → オファー

## 3. 市場の傾向

- 価格帯の相場: 無料〜1,500円
- 最も売れている価格帯: 300〜800円
- よく使われる訴求:
  - 「すぐ使えるプロンプト・テンプレ」
  - 「誰にも言えない悩みをAIに相談」
  - 「心が軽くなる」「気持ちが楽になる」
- 読者が求めていること TOP3:
  1. 具体的なChatGPTへの質問の型が知りたい
  2. 感情を整理する方法が知りたい
  3. 彼氏とのすれ違いを解決するヒントが欲しい
- 競合が扱っていない空白地帯:
  - **「彼氏との関係」に特化したChatGPT活用法**（既存は恋愛全般・男女不問）
  - **感情的になりやすい女性専用**の質問フレーム
  - 感情を整理してからChatGPTに聞くという「順番」の教え方

## 6. 差別化ポイント

### 差別化1: 「彼氏との関係」に完全特化
- 競合の現状: 恋愛全般・男女不問・相談相手としてのAI全般
- 自分ならこう差別化: 彼氏がいる女性の「喧嘩・すれ違い・伝え方」に絞る
- 読者にとっての価値: 「自分のための」コンテンツと感じられる
- タイトル案: 「彼氏との関係に悩む女性のためのChatGPT活用術」

### 差別化2: 「感情的になりやすい女性専用」のフレーム
- 競合の現状: 感情整理のプロセスを教えるコンテンツがない
- 自分ならこう差別化: 感情を整理してから質問する「感情整理ファースト」メソッドを体系化
- 読者にとっての価値: 「私のための方法だ」と感じる特化感
- タイトル案: 「感情的な私でも使える！ChatGPT質問の型」

### 差別化3: 「質問の失敗例→改善」の逆転ストーリー構造
- 競合の現状: 「正しい使い方」を教えるだけ。失敗談がない
- 自分ならこう差別化: 「感情のまま質問したら的外れだった実体験」から始まる共感ストーリー
- 読者にとっての価値: 「自分だけじゃなかった」という安心感
- タイトル案: 「ChatGPTに相談しても的外れだった私が、質問を変えたら彼氏との関係が変わった話」

## 7. コンセプト設計への引き継ぎメモ

- 推奨コンセプトの方向性: **「感情的になりやすい女性 × 彼氏との関係改善 × ChatGPT質問の型」**という三重の特化で空白地帯を狙う
- ターゲットの絞り方の提案: 20代前半・彼氏持ち・感情的になりがち・ChatGPTを使ったことがある（または興味あり）
- 避けるべきアプローチ: AI彼氏ロールプレイ系（競合多数）・男女不問の汎用相談系（差別化できない）
- 価格帯の提案: ローンチ価格 **980円**→通常 **1,480円**→最終 **1,980円**（体系的メソッド＋プロンプトテンプレ付きで正当化）
- コンテンツマージ元の候補3商品:
  1. 「ChatGPTで恋愛相談をするときに使ってほしいプロンプト」← プロンプトテンプレの具体性
  2. 「ChatGPTで恋愛相談役を作るテンプレ3選」← 実用的なテンプレ形式・読みやすい構成
  3. 「ChatGPTに恋愛相談する いい点と悪い点」← 専門性・副作用まで踏み込む誠実さ
