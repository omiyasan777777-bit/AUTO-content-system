**想定価値：2,000円**

---

## はじめに

別居婚で最も揉めるのは**お金の問題**です。

「どちらがいくら出すか」「交通費は誰が払うか」「将来の貯蓄はどうするか」——
これを曖昧にしたまま始めると、必ずどちらかが不満を溜め込みます。

このワークシートに二人で数字を入れるだけで、
**1ヶ月の別居婚コスト** と **年間の負担バランス** が可視化できます。

**判定チェック:**
```
□ どちらも月間余剰がプラスになっている → 経済的に持続可能

□ 余剰が少なすぎて貯蓄できない → コスト削減か収入増の検討が必要
```

## ■ お金の話し合いを円滑にするコツ

1. **数字を先に出す、感情は後で** → 「高い」「安い」の前に、まず現実の数字を揃える
2. **「損した」より「どうすれば両方が納得できるか」で話す** → 比較より設計の視点で
3. **年に1回は必ず見直す** → 収入・支出・ライフステージが変わればルールも変える
4. **お金の話ができる関係が、長続きする関係** → タブーにせず、フラットに話せる習慣をつくる
