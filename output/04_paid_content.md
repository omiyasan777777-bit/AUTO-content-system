# 第1章：はじめに〜世界に1冊、わが子への贈り物

「絵が描けないから、手作り絵本は無理だよな…」

そう思ったこと、ありませんか？

子どもの誕生日が近づくたびに、
クリスマスの前になるたびに、
「世界に1冊だけの、わが子のための絵本を作ってあげたい」
と思いながら、あきらめてしまっていた——。

このnoteを手に取ってくださったあなたも、きっと同じ気持ちを持っているはずです。

でも、もし「絵が描けなくても、AIを使えば手書き風の絵本が作れる」
しかも「英語が一切わからなくても、日本語をコピペするだけで完成する」
としたら、どうでしょうか？

そんな方法が、今の時代には実際に存在します。

---

## この教材が生まれた理由

私には子どもがいます。

子ども向けのコンテンツを作る仕事もしてきました。
「絵本ってどう作るんだろう」という知識は、ある程度あったんです。

でも、絵は描けない。

絵本作家さんが描いてくれるような、ほのぼのとした手書き風のイラスト。
あれを自分で再現する力は、私にはありませんでした。

だから最初は「絵本を自分で作るなんて無理だな」と思っていたんです。

そんな私が変わったきっかけは、AIでした。

ある日、「AIが画像を生成できる」というニュースを目にして、
試しにChatGPT（DALL-E3）に「手書き風の子ども向けイラストを作って」と入力してみたんです。

最初は全然ダメでした。
なんかリアルすぎる絵が出てきたり、
「手書き風」のはずがプロのイラストみたいになったり。
毎回キャラクターの顔が変わって、別人みたいになったり。

それでも諦めなかった理由は、
**「あの子が目を輝かせる顔が見たい」** という想いだけでした。

試行錯誤50回以上。
やっと「これだ！」と思える手書き風の絵が出てきた瞬間、
声が出るほど嬉しかったのを今でも覚えています。

**STEP 1: AI絵本作成の準備をする**

## 2-1: 迷わず選べるAIツール選定ガイド

### なぜDALL-E3（ChatGPT）なのか

AI画像生成ツールを選ぶとき、以下の3つの基準で比較してみましょう。

| ツール | 日本語対応 | 難易度 | 月額費用 | 手書き風の得意度 |
|--------|-----------|--------|---------|----------------|
| **DALL-E3（ChatGPT）** | ✅ 日本語OK | 初心者向け | 無料〜2,700円 | ◎ |
| Midjourney | ❌ 英語必須 | 中〜上級者 | 約1,500円〜 | ◎ |
| Stable Diffusion | △ 設定が必要 | 上級者 | 無料（自己インストール） | ◎ |
| Adobe Firefly | ✅ 日本語OK | 初心者向け | 無料〜 | △ |
| Canva AI | ✅ 日本語OK | 初心者向け | 無料〜1,500円 | △ |

**DALL-E3を選ぶ3つの理由:**

① **日本語でそのまま指示できる**
ChatGPTに日本語で「〜のような絵を描いて」と書くだけで生成できます。
英語でプロンプトを書く必要がありません。

② **会話形式で修正できる**
「もっと手書き感を強くして」「キャラを少し笑顔にして」と日本語で指示するだけで、
生成結果を対話しながら調整できます。

③ **無料プランでも始められる**
ChatGPTの無料プランでもDALL-E3は利用できます（1日の利用数に制限あり）。
本格的に使うならPlus（月2,700円）がおすすめですが、まずは無料で試せます。

✅ **POINT**: MidjourneyやStable Diffusionの方が高品質という意見もありますが、英語が苦手な初心者には圧倒的にハードルが高い。「完璧な絵」より「子どもに喜んでもらえる絵」を作ることが目標なので、まずは使いやすさ優先でOKです。

「画像を1枚生成してください。青い空のイラストです。」

絵が生成されたら準備完了です。

**3. 設定のポイント（1点だけ）**

エラーメッセージや説明も日本語で表示されて安心です。

## 2-2: 英語ゼロ！日本語プロンプトで始める方法

「え、本当に日本語だけで大丈夫なの？」

大丈夫です。DALL-E3は日本語の指示も理解します。
ただし、より意図通りの絵を生成するためには「書き方のコツ」があります。

### 日本語プロンプトの基本構造

以下の4要素を含めると、格段に良い絵が生成されます。

```
[スタイル指定] + [主人公・キャラクター] + [場面・背景] + [全体の雰囲気]
```

**具体例:**
```
手書き風の水彩画スタイルで、
丸い目の小さな女の子が公園で遊んでいる場面を描いてください。
柔らかいパステルカラーを使い、クレヨンで描いたような温かい雰囲気にしてください。
白い余白のある絵本イラストです。
```

これだけで、手書き風の絵本イラストが生成されます。

### 日本語プロンプトの「効くワード」リスト

以下のワードを組み合わせると、手書き風の絵本らしさが増します。

**スタイル系:**
- 「手書き風の水彩画スタイルで」
- 「クレヨンで描いたような」
- 「絵本のイラスト風に」
- 「子どもが描いたような素朴な感じで」
- 「鉛筆と水彩を組み合わせたような」

**キャラクター系:**
- 「丸い目の」「くりくりした目の」
- 「頭が少し大きめの」（デフォルメ感が出る）
- 「ほっぺが赤い」「笑顔の」

**雰囲気系:**
- 「柔らかいパステルカラーで」
- 「温かみのある」「ほのぼのした」
- 「白い余白のある」（絵本ページらしくなる）
- 「シンプルで可愛らしい」

✅ **POINT**: 「リアルな写真風」「3DCGっぽい」にならないようにするのがポイントです。明示的に「イラスト風」「絵本スタイル」「水彩」などのキーワードを入れましょう。

## 2-3: 0〜8歳・年齢別「刺さる絵本スタイル」選び方

子どもの年齢によって「好きな絵のスタイル」は変わります。
お子さんの年齢に合わせて、プロンプトのスタイルを選びましょう。

### 年齢別スタイルガイド

**0〜2歳向け: ビビッドカラー・シンプル形状**
- 色: 赤・青・黄のはっきりした色（パステルより鮮やか）
- キャラクター: 大きな目・丸い顔・単純な形
- 背景: 白背景または単色背景（ごちゃつかない）
- プロンプトキーワード: 「明るい色使いで」「大きくシンプルな形の」「赤ちゃん向け絵本スタイルの」

**3〜5歳向け: 温かいパステル・キャラクター中心**
- 色: パステルカラー（ピンク・薄いブルー・黄緑）
- キャラクター: 表情豊か・動きのあるポーズ
- 背景: 簡単な背景（公園・家の中など）
- プロンプトキーワード: 「パステルカラーの水彩画スタイルで」「表情豊かな」「絵本のような温かい雰囲気の」

**6〜8歳向け: ディテールあり・ストーリー感**
- 色: 少し落ち着いた色（深みのある色使いもOK）
- キャラクター: 少し細かい描写OK（服のデザインや小物）
- 背景: しっかりした背景（森・街・海など）
- プロンプトキーワード: 「手書き風の水彩イラストで」「ディテールのある」「物語性のある絵本スタイルで」

## 3-1: 「手書き風」を完璧に再現する黄金プロンプト

### 手書き風を生み出す5つのコア要素

手書き風の絵本イラストを生成するために必要な要素は、次の5つです。

```
① テクスチャ指定（どんな画材で描いたか）
② 線の質感（線の荒さ・ぶれ感）
③ 色の質感（絵の具・クレヨンの滲みや重なり）
④ 背景・余白の設計（紙の質感）
⑤ デフォルメ度（キャラクターの親しみやすさ）
```

この5つを組み合わせたのが、以下の「黄金プロンプト」です。

---

### 🎨 手書き風黄金プロンプト（コピペOK）

**【基本版：温かい水彩×クレヨン混合スタイル】**
```
子どもが実際にクレヨンと水彩で描いたような絵本イラストです。
線は少し不均一でぶれており、温かみがあります。
水彩絵の具の滲みと重なりがあり、柔らかいパステルカラーを使っています。
キャラクターは頭が大きめのデフォルメ体型で、丸くてかわいらしい顔です。
紙の白いテクスチャが透けて見えており、余白のある構図です。
絵本のひとコマのような構成で、背景はシンプルです。
```

**【応用版：鉛筆×水彩の素朴スタイル】**
```
鉛筆で輪郭を描き、水彩絵の具で色を塗ったような手書き絵本スタイルです。
鉛筆の線がうっすら見えており、素朴で手作り感があります。
色は少しはみ出したり、薄く重なったりしており、人間味のある仕上がりです。
白い画用紙に描いたような質感で、余白が多い絵本らしいレイアウトです。
登場キャラクターは丸くてかわいいデフォルメ調です。
```

**【応用版：クレヨン単色タッチのシンプルスタイル】**
```
子どもがクレヨンだけで描いたような、シンプルで素朴な絵本イラストです。
線はぐるぐると丸みがあり、塗りはクレヨン特有のかすれと重なりがあります。
色数は少なく（3〜4色程度）、明るくはっきりした配色です。
白い背景に大きくキャラクターが描かれており、絵本の挿絵らしい構成です。
```

✅ **POINT**: この3パターンを覚えれば、ほとんどの「手書き風」ニーズに対応できます。最初は「基本版」から試して、気に入らない部分を調整していきましょう。

## 3-2: キャラ崩れゼロ！統一プロンプト設計法

手書き風絵本で最も困るのが「ページをめくるたびにキャラクターの顔が変わる問題」です。

1ページ目は金髪の女の子だったのに、
3ページ目では茶髪になっていた——。

これはAI画像生成の特性上、完全には防げませんが、
「キャラクター固定プロンプト」を使うことで大幅に改善できます。

### キャラクター統一の3ステップ

**STEP 1: キャラクターカードを作る**

最初に「このキャラクターの設定書」をChatGPTとの会話の中で確立します。

以下をコピペして、キャラクターの詳細を埋めて送ってください。

```
これから絵本のキャラクターを設定します。
以下の設定で固定のキャラクターを作成してください。

名前: （例：はなちゃん）
年齢: （例：4歳）
髪の色・スタイル: （例：黒髪のおかっぱ）
目の色: （例：茶色の大きな丸い目）
服の特徴: （例：赤いリボン付きのワンピース）
身長・体型: （例：頭が少し大きめのかわいいデフォルメ体型）
性格・表情の特徴: （例：いつも笑顔で元気いっぱい）

このキャラクターを全ページで統一して描いてください。
```

**STEP 2: キャラクターの「固定ワード」を作る**

キャラクターカードが完成したら、毎回のプロンプトの先頭に「固定ワード」を付けます。

```
【キャラクター固定】
名前：はなちゃん（4歳）、黒髪おかっぱ、茶色の丸い目、
赤いリボン付きワンピース、頭大きめのデフォルメ体型、笑顔

[手書き風黄金プロンプトを続ける]
[場面の描写を追加]
```

**STEP 3: 会話の流れを途切れさせない**

ChatGPTで絵本を作るとき、**1つの会話（チャット）の中で全ページ分を生成**するのが重要です。

新しいチャットを開くとキャラクターの記憶がリセットされるため、
必ず同じ会話の流れの中で「次のページも」と続けていきましょう。

## 3-3: 世界観ブレなし！背景・小物プロンプト集

キャラクターの次に重要なのが「背景の統一感」です。

ページごとに草原、海、宇宙とバラバラになると
「1冊の絵本」としての世界観が壊れてしまいます。

### 世界観を統一する3つのルール

**ルール1: テーマの場所を1〜2つに絞る**

絵本全体を通して登場する「メインの場所」を決めます。

例：
- 森の中の小屋を中心に物語が進む絵本
- 一軒家の各部屋を舞台にした絵本
- 海辺の村を舞台にした絵本

メインの場所を決めたら、毎回のプロンプトに「〜の場所で」と明記します。

**ルール2: カラーパレットを固定する**

「この絵本の色のトーン」をあらかじめ決めておきます。

```
カラーパレットの指定例:
「薄いグリーンとクリーム色を基調としたパステルカラーで、
全体的に温かみのある色調にしてください」
```

このカラーパレット指定を毎回のプロンプトの最後に加えるだけで、
ページをめくっても「同じ世界観」が保たれます。

**ルール3: 象徴的な小物を固定する**

絵本の世界観を代表する「小道具」を1〜2つ設定します。

例：
- どのページにも小さなウサギのぬいぐるみが登場する
- 主人公がいつも同じリュックサックを背負っている
- 家の窓から同じ大きな木が見えている

これがあるだけで、ページをめくるたびに「あ、この世界だ」と子どもが安心します。

## 3-4: 「なんか違う」を解決するプロンプト修正術

頑張ってプロンプトを書いたのに「なんか違う…」となるのは、
手書き風絵本作りでよくあることです。

まずはこのチェックシートで「何が違うのか」を特定しましょう。

### プロンプト修正チェックシート

**Q1: 絵がリアルすぎる（写真みたい）**
→ 解決策: 「写真のような」「リアル」「フォトリアリスティック」という要素が入っていないか確認。
「手書き風」「絵本スタイル」「クレヨン」「水彩」を追加。

**Q2: 線がシャープすぎる（プロのイラストみたい）**
→ 解決策: 「線は少し不均一でぶれており」「素朴な手書き感がある」を追加。
「デジタルイラスト風」という要素を打ち消す言葉を入れる。

**Q3: 色が鮮やかすぎる・ビビッドすぎる**
→ 解決策: 「柔らかいパステルカラーで」「くすんだ色調で」「水彩絵の具の薄い色で」を追加。

**Q4: 背景が複雑すぎる**
→ 解決策: 「背景はシンプルに」「白いまたは単色の背景で」「余白を多めに」を追加。

**Q5: キャラクターが大人っぽすぎる**
→ 解決策: 「デフォルメ体型」「頭が大きめの」「子ども向け絵本のかわいいキャラクター風に」を追加。

✅ **POINT**: 「なんか違う」と感じたら、ChatGPTに「もっと手書き風にして」「子どもが描いたようにして」と日本語でそのまま追加指示できます。生成結果を見ながら対話するのが最短の解決策です。

## 4-2: 日英バイリンガル絵本のテキスト設計法

バイリンガル絵本のテキストを設計するときの基本原則は、
**「シンプルさ」** です。

1ページあたりの文章が多すぎると、子どもが飽きてしまいます。

### バイリンガル絵本の文字数ガイドライン

| 対象年齢 | 日本語テキスト（1ページ） | 英語テキスト（1ページ） |
|---------|----------------------|---------------------|
| 0〜2歳 | 5〜10文字（単語1〜2つ） | 1〜3単語 |
| 3〜5歳 | 15〜30文字（1〜2文） | 1〜2文（短文） |
| 6〜8歳 | 30〜60文字（2〜3文） | 2〜3文 |

### ChatGPTに英語テキストを作ってもらう方法

日本語でストーリーを伝えるだけで、英語のテキストをChatGPTが作ってくれます。

以下のプロンプトをコピペして使ってください。

**【英語テキスト生成プロンプト（コピペOK）】**
```
以下の日本語テキストを、絵本に使える英語に翻訳してください。
対象年齢は（　　）歳です。
シンプルで読みやすい英語にしてください。
難しい単語は使わず、子どもが覚えられる表現にしてください。

【日本語テキスト】
（ここに日本語の文章を貼り付ける）
```

### バイリンガルテキストの配置パターン

**配置パターンA: 上下分割（おすすめ）**
```
[イラスト]

日本語テキスト
（改行）
English text
```

**配置パターンB: 日本語大・英語小**
```
[イラスト]

大きな日本語テキスト
小さな English text
```

**配置パターンC: 左右分割**
```
[イラスト]
日本語テキスト | English text
```

✅ **POINT**: 配置パターンAが最もシンプルで読みやすく、初心者におすすめです。日本語を主体にして、英語はサブテキストとして添える設計にすると、子どもが迷わず読めます。

### よくある質問 Q&A

**Q: 著作権はどうなりますか？**
A: DALL-E3（ChatGPT）で生成した画像は、商業目的以外の個人利用であれば基本的に問題ありません。ただし、販売や公開を目的とする場合はOpenAIの利用規約を必ずご確認ください。家族への贈り物・個人利用の範囲であれば安心して使えます。

**Q: 毎回同じクオリティの絵が出ますか？**
A: AIの特性上、まったく同じ絵は出ません。ただし、同じプロンプトを使えば近いスタイルの絵が生成されます。「少し違う」と感じたら再生成してみてください。

**Q: 有料プランが必要ですか？**
A: 無料プランのChatGPTでも1日数枚の画像生成は可能です。8ページ分を作るだけなら無料で十分できます。頻繁に作る場合はPlus（月2,700円）が快適です。

**Q: スマホだけでも作れますか？**
A: 可能です。ChatGPTのスマホアプリで画像を生成し、Canvaのスマホアプリでテキストを加えることで、パソコンがなくても完成できます。
