## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 30日以内に楽天アフィリエイトの最初の1万円を突破できる
- サブベネフィット1: 商品選びで迷わなくなる（ジャンル・価格帯・料率の3基準）
- サブベネフィット2: 投稿のネタ切れがなくなる（ネタ帳運用）
- サブベネフィット3: 一度当たった投稿を資産として使い回せるようになる
- 感情的ベネフィット: 「センスがないから稼げない」という不安から解放され、自分の力で稼げた自信が持てる

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: 9ルール仕組み化メソッド
- 他商品にない要素: 精神論ではなく「画像・価格帯・料率」という数値基準に落とし込んだ商品選定フレーム
- 自分だけが語れる理由: 実際に30日で1万円を達成した本人の実践記録そのものだから

■ 要素3: 実績・証拠
- 自分の実績: 30日間で楽天アフィリエイト収益1万円達成
- 購入者の実績（購入者やモニターの感想）: モニター募集で先行実践者の声を集め、随時追記予定
- 具体的な数字: 30日、9つのルール、毎日5投稿、1,000〜3,000円・料率10%以上の商品選定基準

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: 楽天のセールシーズンは投稿の伸ばしどきであり、今始めるほど早く「型」が身につく
- 限定要素（部数/期間/特典）: ローンチ価格は先着100名 or 7日間限定
- 見逃した場合の損失: 自己流のまま投稿を続け、成果が出ないまま挫折してしまうリスク
```

### 詳細ステップ

#### 大ステップ1: ジャンル・商品選定基準を身につける

##### 小ステップ1-1: 「画像で惹ける」ジャンルだけを選ぶ（10分）
課題: どのジャンルを投稿すればいいか分からず、闇雲に商品を選んでしまう
解決策: スクロール中に思わず止まってしまう「画像映え」するジャンル（食品・コスメ・ガジェット・インテリア等）に絞る
何をするか: 楽天市場で候補ジャンルをリストアップし、画像の第一印象だけで「止まるかどうか」を判定する
なぜやるか: 楽天アフィリエイトは文章より先に画像で興味を引くことが最優先だから
コツ / 注意点: 自分が好きなジャンルより「画像映えするか」を優先する
成果物: 投稿対象にするジャンルのリスト
使うツール / リソース: 楽天市場ランキングページ

##### 小ステップ1-2: 1,000〜3,000円の商品を中心にする（10分）
課題: 高額商品の方が報酬が大きいと思い込み、高単価商品ばかり選んでしまう
解決策: 「今なら買うか」と即決できる価格帯の商品を中心に選ぶ
何をするか: 候補商品の価格を確認し、1,000〜3,000円の範囲に当てはまるかチェックする
なぜやるか: 迷わず購入できる価格帯の方が成約率が高くなりやすいから
コツ / 注意点: 高額商品を否定するわけではなく、「初心者はまずこの価格帯から」という位置づけ
成果物: 価格帯でフィルタリングした商品候補リスト
使うツール / リソース: 楽天市場の価格フィルタ機能

##### 小ステップ1-3: 料率10%以上の商品を優先する（10分）
課題: 報酬率を確認せずに商品を選び、労力に見合わない結果になる
解決策: アフィリエイトリンク取得画面で表示される料率を必ず確認し、10%以上を優先する
何をするか: 候補商品の料率を確認し、同じような商品なら料率が高い方を選ぶ
なぜやるか: 同じ売上でも利益額は大きく変わるため、頑張るなら効率がいい商品を選ぶべきだから
コツ / 注意点: 料率は商品によって変動することがあるため、投稿の都度確認する
成果物: 「ジャンル×価格帯×料率」の3基準を満たした投稿候補リスト
使うツール / リソース: 楽天アフィリエイトの商品検索・料率表示画面

#### 大ステップ2: 投稿頻度・タイミングを設計する

##### 小ステップ2-1: 毎日5投稿を最低ラインにする（15分/日）
課題: 「質の高い投稿」にこだわりすぎて投稿数が増えず、当たる投稿に出会えない
解決策: 最初は質より量を優先し、毎日5投稿を最低ラインにする
何をするか: 3基準を満たした商品候補から、1日5件を選んで投稿する
なぜやるか: 投稿数が増えるほど「当たる投稿」に出会う確率が上がるから
コツ / 注意点: 5投稿がきつい日は簡易フォーマットでもいいので数を優先する
成果物: 毎日投稿を継続するルーティン
使うツール / リソース: 投稿スケジュール表（特典で提供）

##### 小ステップ2-2: 5のつく日・セール前は投稿量を倍にする（追加15分）
課題: いつ投稿を増やせばいいかのタイミングが分からない
解決策: 楽天ユーザーが最も買いやすい「5のつく日」やセール前に投稿量を倍にする
何をするか: カレンダーで5のつく日・楽天セール時期を確認し、その前後は投稿数を10件に増やす
なぜやるか: 需要が増えるタイミングに合わせて露出を増やすことで、成約の母数が増えるから
コツ / 注意点: 普段の投稿の質を落とさないよう、事前にネタをストックしておく（大ステップ3と連動）
成果物: セール・特定日に合わせた投稿カレンダー
使うツール / リソース: 楽天セールカレンダー

#### 大ステップ3: 投稿を資産化してネタ切れを防ぐ

##### 小ステップ3-1: 反応が良かった投稿は3日後にもう一度投稿する（5分）
課題: 一度伸びた投稿を「もう使った」と考えて使い捨てにしてしまう
解決策: 反応の良かった投稿を資産と捉え、3日後に再投稿する
何をするか: 過去の投稿の反応を確認し、良かったものを3日後に再度投稿する
なぜやるか: 見ているユーザーは毎回異なるため、再投稿でさらに伸びることが多いから
コツ / 注意点: 全く同じ文面ではなく、画像や一言を少し変えると自然に見える
成果物: 「資産化リスト」（再利用する投稿の一覧）
使うツール / リソース: 投稿履歴・反応記録シート

##### 小ステップ3-2: バズっている競合投稿を50個ストックする（週1回30分）
課題: 投稿ネタが尽きて、毎回ゼロから考えることになる
解決策: 競合のバズっている投稿を50個ストックし、自分用のネタ帳を作る
何をするか: 同ジャンルで反応の良い投稿を見つけ、タイトル・見せ方・切り口をメモしてストックする
なぜやるか: ゼロから考えず、型を参考にすることで投稿作成のスピードと質が上がるから
コツ / 注意点: そのままコピーするのではなく、切り口だけを借りて自分の言葉で書く
成果物: 50個のネタ帳（テンプレ提供）
使うツール / リソース: ネタ帳テンプレート（特典で提供）

#### 大ステップ4: 購買心理を動かし世界観を統一する

##### 小ステップ4-1: 「今買う理由」を必ず添える（5分/投稿）
課題: 商品を紹介するだけで終わり、購入の後押しができていない
解決策: 季節・新生活・セール・梅雨などのタイミングを理由として添える
何をするか: 投稿ごとに「なぜ今この商品が必要なのか」を一言添える
なぜやるか: 季節やイベントがあるだけで購入理由になり、後押しになるから
コツ / 注意点: 理由はこじつけでもいいので、必ず1つは入れる
成果物: 「今買う理由フレーズ集」（特典で提供）
使うツール / リソース: 季節・イベントカレンダー

##### 小ステップ4-2: トンマナを統一する（初回設計30分）
課題: 投稿ごとに色・文章・画像のスタイルがバラバラで、印象に残らない
解決策: 色・文章・画像・投稿スタイルを統一し、「この人の投稿だ」と分かるようにする
何をするか: 使用する色・文体・画像のトーンを最初に決めて、以降すべての投稿に適用する
なぜやるか: トンマナが統一されているだけで信頼につながり、フォロー・リピートされやすくなるから
コツ / 注意点: 完璧を目指さず、最低限「色」と「文体」だけでも統一する
成果物: トンマナ統一ルール（チェックリスト形式で特典提供）
使うツール / リソース: トンマナ設計チェックリスト（特典で提供）

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「特別なスキル・フォロワー数ではなく、仕組みで最初の1万円を突破した」という再現性
- ノウハウ章で画像があると効果的な箇所: ジャンル選定の具体例（第2章）、投稿カレンダー例（第3章）、ネタ帳の作り方（第4章）
- 読者がつまずきやすいポイント: 「毎日5投稿」の継続、価格帯・料率の確認を面倒に感じて省略してしまうこと
- ワーク / テンプレートが必要な箇所: 第2章（商品選定チェックリスト）、第3章（投稿カレンダー）、第4章（ネタ帳テンプレ・トンマナチェックリスト）
- 世界観に合った表現の具体例: 「ルールを積み重ねる」「仕組みで当てる」「淡々と実践する」
- コンテンツ融合法の実行方針:
