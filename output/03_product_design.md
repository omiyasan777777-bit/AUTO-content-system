## PART A: オファー設計

### 訴求4要素

要素1: ベネフィット（得られる成果）
- メインベネフィット: 今日中に初動画完成・月20時間で収益化チャンネルを育てられる
- サブベネフィット1: 月収61万円チャンネルと同じ設計図が手に入る
- サブベネフィット2: 11Labsリスクを回避した安全な収益化ルートがわかる
- サブベネフィット3: 日本語圏の先行者優位をいま取れる
- 感情的ベネフィット: 「自分の力で稼げた」という自己証明・会社依存からの解放感

要素2: 独自性
- 独自メソッド名: **61万円再現メソッド**
- 他商品にない要素: 11Labsリスク開示・ヘタウマ絵の心理学的根拠・ファイル名＝タイムスタンプ自動化
- 自分だけが語れる理由: 実在する月収61万円チャンネルを逆解析し、日本語版として再現した当事者

要素3: 実績・証拠
- 参照チャンネルの実績: 登録者13万人・動画12本・開始1ヶ月・再生数700万回・月収61万円
- 制作時間の実績: 動画1本20分以内（AIワークフロー確立後）
- ツール実績: Claude Code × Hixfield × Turboscribeの3ツールで画像100枚を自動生成

要素4: 緊急性・限定性
- なぜ今買うべきか: 日本語圏では同じ手法を実践している人がまだほぼいない。先行者優位は今しか取れない
- 限定要素: ローンチ期間限定500円（通常980円に値上げ予定）
- 見逃した場合の損失: 競合が増える前に動けない・収益化の黄金期を逃す

### オファー構成

メイン商品
- コンテンツ本体: 15,000字以内・6章構成
- 形式: テキスト（note有料記事）

価格設計
- 本体価格（ローンチ）: 500円
- 通常価格: 980円（リサーチ競合相場の最安値帯に合わせる）
- 最終価格: 1,480円
- 価格正当化ロジック:
  - 比較対象1: Claude Proプラン = 月額3,000円 → 本記事は500円で「元を取る方法」まで教える
  - 比較対象2: YouTube動画制作の外注費 = 1本3〜5万円 → AIで20分・ほぼ無料で作れる
  - 得られる成果の金銭的価値: 月収61万円チャンネルの設計図を500円で手に入れる

バックエンド導線
- 本商品の先に用意する上位商品: ニッチ選定個別相談・Claude Code伴走サポート
- 本商品内でのバックエンド導線: まとめ章で「より深く学びたい方へ」として誘導

## PART B: カリキュラム設計

### 大ステップ一覧

大ステップ1: 月収61万円チャンネルの正体を解剖する → 成果物: 「自分のニッチ」が決まった状態
大ステップ2: 3ツールをセットアップする → 成果物: Claude Code + Hixfield + Turboscribeが使える状態
大ステップ3: 7ステップで動画を完成させる → 成果物: 動画1本が20分以内に完成した状態
大ステップ4: 稼ぐために知っておくべきリスクと戦略 → 成果物: 安全に収益化できるロードマップ

### カリキュラムテーブル

大ステップ1: 月収61万円チャンネルの正体を解剖する
- 1-1: 月収61万円チャンネルの3つの数字 / ヘタウマ絵がなぜ刺さるか / 基本構造の理解
- 1-2: ニッチ選定の方法（VidIQ活用）/ 日本語展開で先行者優位を取る手法

大ステップ2: 3ツールをセットアップする
- 2-1: Claude Code Proプランの登録と初期設定
- 2-2: Hixfieldのセットアップと画像生成テスト
- 2-3: Turboscribeのセットアップとタイムスタンプ付き文字起こしの確認

大ステップ3: 7ステップで動画を完成させる
- 3-1: スクリプトをAIで作成する（テンプレ付き）
- 3-2: ナレーション録音の方法（自分の声 or Fiverr外注）
- 3-3: Turboscribeでタイムスタンプ付き文字起こし → Claude Codeに投げる
- 3-4: 画像100枚を自動生成・ファイル名＝タイムスタンプで自動保存
- 3-5: 動画編集ソフトに並べて完成（10分で完了）

大ステップ4: 稼ぐために知っておくべきリスクと戦略
- 4-1: 11Labsを使ってはいけない理由・安全な代替案
- 4-2: 収益化の条件と現実的タイムライン
- 4-3: ニッチを間違えると全部無駄になる・選定基準

### 章構成マップ

第1章: 導入（この設計図で何ができるか） / 文字数目安: 2,000字
第2章: 月収61万円チャンネルの正体を解剖する（大ステップ1） / 3,000〜4,000字
第3章: 3ツールのセットアップ（大ステップ2） / 3,000〜4,000字
第4章: 7ステップで動画を完成させる（大ステップ3） / 4,000〜5,000字
第5章: 11Labsを使ってはいけない理由とリスク管理（大ステップ4の前半） / 2,000〜3,000字
第6章（まとめ）: 先行者優位・ニッチ選定・最初の一歩 / 1,500字

合計目安: 15,500〜19,500字（フロントエンド基準15,000字を超える想定）

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「月収61万円チャンネルが動画12本で達成できる理由」と「この設計図で今日中に動画が完成する」というゴールの提示
- ノウハウ章で画像があると効果的な箇所: 3ツールのセットアップ画面・ファイル命名規則の例・動画編集ソフトへの並べ方
- 読者がつまずきやすいポイント: Claude Codeの初期設定・Hixfieldへの連携・ナレーション録音の始め方
- ワーク / テンプレートが必要な箇所: スクリプトテンプレート・ニッチ選定チェックリスト
- 世界観に合った表現の具体例: 「設計図通りに組み立ててみましょう」「正体を解剖すると○○が見えてきます」
- コンテンツ融合法の実行方針:
  - 競合1（ロードマップ型）: 全体像を第1章で「設計図のステップ」として提示
  - 競合2（自動化仕組み）: 第4章で「自動化の核心＝ファイル名＝タイムスタンプ」として説明
  - 競合3（ツール3選）: 第3章でClaude Code・Hixfield・Turboscribeの3点に絞って解説
  - 自分の独自要素（ナレッジより注入）: 読ませる文章の型（冒頭を読者の独り言から始める・1段落4行・抽象→具体→抽象）を各章の解説文に反映する
