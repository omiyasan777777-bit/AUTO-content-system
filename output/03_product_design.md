# 商品設計書

生成日: 2026-06-18
コンセプト参照: output/02_concept.md
コンセプト要約: 婚外恋愛の「崩れる瞬間の心理」を五感で描いた女性向け官能小説。読後に自分の感情が言語化される体験を提供する。

---

## PART A: オファー設計

### 訴求4要素

要素1: ベネフィット（得られる成果）
- メインベネフィット: 「一度だけ」と決めたのに崩れてしまった経験を、言語化・肯定される読書体験
- サブベネフィット1: 日常では誰にも話せない感情が「これでいい」と解放される
- サブベネフィット2: フランス書院文庫文体の文学的エロスを楽しむ官能体験
- サブベネフィット3: スクリーンショットしたくなる「自分の気持ちを言い当てた一文」との出会い
- 感情的ベネフィット: 「裁かれない」という安心感と、生きている実感

要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: 崩れる心理×五感官能法
- 他商品にない要素: 行為描写より「触れる前の緊張感と崩れる心理過程」に特化。罪悪感を善悪二元論で描かない
- 自分だけが語れる理由: 「一度だけ」という言葉の嘘を知っている当事者視点

要素3: 実績・証拠
- 婚外恋愛テーマは note #不倫恋愛タグで常に人気（スキ数50〜200以上の作品多数）
- 女性向け官能小説は30〜50代既婚女性の購買力が高いジャンル
- フランス書院文庫は累計1億冊超の実績ある官能ジャンル

要素4: 緊急性・限定性
- なぜ今買うべきか: ローンチ価格は期間限定。「この感情を言語化してくれる場所」は今しか読めない
- 限定要素: ローンチ価格（期間：1週間 or 先着50名）
- 見逃した場合の損失: 「私のことが書いてある」と感じる読書体験を、割増価格で買うことになる

### オファー構成

メイン商品
- コンテンツ本体: 約15,000〜20,000字（フロントエンド）/ 全5章構成
- 形式: テキスト（官能小説）

価格設計
- 本体価格: 1,480円
- ローンチ価格: 980円（1週間限定 or 先着50名）
- 値上げ計画: 980円 → 1,480円 → 1,980円（最終価格）
- 価格正当化ロジック:
  - 比較対象1: フランス書院文庫1冊 = 900〜1,100円 → 同等以上の文学的官能体験が980円
  - 比較対象2: 他の婚外恋愛note有料記事 = 300〜800円 → 5倍の文字数・心理描写の深さで980円
  - 1日あたり換算: 約27円（1ヶ月読み返す場合）

## PART C: 世界観設計

### ワーディング

トーン
- 全体の雰囲気: 静謐・官能的・文学的。説明しすぎない。余白を大事にする
- 一人称: わたし
- 読者の呼び方: （読者への語りかけはしない。物語に没入させる構造）

キーワード（繰り返し使うワード）
- 崩れる / 触れる / 知ってしまった / 諦めたふり / 声のトーン / 指先 / 息 / 夜 / 名前 / 温度

使わないワード
- 不倫・浮気（直接的な断罪ワードは使わない）/ ハッピーエンド / 反省 / 後悔（善悪軸のワードを排除）

章タイトルの統一パターン
- 短い詩的フレーズ（体言止め or 「〜た夜」「〜の〇〇」形式）
- 例：「声の温度」「触れるまでの永遠」「名前が官能になる夜」

メタファー
- 全体のメタファー: **「解禁」**（禁じられていたものが、一瞬だけ解き放たれる）
- 派生表現: 「解禁された息」「解禁の指先」「解禁を知ってしまった体」

### ビジュアルイメージ

本文内の装飾ルール
- 見出しの装飾: シンプル（装飾なし。タイトルのみ）
- 強調の方法: 太字（クライマックスの一文のみ。多用しない）
- 区切りのスタイル: 空白行（---は使わない。余白で場面転換）

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 夫との「倦怠感」の描写を丁寧に。読者が「自分のことだ」と感じる具体的な日常シーン
- 章で画像があると効果的な箇所: 本作はテキストのみで構成（官能小説として純粋に文章体験に集中）
- 読者がつまずきやすいポイント: 第3章のクライマックス（崩れる瞬間）の描写。ここを薄くすると作品全体が崩れる
- ワーク / テンプレートが必要な箇所: なし（小説形式のため）
- 世界観に合った表現の具体例:
  - 「その声のトーンが、夫のものと違った」
  - 「触れるまでの時間が、永遠のように重かった」
  - 「諦めたふりは、もう嘘だと知っていた」
  - 「名前を呼ばれた瞬間、わたしは解禁された」
- コンテンツ融合法の実行方針:
  - 女性向け官能小説【りんご】: 等身大の年齢設定・内面独白の語り口
  - 最後の記憶: 感覚の記憶（声・温度・息）による官能表現の技法
  - #婚外恋愛タグ体験記: 「一度だけのつもりだった」「久しぶりに女として見てもらえた」等のリアルワード
  - 独自要素: 崩れる「前」の心理積み上げに特化した構成 / 善悪を問わない「業」の視点
