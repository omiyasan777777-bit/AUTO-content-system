## 1. コアコンセプト（実利的価値）

**「投資初心者の会社員が、テクニカル分析だけで3ヶ月以内に勝率60%を超える投資ルーティンを確立する」**

### コンセプトの5要素分解

| 要素 | 内容 |
|------|------|
| ターゲット | 投資初心者の会社員（投資歴0〜3年、日中は仕事で相場に張り付けない） |
| ゴール | 3ヶ月以内に勝率60%を超える投資ルーティンを確立する |
| プロセス | テクニカル分析のスキルを習得し、毎朝15分のチャートチェックだけで実践できる再現性あるシステムを作る |
| 独自性 | ファンダメンタル不要・テクニカル特化・2026年相場対応・会社員向け時間設計 |
| ストーリー | 感情売買で負け続けた初心者が、テクニカルという「ルール」を手に入れて、安定した勝率を実現するまでの道筋 |

## 2. 抽象的コンセプト（世界観）

### テーマ: 「チャートを読む眼を持つ者が、相場で生き残る」

#### 世界観キーワード
- **「チャートの処方箋」**: 感情という病を、テクニカルという薬で治す
- **「相場の羅針盤」**: チャートは嘘をつかない。自分の判断軸を持つ投資家になる
- **「感情から解放される投資」**: ルールに従って動けば、もう夜眠れなくなることはない
- **「再現性のある型」**: 一度身につけたスキルは、相場が変わっても使える武器になる

#### 世界観ワーディング（基本トーン）
- 「チャートには答えがある」
- 「感情が負けを作り、ルールが勝ちを作る」
- 「1日15分で相場が変わる」
- 「あなたのチャートを読む眼が、2026年の相場を制する」

### メインコンセプト文
```
感情で売買するから、負ける。

テクニカル分析は難しくない。
必要なのは「3つのインジケーター」と「毎朝15分のルーティン」だけ。

2026年の相場は、AI・半導体・日銀政策転換で大きく動く。
チャートの読み方を知っている人だけが、この波に乗れる。

投資歴10年のテクニカル分析特化者が、
初心者が「最初に覚えるべき型」を、すべて詰め込んだ。
```

```
このnoteを読み終えると：
・チャートを見て「今が買いか売りか」が判断できるようになる

・毎朝15分の投資ルーティンが確立され、本業と両立できるようになる
・3ヶ月後、勝率60%超えを目指す具体的なロードマップが手に入る
```

## 5. 仮想ストーリー（読者の変化ストーリー）

### ビフォー: ターゲットの現在地
```
田中さん（32歳・会社員）のケース：

「投資を始めて1年が経った。でも、毎回なんとなくで銘柄を選んで、
上がったり下がったりを繰り返すだけ。
損切りもできないから、含み損がどんどん膨らんでいる。

チャートの本を買ってみたけど、専門用語ばかりで何も分からない。
そもそも会社員だから、日中に株価を見ていられない。

もうこのまま投資をやめようかと思っている……」
```

### ターニングポイント: このnoteとの出会い
```
このnoteで出会ったのは「3つのインジケーターを使うだけでいい」という事実。

移動平均線・RSI・MACD。
この3つだけを使って、毎朝15分でエントリー候補を絞り込む。

「これなら自分でもできるかもしれない」
初めてそう思えた。
```

### アフター: 3ヶ月後の田中さん
```
3ヶ月後、田中さんのトレードが変わった。

損切りができるようになった。
感情で動かなくなった。
チャートを見て「今は買うべきでない」と判断できるようになった。

まだ完璧ではない。でも、毎月のトレードが少しずつプラスに近づいている。
「投資って、センスじゃなくてスキルだったんだ」

その確信が、田中さんの投資を変えた。
```

繰り返しポイント:

3. 各章の冒頭・締め（2〜3回）
4. まとめ・コールトゥアクション（1回）

## 8. 3つの刷り込み設計

| 刷り込み | 内容 | 実現方法 |
|---------|------|---------|
| ①教育者認知 | 「テクニカル分析のスキルを教えられる専門家」として認知 | 10年の実績・バックテスト検証結果・体系的な解説 |
| ②結果出せる認知 | 「この手法で勝率60%が実現できる」という確信を持たせる | 具体的な数値・チャートパターン解説・実践ステップ |
| ③仲間認知 | 「この人の知識があれば一緒に勝てる」という信頼感 | 著者自身の失敗体験・共感ストーリー・読者への語りかけ |

## 9. Phase 3（商品設計）への引き継ぎメモ

### コンセプトの軸（ブレ禁止）
- ターゲット: 投資初心者の会社員（投資歴0〜3年）
- コア訴求: テクニカル分析 × 毎朝15分 × 3ヶ月で勝率60%
- 差別化: スキル習得型 × 2026年時代性 × 会社員ルーティン特化

### タイトルの方向性
- 「2026年版」を冠に付け時代性を訴求
- 「会社員」「初心者」をターゲット明記
- 「毎朝15分」「勝率60%」「3ヶ月」など数字で具体性を訴求
- テクニカル分析の「処方箋」「羅針盤」などのメタファーも検討

### 構成設計の指針
- 4つの大ステップ（STEP）構造
- 各STEPで1つの大テーマを完結
- フロントエンドとして15,000〜20,000字の情報密度
- 価格: ローンチ980円 → 通常1,480円 → 最終1,980円
