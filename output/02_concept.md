## 悩みの深掘り

```
表面の悩み: 「フォロワーがいないから楽天アフィリエイトで稼げない」
  ↓ なぜそれが悩みなのか？
第2層: 「投稿を頑張っても誰にも届かない・反応ゼロが続いている」
  ↓ それが解決しないとどうなる？
第3層: 「副業で稼ぐことを諦めるしかない。ずっと会社の給料だけで生きていくしかない」
  ↓ 本当はどうなりたい？
第4層: 「会社に依存しない収入の柱を作りたい。スキマ時間で自分の力でお金を稼いでみたい」
  ↓ それが叶ったらどんな感情？
第5層: 「自分でも稼げるんだ、という自信と、未来への選択肢が広がる感覚」
```

→ セールスレターでは「フォロワーがいないと無理」という思い込みの解除 → 「あなたにも稼げる」という希望 → 自信と選択肢の拡大、という全層に訴求する

```
自分の経験: バズ商品を追いかけて成果ゼロ → 自分の悩み起点に切り替えて初成果を出した
  ↓
ターゲットの悩み: 何を投稿すればいいかわからない・商品選びで詰まっている
  ↓
提供する価値: 「商品選びの軸（プチストレスメモ）」と「PR感を消す投稿の型」のセットで、最初のタップ・報酬を体験させる
  ↓
独自の切り口: フォロワー数ではなく「投稿の設計」と「商品選びの哲学」が勝敗を決めるという逆説的な発想。Threads・X両対応。
```

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 副業を始めたい・または始めたばかりのSNS初心者
- 条件1: Threads・Xのフォロワーがほぼいない（〜100人程度）
- 条件2: 「何を投稿すればいいかわからない」または「投稿してもリンクを踏まれない」という壁にぶつかっている
- 条件3: 楽天アフィリエイトに興味があるか、登録済みだが収益がゼロに近い
- ペルソナ名: フォロワー0のみかさん（30代・会社員・副業初心者）

### どこまで（ゴール）
- ゴール: 最初の楽天アフィリエイト報酬（初タップ〜初報酬）を1〜2本の投稿で体験し、月3,000円〜5,000円の収益の土台を作る
- 達成判定基準: アフィリエイトリンクへのタップが1件以上発生・初報酬の確認

### どうやって（手段）
- 手段: 「プチストレスメモ」で商品を選び、「PR感ゼロ冒頭フック」で設計したテキスト投稿を公開する
- 独自メソッド名: **プチストレス式アフィリ投稿法**
- 独自性: バズ商品追いかけ不要。自分の日常のイラっとをメモするだけで無限にネタが生まれる。Threads・X両対応の投稿設計。

## コンセプト二層構造

### コアコンセプト（実利的価値）
> フォロワーがほぼいないSNS初心者が、自分のプチストレスメモと4ステップ投稿の型を使うことで、1〜2本の投稿から楽天アフィリエイトの初報酬を体験できる実践ガイド

### 抽象的コンセプト（世界観）
> **「ストレスが、お金になる」設計図**
- ワーディングの方向性: 日常の小さなイラっとや面倒が「そのままネタ」になる安心感。難しい知識不要・バズ不要・フォロワー不要という三段の解放感。
- ビジュアルイメージ: 手書きのメモ帳→投稿→タップ音（報酬の通知音）というシンプルな流れ

## 仮想ストーリー

【過去】
自分もフォロワーがほぼいない状態で楽天アフィリエイトを始めた。最初はSNSで話題の商品・バズっている商品を片っ端から紹介していた。でも埋もれるだけ。リンクを貼っても誰も踏んでくれなかった。「フォロワーがいない自分には無理なのかもしれない」と諦めかけた。

【転機】
ある日、自分が日常でイラっとしたことをそのまま投稿のネタにしてみた。「これ面倒だよな」という体験から商品を選び、「私も昔これで困ってた」という冒頭から入る投稿を1本作った。そして「PR」という文字を冒頭に出さず、悩みへの共感から始めた。

【変化】
その1本でリンクがタップされた。楽天のクッキーの仕組みで、紹介した商品以外も含めて報酬が発生した。「リンクをタップさせることだけ考えればいい」「自分のストレスがそのままネタになる」という2つの気づきが、すべてを変えた。

【使命】
「フォロワーがいないから無理」と諦めているすべての副業初心者に届けたい。フォロワー数よりも投稿の設計と商品選びの軸の方がはるかに大事だと。

## 商品設計への引き継ぎメモ

- このコンセプトから導かれる大まかなステップ案:
  1. 楽天アフィリエイトの仕組み（なぜフォロワー0でも稼げるか）
  2. 商品選びの軸（プチストレスメモの使い方）
  3. 投稿の4ステップ構成（PR感ゼロ冒頭フック → 結論 → 一次情報 → リンク誘導）
  4. Threads・X別の投稿設計の違い
  5. 実際に1本作って投稿するアクションプラン
- 重点的に掘り下げるべきノウハウ: プチストレスメモの具体例・冒頭フックの複数パターン・楽天クッキーの仕組み図解
- 省略可能な要素: 高度なSNSアルゴリズム論・アカウントブランディング論（初心者には早い）
- 世界観に合ったワーディング例: 「ストレスが報酬に変わる瞬間」「イラっとしたらメモ帳を開く」「1行目で勝負は決まる」「タップさせることだけ考える」
- コンテンツマージ元3商品の取り入れ方:
  1. K_san（全記録型）← 実録体験談・ビフォアフのリアリティある書き方
  2. あざらちママ（教科書型）← 仕組み解説のわかりやすさ・クッキー説明の体系性
  3. nicolala♪（ロードマップ型）← ステップバイステップの進め方・「なぜ稼げないか」の原因分析スタイル
