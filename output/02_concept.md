## 悩みの深掘り

```
表面の悩み: 楽天アフィリエイトを始めたが、何を投稿すればいいか分からず成果が出ない
  ↓ なぜそれが悩みなのか？
第2層: 自分のセンスや投稿の仕方が悪いのではないかと不安になる
  ↓ それが解決しないとどうなる？
第3層: 何の成果も出ないまま挫折し、副業自体を諦めてしまう
  ↓ 本当はどうなりたい？
第4層: 会社の給料だけに頼らず、自分の力で継続的な副収入を得たい
  ↓ それが叶ったらどんな感情？
第5層: 「自分にもできた」という自信と、収入源が増えたことによる安心感
```

## 価値の転換

```
自分の経験: 特別なスキルもフォロワーもない状態から、9つのルールを淡々と実践して30日で楽天アフィリエイト1万円を達成した
  ↓
ターゲットの悩み: 何を投稿すればいいか分からず、成果が出ずに心が折れそうになっている
  ↓
提供する価値: ジャンル・価格帯・料率・投稿頻度・再投稿・ネタ収集・訴求・トンマナという「迷わず動ける9つの基準」
  ↓
独自の切り口: 精神論ではなく、数値基準に落とし込んだ「仕組み」として楽天アフィリエイトを攻略する
```

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 楽天アフィリエイトに取り組む副業初心者
- 条件1: 楽天アフィリエイトのアカウントは作ったが、まだ思うような成果が出ていない
- 条件2: 何を投稿すればいいか、どの商品を紹介すればいいか判断基準がない
- 条件3: フォロワー数や特別なスキルに自信がない
- ペルソナ名: 「あかりさん」26歳・会社員。楽天アフィリエイトを始めて2週間、投稿はしているが反応もクリックも伸びず、このまま続けていいのか不安になっている

### どこまで（ゴール）
- ゴール: 30日以内に楽天アフィリエイトで最初の1万円を突破する
- 達成判定基準: 楽天アフィリエイトの管理画面で累計報酬額が1万円を超えること

### どうやって（手段）
- 手段: 「画像で惹くジャンル選定」「価格帯・料率の数値基準」「毎日5投稿と資産化する再投稿」という9つのルールの実践
- 独自メソッド名: 「9ルール仕組み化メソッド」
- 独自性: センス・フォロワー数に依存せず、誰でも同じ基準で判断・実行できる再現性

## コンセプト二層構造

### コアコンセプト（実利的価値）
> 楽天アフィリエイト初心者が、9つのルールを淡々と実践するだけで30日以内に最初の1万円を突破する方法

### 抽象的コンセプト（世界観）
> 「仕組み化ノート」— センスや才能ではなく、誰でも同じ手順を踏めば同じ結果にたどり着ける「再現性の設計図」という世界観
- ワーディングの方向性: 「ルール」「基準」「仕組み」「淡々と」「積み重ね」といった、感覚論を排した言葉を軸にする
- ビジュアルイメージ: チェックリスト・ルールカードのような、整理された印象

## 仮想ストーリー

```
【過去】
自分も「何を投稿すればいいのか分からない」状態だった。
楽天アフィリエイトのアカウントは作ったものの、
何となく良さそうな商品を投稿するだけで、反応もクリックもほとんどない日々が続いた。

【転機】
「センスの問題ではなく、基準がないだけだ」と気づいた瞬間があった。
画像で止まってもらえるジャンルかどうか、価格帯は買いやすいか、料率は効率がいいか。
一つひとつを数値の基準として言語化した瞬間、投稿の迷いが消えた。

【変化】
9つのルールを淡々と実践したことで、
30日という期間で楽天アフィリエイト収益1万円という最初の壁を突破できた。
一度当たった投稿を資産として使い回す感覚も掴めた。

【使命】
だからこそ、同じように「何を投稿すればいいか分からない」と悩む初心者に、
センスや根性ではなく「仕組み」で最初の壁を越えてほしいと思い、この教材を作った。
```
