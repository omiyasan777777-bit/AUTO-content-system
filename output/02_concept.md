## 悩みの深掘り

```
表面の悩み: 「最初の一歩が踏み出せない・先延ばししてしまう」
  ↓ なぜそれが悩みなのか？
第2層: 「やる気が出るのを待っているが、いつまでたっても出ない。
        ポモドーロも試したが"25分も集中できない自分"に自己嫌悪を感じた」
  ↓ それが解決しないとどうなる？
第3層: 「締め切り直前の追い込みでしか動けないまま人生が過ぎていく。
        やりたいことが何も進まない・自分は変われないという恐怖」
  ↓ 本当はどうなりたい？
第4層: 「"やろう"と思った瞬間に動ける自分になりたい。
        タスクを前にして逃げない、意志の強い自分でありたい」
  ↓ それが叶ったらどんな感情？
第5層: 「自分をコントロールできている感覚・罪悪感ゼロの毎日・
        "自分はやればできる"という自己効力感の回復」
```

→ セールスレターではこの**全層**に訴求する。特に「自己嫌悪の解放」と「自己効力感の回復」が感情的クライマックスになる。

## 価値の転換

```
自分の経験: ポモドーロを集中ツールとして使って挫折した経験がある
  ↓
ターゲットの悩み: 25分集中できない自分はダメだ、という誤った使い方による挫折
  ↓
提供する価値: ポモドーロの使い方を「集中する」から「着手する」に転換する設計を教える
  ↓
独自の切り口: 「ポモドーロは集中ツールではなく、やる気のない状態でも
              動き出しを強制する着手ツールだ」という完全な再定義
```

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 先延ばし癖がある社会人・学生（20〜35歳）
- 条件1: ポモドーロを知っているか試したことがあるが「25分集中できない」と感じた人
- 条件2: 「やる気が出たら始めよう」と待ち続けて動けないループにはまっている人
- 条件3: 作業や勉強の「最初の一歩」だけがどうしても踏み出せない状態
- ペルソナ名: 「着手できない田中さん（28歳・会社員）」

### どこまで（ゴール）
- ゴール: 「今日から24時間以内に、先延ばしにしていた作業に1分以内で着手できるようになる」
- 達成判定基準: タイマーをセットして1分だけ作業を始められたか（yes/no）

### どうやって（手段）
- 手段: ポモドーロを「集中継続ツール」から「着手専用ツール」に再設計し、最小着手単位（1分）から始める段階的着手法
- 独自メソッド名: **「1分着手ポモドーロ」**（または「着手ファーストポモドーロ」）
- 独自性: ①25分を目標にしない ②集中しなくていい ③タイマーをセットすることだけが最初のゴール、という3つの逆転設計

## コンセプト二層構造

### コアコンセプト（実利的価値）
> 先延ばし癖がある人が、ポモドーロを「着手専用ツール」として再設計することで、今日から24時間以内に"1分着手"を実現し、やる気待ちのループから抜け出す

### 抽象的コンセプト（世界観）
> **「スタートボタン」**── 押せば動き出す、たった一つの仕掛け

- ワーディングの方向性: 「仕掛け」「スイッチ」「ボタン」「着火」「点火」「エンジンをかける」「動き出す」「トリガー」
- ビジュアルイメージ: エンジンのかかる瞬間・ロケットの発射カウントダウン・電源ボタンが光る瞬間

## コンセプト文（完成形）

```
先延ばし癖があり、作業や勉強を始めたくても始められない人
・ポモドーロを試したけど25分集中できなかった
・やる気が出るのを待っているが永遠に出ない
・タスクを前にしてスマホや別のことに逃げてしまう
という人が、

ポモドーロを「集中継続ツール」として使うのをやめ、
「まず1分だけ着手するためのスタートボタン」として再設計することで

今日から24時間以内に先延ばしにしていた作業に着手し、
"やる気は動いてから出る"という作業興奮の仕組みを体で理解できる着手術

独自性: 「25分集中しなくていい・集中できなくてもOK・タイマーをセットするだけが最初のゴール」という3つの逆転設計による"1分着手ポモドーロ"
ストーリー: ポモドーロを「集中ツール」として使い、25分集中できない自分に自己嫌悪を感じた経験。本来ポモドーロは"集中する"ための道具ではなく"動き出し"を強制するための道具だったと気づいたとき、先延ばしが消えた。
```

## 仮想ストーリー

```
【過去】
「ポモドーロテクニックを試したことがある。
25分タイマーをセットして、作業しようとした。
でも5分でスマホを触り、8分で別のタブを開き、
15分で"今日はもうやめよう"と思った。

また失敗した。自分に集中力がないのは知っていた。
でも25分すら続けられないのか、と自己嫌悪は深まった。」

【転機】
「あるとき気づいた。
自分はポモドーロを"集中するための道具"だと思っていた。
でも本当は違った。

ポモドーロの発案者・フランチェスコ・チリロが最初に設計したのは
"とにかく作業を始める"ためのシステムだった。
集中できるかどうかは、問題じゃなかった。」

【変化】
「使い方を変えた。
25分集中するのをやめた。
タイマーをセットして、1分だけ作業を始めることだけを目標にした。

すると不思議なことが起きた。
1分経っても止まりたくなかった。
5分、10分、気づいたら25分経っていた。

"やる気は動く前に出るのではなく、動いてから出る"
作業興奮という脳の仕組みを、はじめて体で感じた瞬間だった。」

【使命】
「だから、先延ばし癖に悩む人に伝えたい。
あなたに集中力がないのではない。
ポモドーロの使い方が間違っていただけだ。

25分集中しなくていい。
タイマーをセットして、1分だけ始めればいい。
それだけで、先延ばしの人生は変わる。」
```

  2. 第1章: 「やる気は出してから動くのではなく、動いてから出る」（作業興奮理論）
  3. 第2章: 「ポモドーロを着手ツールに再設計する」（1分着手ポモドーロの設計）
  4. 第3章: 「今日から使える1分着手ポモドーロの手順書」（実践フォーマット）
  5. 終章: 「着手できる自分を習慣にする」（継続設計）

- 重点的に掘り下げるべきノウハウ:
  - 1分着手の具体的な手順（タイマーの使い方・最初の1分にやること）
  - 作業興奮のメカニズム（脳科学的説明）
  - 「25分集中しなくていい」という誤解解除

- 省略可能な要素:
  - 先延ばしの脳科学的な詳細説明（競合が飽和している）

- 世界観に合ったワーディング例:
  - 「スタートボタンを押す」「エンジンをかける」「着火する」
