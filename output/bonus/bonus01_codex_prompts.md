## ⚙️ 事前準備：IDの確認

以下の2つのIDを手元に用意してください。

```
楽天アプリID（applicationId）: webservice.rakuten.co.jp で取得
楽天アフィリエイトID（affiliateId）: affiliate.rakuten.co.jp で確認
```

テンプレート内の以下の部分を自分のIDに書き換えて使用してください。
- `YOUR_APP_ID` → 自分のアプリID
- `YOUR_AFFILIATE_ID` → 自分のアフィリエイトID

## 📋 指示書07：CSVから投稿文を一括生成＋テキスト保存

```

- 言語: Python
- 全て日本語で生成してください
- OpenAI APIを使用して投稿文を生成してください
  - model: "gpt-4o-mini"
  - openai_api_key: "YOUR_OPENAI_API_KEY"（OpenAIのAPIキーに書き換えてください）
- 実行後に「○件の投稿文を生成してposts_YYYYMMDD.txtに保存しました」と表示してください
```

## 📋 指示書08：自動実行メインスクリプト（商品検索〜投稿文生成まで一括）

```
【作業の目的】
楽天商品検索から投稿文生成までを一括で実行するメインスクリプトを作成してください。

【やってほしいこと】
以下の処理を順番に実行するPythonスクリプト「main.py」を作成してください。

処理の流れ:
1. keywords.txtからキーワードを読み込む
2. 楽天市場APIで各キーワードの売れ筋商品を検索（1キーワード5件）
3. レビュー数50件以上・評価3.8以上でフィルタリング
4. フィルタリング後の商品をproducts_YYYYMMDD.csvに保存
5. 各商品のnote用記事文とX用短文を生成
6. posts_YYYYMMDD.txtに保存
7. 実行ログをlog_YYYYMMDD.txtに記録（開始時刻・商品数・完了時刻・エラー情報）

【設定値（スクリプトの先頭にまとめて定義）】
- RAKUTEN_APP_ID = "YOUR_APP_ID"
- RAKUTEN_AFFILIATE_ID = "YOUR_AFFILIATE_ID"
- OPENAI_API_KEY = "YOUR_OPENAI_API_KEY"
- HITS_PER_KEYWORD = 5
- MIN_REVIEW_COUNT = 50
- MIN_REVIEW_SCORE = 3.8

【制約・条件】
- 言語: Python
- エラーが発生した場合もスクリプト全体を止めず、ログに記録して続行してください
- 全処理完了後に「完了: 商品○件、投稿文○件を生成しました」と表示してください
- コメントを日本語で記述してください
```

## 💡 使い方のコツ

**指示書のカスタマイズ方法**

- キーワードリストは自分の戦略に合わせて自由に変更できます
- `hits`の数を増やすと取得件数が増えます（楽天APIの上限は30件）
- フィルタリング条件（レビュー数・評価・価格帯）は特典④のジャンル一覧を参考に調整してください
- 指示書09と組み合わせて、タスクスケジューラーから指示書08のmain.pyを毎日自動実行するのが推奨構成です

**エラーが出たときの対処**

エラーメッセージをそのままCodexに貼り付けて「このエラーを修正してください」と指示するだけで、Codexが自動で修正コードを生成してくれます。
