# 【特典④】ジャンル別・稼ぎやすい楽天商品カテゴリ一覧
## アフィリエイト単価が高いジャンル30選

> このリストを参考に `keywords.txt` にキーワードを設定してください。
> 「高単価 × 高需要 × 季節性あり」の3条件を満たすジャンルが最も稼ぎやすいです。

---

## 🏆 稼ぎやすさの判断基準

| 条件 | 説明 |
|------|------|
| **単価** | 商品価格が高いほどアフィリ報酬も高くなる（楽天の報酬率は概ね1〜3%） |
| **需要** | 楽天での検索数・購買意欲が高いジャンル |
| **競合** | 競合の多さ（少ないほど上位表示しやすい） |
| **季節性** | 季節・イベントに合わせた需要の波 |

**推奨キーワードの選び方:**
- 単価3,000円以上の商品を扱うジャンルを優先する
- レビュー数100件以上の商品が多いジャンルは購買意欲が高い証拠
- 今の季節・直近のイベントに合ったジャンルを優先的に設定する

## 💡 キーワード設定のコツ

**やってはいけないキーワード**
- `服`・`靴`・`バッグ` など広すぎるワード（競合が多すぎる）
- `安い`・`激安`・`最安値` など価格訴求ワード（購買意欲が低い層が来る）

**効果的なキーワードの型**
- `[商品カテゴリ] + [ターゲット属性]` 例：`電動歯ブラシ 子供用`
- `[商品カテゴリ] + [スペック]` 例：`空気清浄機 8畳`
- `[商品カテゴリ] + [悩み解決]` 例：`マットレス 腰痛 低反発`
- `[商品カテゴリ] + [シーン]` 例：`プロテイン 女性 ダイエット`

**keywords.txtへの設定例（テンプレート）**
```
# 通年高需要ジャンル
ロボット掃除機
電動歯ブラシ
ホームジム 器具
空気清浄機 8畳

# 季節対応（月ごとに入れ替え）
空気清浄機 花粉    # 2〜5月向け
# 除湿器 衣類乾燥  # 6〜8月向け（コメントアウトして季節で切り替え）

# 高単価狙い
美顔器 リフトアップ
マットレス 腰痛
ペット 自動給餌器
```
