# 新規アカウント立ち上げチェックリスト
### ── てどまりサロン 購入者限定特典 ──

---

## このチェックリストの使い方

新規アカウントを立ち上げる際に、**上から順番にチェックしながら進める**だけでOKです。
ここに書いてあることを全部やれば、ゼロからでも最短で「稼げるアカウント」の土台が完成します。

各項目の横に「□」があるので、完了したらチェックを入れてください。

---

## 【PHASE 1】アカウント設計（開始前）

### ▼ ポジション設定
- [ ] ジャンルを1つに絞った（恋愛 / 美容 / 副業 / 夜職 / etc）
- [ ] ターゲット（誰の悩みを解決するか）を1行で言語化した
- [ ] 競合アカウントを5つ以上リサーチした
- [ ] 競合と自分の差別化ポイントを1つ決めた
- [ ] 自分の発信できる「1次情報」をリストアップした

> 💡 ポジション設定メモ欄
> ターゲット：
> 差別化ポイント：
> 自分だけが語れること：

---

### ▼ プロフィール設計
- [ ] アイコンを設定した（顔出し or キャラ or 雰囲気が伝わるもの）
- [ ] ユーザー名にジャンル or ターゲットが分かるキーワードを入れた
- [ ] プロフィール文に「誰の・何の悩みを・どう解決するか」を書いた
- [ ] プロフィール文に「自分の実績・証拠」を最低1つ書いた
- [ ] ハイライト or 固定投稿で「自己紹介」「実績」を設置した
- [ ] リンク欄にnote or LINE公式 or ランディングページを設定した

> 💡 プロフィール文テンプレ
> 「〇〇（ターゲット）の△△（悩み）を解決する□□（あなたの肩書き）」
> 実績：〜〜〜
> ↓詳しくはこちら（リンク）

---

## 【PHASE 2】初期コンテンツ仕込み（立ち上げ直後）

### ▼ 最初の9〜12投稿（グリッド設計）
- [ ] 自己紹介投稿を作った（経歴・失敗談・何者かが伝わるもの）
- [ ] 「共感系」投稿を最低3つ用意した（ターゲットの悩みあるあるを語る）
- [ ] 「ノウハウ系」投稿を最低3つ用意した（具体的なTips or How to）
- [ ] 「実績系」投稿を最低1つ用意した（数字・ビフォーアフターなど証拠を出す）
- [ ] 投稿のフォーマット（デザイン・文体）を統一した
- [ ] 1投稿目〜9投稿目まで一気に投稿 or スケジュール設定した

> 💡 グリッド配置の基本
> 共感 → ノウハウ → 実績 → 共感 → ノウハウ → 実績…の繰り返しが理想

---

### ▼ ハッシュタグ・SEO設定
- [ ] ジャンルのビッグキーワード（10万投稿以上）を3つ選んだ
- [ ] ミドルキーワード（1万〜10万投稿）を5つ選んだ
- [ ] スモールキーワード（1,000〜1万投稿）を5つ選んだ
- [ ] ハッシュタグを投稿ごとに組み合わせて使うリストを作った
- [ ] 投稿文にもキーワードを自然に盛り込む習慣をつけた

---

## 【PHASE 3】フォロワー獲得アクション（立ち上げ1〜4週間）

### ▼ 毎日のルーティン
- [ ] 1日1〜3投稿のスケジュールを決めた
- [ ] 競合フォロワーへのいいね・コメントを1日30件以上する
- [ ] 自分のターゲット層のアカウントをフォローする（1日20〜30件）
- [ ] コメントへの返信を24時間以内に行っている
- [ ] DMでのコミュニケーションを積極的に取っている

### ▼ バズり投稿の仕込み
- [ ] 「〇〇したら人生変わった」系のストーリー投稿を用意した
- [ ] 「〇〇TOP5」「〇〇な人の特徴」などリスト形式投稿を用意した
- [ ] 「業界の裏話」「意外と知らない〇〇」系の暴露系投稿を用意した
- [ ] リールorショート動画を週1本以上投稿している（拡散力が段違い）

---

## 【PHASE 4】マネタイズ準備（フォロワー100〜500人目安）

### ▼ 無料→有料の導線設計
- [ ] 無料のプレゼントコンテンツ（PDF or チェックリスト）を1つ作った
- [ ] LINE公式 or メルマガへの誘導を投稿・プロフィールに設置した
- [ ] noteアカウントを開設し、無料記事を最低3本投稿した
- [ ] 有料コンテンツのテーマを1つ決めた

### ▼ 初売りの準備
- [ ] ターゲットの悩みリサーチ（DMやコメントで生の声を集めた）
- [ ] 販売する商品のタイトルを5パターン考えた
- [ ] 価格設定を決めた（初回は500〜3,000円が売れやすい）
- [ ] 購入者の声を集める仕組みを考えた（特典として感想をもらう）

---

## 【PHASE 5】継続・スケールアップ（初収益後）

### ▼ 継続のための仕組み
- [ ] 週次・月次の振り返りをする習慣をつけた
- [ ] インプレッション・フォロワー増加数・売上を記録している
- [ ] 伸びた投稿の傾向を分析して、同系統の投稿を増やしている
- [ ] 全く伸びない投稿パターンを把握して改善している

### ▼ スケールアップ
- [ ] 高単価商品（コンサル・コミュニティ）の企画を始めた
- [ ] コラボ相手・相互紹介できる同ジャンルの発信者を探した
- [ ] 複数SNS（Instagram→X→YouTube etc）への展開を検討した
- [ ] リピーターへのフォローアップ施策を考えた

| Day | やること |
|-----|---------|
| Day 1 | プロフィール完成・1投稿目（自己紹介）UP |
| Day 2 | 2〜3投稿目UP・競合100アカウントをフォロー |
| Day 3 | 4〜5投稿目UP・競合フォロワーへいいね100件 |
| Day 4 | 6投稿目UP・コメント返し・DMコミュ開始 |
| Day 5 | リール or 動画コンテンツを1本投稿 |
| Day 6 | 7〜9投稿目UP・グリッドを整える |
| Day 7 | 1週間の数値を確認・改善ポイントを把握する |

## よくある失敗とその対策

| よくある失敗 | 対策 |
|------------|------|
| 投稿が続かない | 週の投稿テーマをあらかじめ決めておく |
| フォロワーが増えない | 競合フォロワーへのアプローチが足りない（1日30件以上） |
| いいねはつくが売れない | プロフィール→LP→購入の導線が切れている |
| 何を発信すればいいか迷う | ターゲットの「悩み・欲求」に戻って考える |
| バズったのに売上に繋がらない | バズり投稿から「続き」としての有料コンテンツへ誘導できていない |

## 使い方まとめ

```
PHASE 1（アカウント設計）→ PHASE 2（初期コンテンツ）
→ PHASE 3（フォロワー獲得）→ PHASE 4（マネタイズ準備）
→ PHASE 5（継続・スケール）
```

このチェックリストを全部埋めれば、0→1の最短ルートは完成します。
サロン内で分からないことや詰まったことがあれば気軽に質問してください。
