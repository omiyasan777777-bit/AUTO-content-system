# セールスレターの型 一覧

Phase 5（セールスレター作成）で使う「型」をここで管理します。
最初は**デフォルト型のみ**。ユーザーが「お手本にしたい既存レター」を貼り付けると、
AIが逆解析（リバースエンジニアリング）して**新しい型を自動生成・追加**します。

| ファイル | 型名 | builtin |
|---------|------|---------|
| 01_王道17セクション感情型.md | 王道17セクション感情型（デフォルト） | ✅ |


## 型の追加方法（逆解析）
Web UI 右下の「📄 → ➕ 型を追加」、またはチャットで「レターの型を追加」と入力し、
**お手本にしたいセールスレターの全文（noteの無料部分など）を貼り付ける**だけ。
AIが `prompts/reverse_engineer_letter.md` の手順でレターを解剖し、
「誰でも同レベルを再現できる型」を生成して `sales_templates/` に保存、この表も更新します。

## 型の切り替え
画面右上の「レターの型」プルダウンでいつでも切り替え可能。
選んだ型が Phase 5 のセールスレター生成に使われます（未選択時は builtin のデフォルト）。
