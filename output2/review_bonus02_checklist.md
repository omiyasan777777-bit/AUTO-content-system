# 【レビュー特典②】noteが売れない原因が丸わかり！教材復習チェックリスト

---

## ✅ このチェックリストの使い方

「noteが売れない」と感じたとき、このチェックリストを上から順番に確認してください。
チェックが入らない項目が、あなたの「今の課題」です。

---

## 📋 STEP1：リサーチ（第4章）

- [ ] ヤフー知恵袋・YouTube・TikTokで顧客の悩みを50件以上集めた
- [ ] 「急募です！」など今すぐ解決したい悩みを10件以上ピックアップした
- [ ] 20〜30代女性の恋愛悩みに絞ってリサーチできた
- [ ] 競合アカウントを大手・中堅・初期の3タイプで各2〜3個見つけた
- [ ] 「24時間以内に購入されています」が出ているnoteを5個以上チェックした
- [ ] 売れているnoteの無料部分を以下5点で分析した
  - [ ] タイトルの型（悩みワード＋数字＋答えチラ見せ）
  - [ ] 価格帯
  - [ ] どんな悩みへの解決策か
  - [ ] 無料部分でどこまで書いているか
  - [ ] 有料部分の章構成
- [ ] 5日間以上、同じnoteの売れ行きを毎日確認した

---

## 📋 STEP2：アカウント設計（第5章）

- [ ] ストーリーの4要素（どん底・発見・実践・成功）をすべて埋めた
- [ ] 「発信者のビフォーアフター」が1〜3行で言えるようになっている
- [ ] ターゲット（誰に向けて発信するか）を具体的に決めた
- [ ] 顧客が「今うまくいっていない理由」を言語化できた
- [ ] 「仮想敵（顧客を間違った方向に導いているもの）」を定義した
- [ ] 解決策（どんな方法で悩みを解決するか）を明確にした
- [ ] 「noteを読んだ後の理想の未来」を感情が動くレベルで書けた
- [ ] 「noteを読まなかった場合の最悪の未来」を具体的に書けた
- [ ] プロフィール文に「ビフォーアフター＋現状＋発信理由」が入っている
- [ ] noteのタイトル案を3本以上作った

---

## 📋 STEP3：note作成（第6章）

- [ ] 穴埋めワークシートをAIに読み込ませてnoteを作成した
- [ ] 無料部分に以下の構成が入っている
  - [ ] フック（冒頭1〜2行で手を止める一文）
  - [ ] 悩みへの共感（「これは私のことだ」と思わせる描写）
  - [ ] 解決策のチラ見せ（答えの入口だけ見せる）
  - [ ] ストーリー（体験談＋具体的変化）
  - [ ] 理想の未来（読んだ後の自分を想像させる）
  - [ ] 後押し（今すぐ行動する理由）
- [ ] タイトルに「悩みワード＋数字＋答えのチラ見せ」が入っている
- [ ] 有料部分は「具体的な解決方法・実践ステップ」まで書かれている
- [ ] 「抽象的アドバイス」ではなく「読者が試せる手順と理由」まで書いた
- [ ] 実体験・データ・具体例を最低3か所以上入れた
- [ ] noteの冒頭3行（現状の言語化・自分の過去を重ねる・何が変わるか）を書いた
- [ ] 「メリット」ではなく「ベネフィット（人生・感情がどう変わるか）」で語った

---

## 📋 STEP4：SNS投稿・運用（第7章）

- [ ] Threadsでジャンル認知（おすすめ欄に恋愛コンテンツが流れてきている）
- [ ] 1日3投稿を最低2週間継続した
- [ ] ThreadsとXの2軸で運用している
- [ ] 投稿の5種類（バズ・共感・理念・ノウハウ・セールス）をバランスよく使っている
- [ ] セールス投稿の比率は全体の2〜4割になっている
- [ ] noteのリンクは「本文ではなくリプ欄」に貼っている
- [ ] フックが「常識破壊・問いかけ・数字・共感・セリフ」のいずれかになっている
- [ ] 投稿に「数字・五感・セリフ・行動描写」のリアルさが入っている
- [ ] セールス投稿を同じ切り口で繰り返していない（3〜5パターン以上作った）
- [ ] 投稿ごとのインプ・いいね数を記録している
- [ ] 反応が良かった投稿をリライトして再利用した

---

## 📋 STEP5：売れない時の改善（第8章）

- [ ] noteのダッシュボードでアクセス数（PV）を確認した
- [ ] PVが少ない → セールス投稿を増やした
- [ ] PVはあるのに売れない → 無料部分の冒頭3行を書き直した
- [ ] セールス投稿の切り口を3〜5パターン変えて試した
- [ ] 自分のnoteの「悩み・原因・解決策・理想の未来・最悪の未来」を人に説明できるレベルで理解した
- [ ] ライバルのnoteと自分のnoteの差別化ポイントを3つ言えるようになった
- [ ] DMで3件以上の相談に丁寧に対応した
- [ ] 購入者から感想をもらい、投稿素材として活用した（掲載許可を取った）

---

## 🔍 チェック結果の見方

| チェック数 | 状態 | アドバイス |
|-----------|------|----------|
| 0〜15個 | リサーチ・設計が不足 | STEP1〜2を最優先で教材に戻って読み直す |
| 16〜30個 | note・投稿の質が課題 | STEP3〜4の未チェック項目を1つずつ実践 |
| 31〜40個 | 改善サイクルが回っていない | STEP5で原因を特定して1点だけ修正する |
| 41〜45個 | 基本はできている | 継続＋DM活用で売上を積み上げる段階 |
