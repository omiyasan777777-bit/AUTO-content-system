# ▼ 特典3：月1万円達成チェックリスト（全20項目）

> 「省エネ設計図」本編 第5章 対応

---

## 使い方

1. 現在できているものに ✅ をつける
2. チェックが少ない項目が「今やること」
3. 週1回見直して進捗を確認する
4. 全20項目にチェックが入ったとき、月1万円の土台は完成

---

## カテゴリ1：土台設計（5項目）

| チェック | 項目 | できていない場合は… |
|---------|------|-------------------|
| ☐ | 1本目の有料noteが公開済みである | → 本編 第4章を実践して1本目を公開しよう |
| ☐ | タイトルに「誰が・何を得られる・どうやって」が含まれている | → タイトル方程式で見直す（本編 第3章） |
| ☐ | 価格が980〜1,980円の範囲に設定されている | → 価格の3軸マトリクスを確認（本編 第3章） |
| ☐ | ハッシュタグが5〜6個設定されている | → 大・中・小カテゴリで5〜6個設定する（本編 第4章） |
| ☐ | 無料公開部分（セールスレター相当）が500字以上ある | → 4段構成テンプレートで書き直す（本編 第4章） |

**カテゴリ1の達成数：　　/5**

| ☐ | 読者が実際に行動できるワークまたはチェックリストがある | → 各ステップの末尾に📝ワークを1つ追加する |
| ☐ | 読者の「悩み→解決→実践」の流れになっている | → 6ブロック構成を確認する（特典2を参照） |
| ☐ | 次のステップへの誘導文が最後にある | → ブロック6（次のステップへ）を追加する（特典2を参照） |

**カテゴリ2の達成数：　　/5**

## 月1万円達成後：次のステップ

月1万円の土台が完成したら、次の目標は **月3万円** です。

**月3万円への設計図（2ステップ）**

**ステップ1：記事を増やす**
目標：20本以上の記事を積み上げる
方法：シリーズ化・転用パターンで月2〜3本ペースに上げる

**ステップ2：高単価記事を1〜2本追加する**
目標：2,980〜4,980円の記事を作る
方法：複数の記事をまとめた「完全版」または「上級編」として展開する

```
月1万円 → 月3万円の計算式例

パターンA: 1,000円×30本販売 = 月3万円
パターンB: 1,000円×10本 + 3,000円×7本 = 月3.1万円
パターンC: 1,480円×20本 = 月2.96万円
```
