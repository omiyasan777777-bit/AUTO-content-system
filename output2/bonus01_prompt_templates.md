## 温泉旅館版

```
楽天トラベルで温泉旅館を紹介するLPを作って下さい。

【デザイン要件】
・和モダン×西欧スタイリッシュの融合デザイン
・ヒーローセクション（旅館の全景写真 + 「極上の湯と料理に出会う旅」等のキャッチコピー）
・温泉・料理・客室・サービスを4つのアイコン付きで紹介するセクション
・宿泊者の声（3件分のプレースホルダー）
・「今すぐ空室を確認する」CTAボタン
・季節感のあるアクセントカラー（抹茶グリーン・深紅・藍色など）

【量産要件 — 最重要】
大元をコピーすれば、何個でも作れるようにして下さい。
差し替え箇所にコメントを入れてください：
・旅館名（「← ここに旅館名を入力」）
・メインビジュアル画像URL（「← ここに画像URLを入力」）
・温泉の種類・特徴（「← ここに温泉の説明を入力」）
・料理の特徴（「← ここに料理説明を入力」）
・楽天アフィリエイトリンクURL（「← ここにアフィリリンクを入力」）
・旅館の所在地（「← ここに所在地を入力」）

【出力形式】
・HTML/CSS/JavaScript（1ファイル）
・モバイルファーストのレスポンシブデザイン
・Google Fonts（Noto Serif JPまたはSawarabi Mincho）
・フェードインアニメーション
・カラーパレット：深みのある落ち着いた和風配色
```

## リゾートホテル版

```
楽天トラベルでリゾートホテルを紹介するLPを作って下さい。

【デザイン要件】
・明るく開放的な南国・リゾート感あふれる西欧スタイルデザイン
・ヒーローセクション（プールまたは海が見える写真 + 「非日常への扉を開く」等のキャッチコピー）
・プール・海・アクティビティ・ダイニングを4つ紹介するセクション
・宿泊者の声（3件分のプレースホルダー）
・「今すぐ予約する（限定プランあり）」CTAボタン
・リゾート感を演出するカラー（ターコイズ・サンドベージュ・コーラルピンク）

【量産要件 — 最重要】
大元をコピーすれば、何個でも作れるようにして下さい。
差し替え箇所にコメントを入れてください：
・ホテル名（「← ここにホテル名を入力」）
・メインビジュアル画像URL（「← ここに画像URLを入力」）
・ホテルの特徴1〜4（「← ここに特徴を入力」）
・ロケーション説明（「← ここに所在地・アクセスを入力」）
・楽天アフィリエイトリンクURL（「← ここにアフィリリンクを入力」）

【出力形式】
・HTML/CSS/JavaScript（1ファイル）
・モバイルファーストのレスポンシブデザイン
・Google Fonts（Playfair DisplayまたはCormorant Garamond）
・スクロール時のパララックス効果（軽量版）
・カラーパレット：明るく爽やかなリゾート配色
```

## 観光地老舗ホテル版

```
楽天トラベルで観光地の老舗ホテルを紹介するLPを作って下さい。

【デザイン要件】
・格調高く信頼感のある老舗感×現代的なスタイリッシュデザイン
・ヒーローセクション（ホテル外観または客室の写真 + 「歴史と格式が宿る、上質な滞在」等のキャッチコピー）
・歴史・立地・料理・サービスを4つのアイコン付きで紹介するセクション
・創業年・受賞歴・メディア掲載歴のバッジセクション
・宿泊者の声（3件分のプレースホルダー）
・「空室・料金を確認する」CTAボタン

【量産要件 — 最重要】
大元をコピーすれば、何個でも作れるようにして下さい。
差し替え箇所にコメントを入れてください：
・ホテル・旅館名（「← ここにホテル名を入力」）
・メインビジュアル画像URL（「← ここに画像URLを入力」）
・創業年（「← ここに創業年を入力」例: 明治15年創業）
・特徴1〜4（「← ここに特徴を入力」）
・観光地名・アクセス（「← ここに所在地を入力」）
・楽天アフィリエイトリンクURL（「← ここにアフィリリンクを入力」）

【出力形式】
・HTML/CSS/JavaScript（1ファイル）
・モバイルファーストのレスポンシブデザイン
・Google Fonts（Cormorant Garamondなど格調あるセリフ体）
・フェードインアニメーション
・カラーパレット：ネイビー・ゴールド・クリーム（格調ある配色）
```

## 使い方

1. 上記のプロンプトをコピーする
2. Manusに貼り付けて送信する
3. 1時間待てばLPが完成する
4. 差し替えコメント（「← ここに〜」）を探して実際のホテル情報に置き換える
5. 楽天アフィリエイトリンクを設置して公開する
