## 2. 実リサーチから判明した市場の実態

### 読者が本当に知りたいこと（検索データ・体験談から）

1. **「最初の3〜6ヶ月は収益ゼロが普通」**という現実 → ほとんどの記事がここを正直に書いていない
2. **初報酬のリアルな時期と金額** → 3〜4ヶ月目、数百〜数千円が多い
3. **月5万円達成に必要な記事数・PV数** → 現実的には100記事前後・月3〜10万PV
4. **主婦向けで稼げる具体的なASP案件名** → A8.net・もしもアフィリエイトが鉄板。案件は転職・スキルアップ・金融・通販
5. **X発信とブログをどう連携させるか** → 検索（SEO）は3〜6ヶ月かかるのでSNSで補う戦略が必須
6. **育児の隙間に何分で何をやるか** → 「10分でできること」「30分でできること」の具体的な仕分けが欲しい

### 競合が扱っていない空白地帯

1. **「パートの時給で副業の価値を可視化する」フレーム** → 誰もやっていない
2. **「なぜ3ヶ月ゼロでも止まらないか」の仕組み的な説明** → 精神論で終わる競合が多い
3. **X発信×ブログ×ASPの三角連携の具体的な手順** → X単独・ブログ単独の記事は多いが、掛け合わせを丁寧に解説したものがない
4. **育児の隙間「10分・30分・60分」別のタスク設計表** → 時間管理を語る記事はあるが「何をするか」の中身が薄い

## 3. 読者リサーチ（リアルな声・体験談から抽出）

### 実際の失敗パターンTOP3

1. **「何から始めていいかわからず止まった」** → 情報過多で動けない
2. **「3ヶ月ゼロで諦めた」** → 「仕様」だと知らなかった
3. **「ブログを書いても誰にも読まれない」** → 集客（SNS）と記事作成を同時に解説してくれる情報源がない

### 使われているリアルな言葉

- 「隙間時間で」「寝かしつけ後に」「育児しながら」
- 「パートより稼げた」「時給で考えると」
- 「スキルなし」「主婦でもできた」「ゼロから」
- 「正直に書きます」「リアルな話」
- 「続けられなかった」「3日坊主」

### 理想の未来TOP3

1. パートを辞めて、自分のペースで稼げるようになりたい
2. 子どもが熱で休んでも収入が途絶えない状態を作りたい
3. 夫の収入に頼らず、自分名義の口座にお金を積みたい

## 5. 差別化ポイント（本商品の勝ち筋）

### 差別化1: 「パートvs副業」の時給対比フレーム
- 競合: 「副業を始めよう」という前向きな訴求のみ
- 本商品: 「時給1,000円パートと記事1本の時給を数字で比較」→ 論理的な納得を作る
- 具体例: 「3,000円案件の記事を書くのに3時間かかれば時給1,000円。でも同じ記事が1年間稼ぎ続ければ時給換算は青天井」

### 差別化2: 「10分・30分・60分別」タスク設計
- 競合: 「隙間時間でできます」という抽象的な表現
- 本商品: 「今日の育児隙間が10分なら○○、30分なら○○、60分確保できたら○○」と具体的タスク仕分け

### 差別化3: 「3ヶ月ゼロの仕組み的説明」
- 競合: 「3ヶ月は結果が出ない（精神論）」
- 本商品: 「なぜ3ヶ月ゼロなのか仕組みで説明」→ Googleの評価サイクル・ASPの承認期間・積み上がりの複利構造を図解感覚で説明

### 差別化4: X発信×ブログ連携の具体的手順
- 競合: どちらか一方の解説がほとんど
- 本商品: 「X投稿でブログに誘導→ブログでASP案件へ誘導」の三角運用を一冊で完結
