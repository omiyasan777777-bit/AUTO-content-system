## 3. 確認済みの技術的事実（リサーチによる裏付け）

■ a-Shellで使えることが確認済みのこと
- Python 3系が動作する
- `pip install` でピュアPythonのパッケージがインストール可能
  - requests: HTTPリクエスト送受信
  - feedparser: RSS/Atomフィード解析（ピュアPython）
  - beautifulsoup4: HTMLパース（ピュアPython）
- ファイル操作（テキストファイルの読み書き・管理）
- iOS Shortcutsとの連携: クリップボード経由でテキストやりとり可能
- ホーム画面のショートカットからa-Shellのスクリプトをワンタップ起動可能

■ アメブロ固有の確認済み事実
- アメブロのRSSフィードURL: `https://ameblo.jp/[ブログID]/rss20.xml`
  - 無料で取得可能・認証不要
  - 記事タイトル・投稿日時・URL・本文抜粋が含まれる
- アメブロのアクセス解析データは**3ヶ月分しか保存されない**（公式仕様）
  - 3ヶ月を過ぎると古いデータが見られなくなる
  - 手動でコピーするか自動保存しないと過去データが消える

■ 技術的に実装可能な4つのユースケース
1. アクセス解析データの自動保存（requests + BeautifulSoup）
2. RSS取得・人気記事分析（feedparser）
3. 定型文テンプレート管理 → ショートカット経由でアメブロに貼り付け
4. ショートカット連携でホーム画面ワンタップ化

## 4. 顧客リサーチ（ターゲットの悩み）

### 悩みTOP5
1. アメブロのアクセス解析が3ヶ月で消えることを知らない or 知っていても対処できていない
2. 人気記事を分析したいが、アメブロの管理画面では十分なデータが見えない
3. 記事の書き出し・定型文を毎回手打ちしていて時間がかかる
4. iPhoneだけでブログの「数値管理」をしている人がほぼいない
5. PCがなくても本格的なブログ運営ができるか不安

### 理想の状態
- アメブロの数値データが自動でCSVに保存され、いつでも振り返れる
- 人気記事のパターンを把握して、次に書く記事に活かせる
- よく使う文章を一瞬で呼び出せて、記事作成時間が大幅短縮できる
- ホーム画面からタップひとつで必要なツールが全部起動する
