### 商品①
**タイトル**: 占い×note副業は「チート」だった。知識ゼロから月30万円稼ぐ再現性MAXの裏技を徹底解説

**URL**: https://note.com/miyuki_fukugyou/n/nc1d5225c2a97

**著者**: みゆき（生成AIの副業紹介）

**価格**: 推定 980〜1,980円（note 副業ノウハウ系の相場帯。タイトルと内容の格から1,480〜1,980円の可能性が高い）

**スキ数・販売規模**: 検索1位〜2位に複数回登場。「再現性MAX」「裏技」等のキャッチで注目度高い。具体的スキ数は非公開だが検索上位常連のため推定スキ50〜150以上

**メインの訴求（約束していること）**:
- 占いの知識ゼロでも月30万円稼げる
- 「チート」と表現するほどの再現性の高さ
- 初心者でも1ヶ月で最初の販売を達成できる

**ターゲット像**:
- 副業を始めたいが何をすべきか分からない女性
- 占いに興味はあるが専門知識を持っていない人
- 生成AIを使った副業に関心がある人（初心者〜中級者）

**タイトル構造**:
```
[驚き・逆説] × note副業は「チート」だった。
[具体的な成果] 知識ゼロから月30万円稼ぐ
[信頼・再現性] 再現性MAXの裏技を徹底解説
```
→ 逆説ワード（チート）＋数字（月30万円）＋再現性の強調 という3点セット

**無料部分の構成（推定）**:
1. 「占いでこんなに稼げるの？」という驚きエピソード（フック）
2. 著者自身の体験（AI＋noteで何ができたか）
3. このノートで分かること（目次・概要）
4. 一部の実績・体験談の提示
5. 有料パートへの誘導

**強み**:
- キャッチーな「チート」「再現性MAX」というコピーが強い
- 検索上位に複数キーワードで登場する露出力
- 生成AI副業という旬のテーマとの掛け合わせ

**弱み**:
- 「月30万円」という数字が誇大に見えリスクがある
- 著者の具体的な実績証拠（スクショ等）が薄い可能性
- 占い×AI の融合の深掘りが浅く、Manusには言及なし

- ChatGPTで鑑定文テンプレートを作成 → noteで高単価販売 → 不労所得化

**ターゲット像**:
- 個別占い鑑定の時間労働から脱したい占い師
- テンプレート・プロンプトを販売して収益化したい副業志望者
- 「不労所得」「自動化」に魅力を感じるビジネス志向の人

**タイトル構造**:
```
[商品の形式] 「AI × 占いnote」販売 自動化マニュアル
[カッコ書きで補足] （ChatGPTで〇〇を作成・販売し、
[最大メリット] 不労所得を得る方法）
```
→ 商品形式の明示（マニュアル）＋具体的手段（ChatGPT）＋最終ベネフィット（不労所得）

1. 「占いの個別鑑定は時間労働」という問題提起

### よく使われるタイトルパターン

1. **数字＋ベネフィット型**
   - 「月〇万円」「〇日で」「〇つの理由」
   - 例: 「知識ゼロから月30万円稼ぐ」「月5万円増」

2. **逆説・驚き型**
   - 「チートだった」「実は〇〇だった」「知らないと損」
   - 例: 「占い×note副業は『チート』だった」

3. **完全ガイド・ロードマップ型**
   - 「完全ロードマップ」「徹底解説」「完全攻略ガイド」
   - 例: 「完全ロードマップ！初心者でも今日から」

4. **初心者安心型**
   - 「知識ゼロ」「初心者でも」「今日から」「スキルゼロ」
   - 例: 「AI初心者が」「占い初心者でも」

5. **自動化・不労所得型**
   - 「自動化」「不労所得」「放置しても」「寝ている間に」
   - 例: 「販売 自動化マニュアル」「放置しても収益が増える」

6. **ツール名×副業型**
   - 「ChatGPT×〇〇」「Manus AI×〇〇」「生成AI×〇〇」

## 市場の空白地帯（差別化チャンス）

現在の競合が埋めていない領域：

1. **Manus×占い の掛け合わせ**
   - Manus系記事は占いに未参入、占い系記事はManusを使っていない
   - この組み合わせはほぼ空白地帯（Tips.jpに1記事あるのみ）

2. **占い副業の「量産システム化」**
   - ChatGPT止まりの競合が多く、Manusのエージェント機能（自律実行・複数タスク並列）を活かした仕組みを解説するコンテンツが皆無

3. **note投稿まで含めた一気通貫フロー**
   - 「Manusで鑑定文作成→note自動投稿→SNS集客→販売」の全フローを1冊で解説するコンテンツが見当たらない

4. **占い×Manus×SNS集客の3点セット**
   - SNS集客まで含めた包括的なシステムを扱った競合は見当たらない

---

## 競合の共通弱点

- 体験談が薄い（著者自身のリアルな数字・スクリーンショットが少ない）
- 「月〇万円」の根拠が曖昧
- Manusを活用した占い副業コンテンツがほぼ存在しない
- テンプレート提供はあるが、Manusの自律エージェント機能を活用した「放置型」の具体的フローが未解説
