# 【特典1】U-NEXT特化 Codexプロンプト集（15本）

> このファイルのプロンプトは **ChatGPT / Claude / Gemini** いずれでも使用できます。
> 使い方：プロンプト全文をそのままAIのチャット欄にコピペ → 送信するだけ。

---

## 使い方ガイド

### 基本の流れ
1. AIツール（ChatGPT無料版 or Claude無料版）を開く
2. 使いたいプロンプトをコピー
3. `【アフィリリンク】` の部分を自分のbit.lyリンクに置き換える
4. AIに送信 → 生成された文をチューニング（語尾・短縮・感情ワード追加）
5. Xに投稿 or 予約投稿にセット

### カスタマイズのコツ
- **ドラマタイトル**を実際の話題作に変えると刺さりが上がる
- **投稿本数**は `10本` → `5本` や `20本` に変えてもOK
- **ターゲット**を自分のフォロワー層に合わせて調整する

## チューニングチートシート

生成された文章に使える「改善フレーズ集」

### 語尾の人間化（例）
| AI生成の語尾 | 人間らしい語尾 |
|------------|------------|
| 〜できます。 | 〜できる✨ |
| 〜しましょう。 | 〜してみて！ |
| 〜おすすめです。 | 〜マジでおすすめ |
| 〜なります。 | 〜なるから |
| 〜ください。 | 〜してほしい |

### 感情ワード（追加用）
- 驚き系：「えっ」「マジで」「これ知らなかった」「衝撃」
- 共感系：「わかりすぎる」「同じ人いる？」「あるある」
- 期待系：「沼確定」「神」「一気見した」「止まらない」
- 緊急系：「今だけ」「見逃せない」「期間限定」
