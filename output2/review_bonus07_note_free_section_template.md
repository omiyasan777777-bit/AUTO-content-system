# 【新レビュー特典⑦】これを埋めるだけ！売れるnote無料部分の穴埋めテンプレート

---

## ✅ はじめに：なぜ「無料部分」が売上を決めるのか

noteの有料部分がどれだけ良くても、無料部分を読んで「欲しい」と思わなければ購入されません。

```
有料部分の質 → リピート率・高評価・口コミを決める
無料部分の質 → 購入率を決める
```

**まず無料部分に全力を注いでください。**

このテンプレートは「穴埋めするだけで無料部分が完成する」構成になっています。

---

## ◆無料部分の基本構成（6ブロック）

```
① フック（タイトル〜冒頭3行）
② 悩みへの共感
③ 解決策のチラ見せ
④ ストーリー（体験談）
⑤ 理想の未来（ベネフィット）
⑥ 後押し（購入への一押し）
```

---

## ◆ブロック①：フック（タイトル〜冒頭3行）

**目的：スクロールを止めて「これは私のために書かれた」と思わせる**

### タイトルの穴埋めテンプレート

```
パターンA：【ターゲットの状況】から【結果】までの全記録
例：「既読スルーが続く彼からやっぱりお前がいいと言わせるまでの全記録」

パターンB：【NG行動・知らないこと】をやっていると【最悪の未来】になる
例：「冷却期間にただ待つだけの女性が一生復縁できない本当の理由」

パターンC：【数字・期間】で【ゴール】した【発信者の属性・方法】
例：「3ヶ月の沈黙後に元彼からプロポーズされた私がやったこと全部」
```

---

### 冒頭3行の穴埋めテンプレート

```
1行目（読者の現状を言語化）：
「【読者が今いる状況を1行で代弁する文】」

例：「彼から連絡が来ない夜、何度もスマホを見てしまうあなたへ」
→ あなたの読者の現状：＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

---

2行目（自分の過去を重ねる）：
「私も【期間・状況】、まったく同じ夜を繰り返していました。」

例：「私も2年前、毎晩同じ夜を過ごしていました。」
→ あなたが重ねる過去：＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

---

3行目（このnoteで何が変わるか）：
「このnoteを読み終わる頃には、【変化・得られるもの】がわかります。」

例：「このnoteを読み終わる頃には、彼が離れた本当の理由と、そこからどう動けばいいかが全部わかります。」
→ あなたのnoteで得られること：＿＿＿＿＿＿＿＿＿＿＿＿＿
```

## ◆ブロック②：悩みへの共感

**目的：「この人は私の気持ちをわかっている」と感じさせる**

### 穴埋めテンプレート

```
こんな気持ち、ありませんか？

・「【悩み①を具体的に・1行で】」
・「【悩み②を具体的に・1行で】」
・「【悩み③を具体的に・1行で】」

↓ あなたの読者の悩みを3つ書く
① ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
② ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
③ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

---

【共感・代弁の文章テンプレ】

「好きだから連絡したい。でも連絡したら嫌われそうで怖い。
この矛盾した気持ち、抱えたまま毎日を過ごしている。」

↓ あなたの読者の矛盾・葛藤を1〜2行で書く：

「うまくいっていないのは、あなたに魅力がないからじゃない。
ただ【本当の原因を1行で】を知らなかっただけです。」

↓ あなたが伝えたい「本当の原因」を書く：

## ◆ブロック③：解決策のチラ見せ

**目的：「続きを読みたい」と思わせる。答えの入口だけ見せる**

なぜなら【本当の原因・男性心理などを1〜2行で】だから。」

↓ あなたのnoteのキモとなる「本当の理由」を書く：

---

【「知ってるだけで変わる」を示すテンプレ】

「これを知っているだけで、
・【変化①】
・【変化②】
が変わります。」

↓ あなたのnoteを読んで変わること（2つ）：

↓ あなたのnoteの章構成（目次）を書く：

## ◆ブロック④：ストーリー（体験談）

**目的：「この人は本当に変われた人だ」と信頼させる**

### 穴埋めテンプレート

```
【自己紹介＋どん底の描写テンプレ】

「はじめまして。【名前・肩書き】です。

【過去の状況を具体的に1〜2行】。
【最もつらかった瞬間を1行・セリフや行動描写で】。

あの頃の私に、今の私が言えるとしたら
『大丈夫。ちゃんと変われるよ』と言いたい。」

↓ あなたのどん底エピソードを書く：

---

【転機→変化テンプレ】

「転機は【気づいたこと・出会ったこと】でした。

そこから【具体的に取り組んだことを1〜2行】。

結果、【変化した状態を具体的に】。」

↓ あなたの転機と変化を書く：
転機：＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
変化：＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

同じように悩んでいる人に、この経験が役立てばと思いこのnoteを書きました。」

## ◆ブロック⑤：理想の未来（ベネフィット）

**目的：「このnoteを読んだら、こんな未来が手に入る」と想像させる**

### 穴埋めテンプレート

```
【このnoteを読んだ後の変化テンプレ】

「このnoteを読んで実践した人は、こんな変化が起きています。

・【具体的な変化①。感情が動くレベルで書く】
・【具体的な変化②。感情が動くレベルで書く】
・【具体的な変化③。感情が動くレベルで書く】」

↓ あなたのnoteを読んだ読者に起きる変化を3つ書く：
① ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
② ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿
③ ＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

---

【「機能」ではなく「感情変化」で書くコツ】

NG：「このnoteでは男性心理を学べます」
　　→ 機能（メリット）を語っている

OK：「このnoteを読み終えた後には、
　　彼がなぜ離れていったのかが腑に落ちて、
　　次にすべき行動が自分で判断できるようになります」
　　→ 感情変化・未来の状態（ベネフィット）を語っている
```

## ◆ブロック⑥：後押し（購入への一押し）

**目的：「今すぐ読みたい」という気持ちを引き出す**

### 穴埋めテンプレート

```
【「このままでいいのか」を問いかけるテンプレ】

「このまま【NG行動・現状】を続けていたら、
【最悪の未来を1行で具体的に】。

でも今日、このnoteを読んで1つだけ動けたら、
半年後の景色は全然違います。」

↓ あなたの読者が放置した場合の最悪の未来を書く：
＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

---

【「あなたのために書いた」テンプレ】

「このnoteは【ターゲットを1行で明確に】ために書きました。

【価格】円で、【価値を一言で表現】。
まずは無料部分だけでも、読んでみてください。」

↓ あなたのターゲットと価格・価値を書く：
ターゲット：＿＿＿＿＿＿＿＿＿＿＿＿
価格：＿＿＿円
価値を一言で：＿＿＿＿＿＿＿＿＿＿＿
```

## ◆完成チェックリスト

無料部分を書き終えたら、以下を確認してください。

```
□ タイトルに「悩みワード＋数字 or 答えのチラ見せ」が入っている
□ 冒頭3行で「現状の言語化→自分の過去を重ねる→このnoteで変わること」が伝わる
□ 共感ブロックで「あなたは悪くない」のトーンが伝わっている
□ 解決策のチラ見せで「続きを読みたい」と思わせる切り方ができている
□ ストーリーに具体的なエピソード（セリフ・数字・行動描写）が入っている
□ ベネフィットが「機能」ではなく「感情変化・未来の状態」で書かれている
□ 後押しで「今すぐ動く理由」が伝わっている
□ 全体を通して「テンプレっぽさ」がなく自分の言葉になっている
```
