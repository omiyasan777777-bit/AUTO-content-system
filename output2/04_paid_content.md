## ① ChatGPTに聞くだけ！最適な楽天カード決済ルートがわかるプロンプト

### この段階での課題

楽天カードって、実は使い方によってポイント還元率が全然違います。

楽天市場での購入、楽天ペイ経由、楽天Edy経由……
どれが一番お得なのか、調べようとしても情報が多すぎて迷子になりませんか？

**そこでAIに整理してもらいます。**

これができると…

- 毎月の決済ルートが明確になる
- 「損してた！」という後悔がなくなる
- 家族カードや法人カードの使い分けも迷わなくなる

### 使い方

以下のプロンプトをそのままChatGPTに貼り付けてください。
**[ ]の中だけ自分の情報に書き換えてOKです。**

---

**📋 プロンプト①：楽天カード最適決済ルート診断**

```
あなたは楽天ポイント最適化の専門家です。
以下の条件で、私が最もポイントを獲得できる決済ルートを教えてください。

【私の情報】
- 楽天カードの種類：[楽天カード / 楽天ゴールドカード / 楽天プレミアムカード]
- 楽天市場の月間利用額：[例：20,000円]
- 楽天ペイの利用有無：[あり / なし]
- 楽天Edyの利用有無：[あり / なし]
- 楽天銀行の口座有無：[あり / なし]
- SPU（スーパーポイントアッププログラム）の達成状況：[わからなければ「不明」でOK]

【聞きたいこと】
1. 今の私のSPU倍率の目安を教えてください
2. 楽天市場での買い物時、どの決済方法が一番ポイントが貯まりますか？
3. 楽天以外（コンビニ・スーパー等）での日常使いはどうするのが最適ですか？
4. 今すぐできる改善点を3つ教えてください

わかりやすく箇条書きで教えてください。
```

---

### ✅ POINT

AIは「あなたの状況」を踏まえて答えてくれます。
ネットの記事は一般論ですが、プロンプトに自分の情報を入れることで
**自分専用のアドバイス**が返ってきます。

### 📝 ワーク

プロンプト①を使ったら、AIの回答を見て「今すぐできる改善点」を1つだけ実行してみてください。
まず1つだけでOKです。

## ② お買い物マラソン前に使う「AI買い回り最適化テンプレ」

### この段階での課題

お買い物マラソンって、気合いを入れて買い物したのに
「あれ、思ったよりポイント少ないな…」ってなること、ありませんか？

原因の多くは **買い回り店舗の選び方** にあります。

何を、どの順番で、どの店舗で買うか。
これをAIに事前に組んでもらうだけで、獲得ポイントが変わります。

### 使い方

マラソン開始の**1〜2日前**にこのプロンプトを使うのがおすすめです。

---

**📋 プロンプト②：お買い物マラソン買い回り最適化**

```
あなたは楽天お買い物マラソンの攻略専門家です。
以下の条件で、私の買い回り計画を最適化してください。

【購入予定リスト】
（以下に買いたいものを列挙してください）
例：
- 日用品（洗剤・シャンプーなど）：3,000円分
- コーヒー豆：2,000円
- Amazonで買う予定だった本：1,500円
- 子供の上着：8,000円
- ふるさと納税（牛肉）：10,000円

【私の現在のSPU倍率】：[例：5倍 / わからなければ「不明」]
【目標ポイント数】：[例：3,000ポイント以上]

【聞きたいこと】
1. 上記リストを何店舗に分散して買うと最もお得ですか？
2. 1店舗あたりの最低購入金額（1,000円）を守るための調整案を教えてください
3. ポイントが最も付きやすい店舗の組み合わせを3パターン提案してください
4. 注意点・見落としがちな落とし穴があれば教えてください

具体的な金額と店舗名を含めて回答してください。
```

---

### ✅ POINT

マラソンは「合計金額」ではなく「店舗数」でポイント倍率が上がる仕組みです。
同じ金額を使うなら、1店舗にまとめるより**複数店舗に分散**するのが正解。
AIにリスト渡せば、その計算を瞬時にやってくれます。

### 📝 ワーク

次回のお買い物マラソン前に、プロンプト②に購入予定リストを入れてシミュレーションしてみてください。
「え、こっちの店舗で買ったほうがいいのか」という発見が必ずあります。

## ③ 5分でできる！AIを使った楽天ふるさと納税シミュレーション

### この段階での課題

ふるさと納税、「やったほうがいいのはわかってる。でも上限額がよくわからない」
そう言って毎年後回しにしている人がいます。

実際、控除上限額を調べるだけで公式サイトやシミュレーターを行ったり来たり……
面倒くさくなって結局やらない、というパターンが多いです。

**AIに聞けば5分で終わります。**

### 使い方

年収と家族構成がわかれば使えます。
（概算で十分。正確な額は税理士や公式サイトで確認してください）

---

**📋 プロンプト③：楽天ふるさと納税シミュレーション**

```
あなたはふるさと納税の専門家です。
以下の条件で、私のふるさと納税の活用プランを教えてください。

【私の情報】
- 給与収入（年収）：[例：450万円]
- 配偶者の有無：[あり（専業主婦・夫） / あり（共働き） / なし]
- 扶養家族：[例：子供1人（15歳未満）/ なし]
- 住宅ローン控除：[あり / なし]
- iDeCoの加入：[あり（月○○円） / なし]

【聞きたいこと】
1. 私のふるさと納税の控除上限額の目安を教えてください（概算でOK）
2. 楽天ふるさと納税を使う場合、最もポイントが貯まるタイミング・方法は？
3. 残り枠（[現在の月] 〜 12月末まで）での活用スケジュールを提案してください
4. 楽天お買い物マラソンと組み合わせる場合のコツを教えてください

注意点や確定申告・ワンストップ特例の使い分けも合わせて教えてください。
```

---

### ✅ POINT

ふるさと納税は**楽天お買い物マラソン期間中**に行うのが最もお得です。
10店舗買い回りの1店舗としてカウントされるので、
同じふるさと納税をするだけでポイント倍率がグッと上がります。

AIがこのタイミングを自動で提案してくれるので、計算いらずです。

### 📝 ワーク

プロンプト③を使って、今年の控除上限額と残り枠を確認してみてください。
「まだ○万円使えた！」という発見、あるはずです。
