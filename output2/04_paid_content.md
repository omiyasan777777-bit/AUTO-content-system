# 第2章：ニッチ特化戦略——「旅行全般」ではなく「ひとり温泉」で稼ぐ理由

## ■ なぜニッチ特化が最強の戦略なのか

検索エンジンでの上位表示は、**「検索意図との一致度」**で決まります。

「旅行 温泉 おすすめ」という広いキーワードで検索している人は、何を求めているのか曖昧です。観光旅行かもしれないし、記念日旅行かもしれない。一人かもしれないし、家族かもしれない。検索意図が広いほど、Googleはその意図に100%応えられる大きなメディアを優先します。

一方、**「ひとり温泉 女性 気まずくない 旅館」**で検索している人はどうでしょうか。

「ひとり旅で温泉に行きたいけど、他のグループ客の中で一人でいるのが気まずいかも……」という具体的な不安を持った人が検索しています。この検索意図に100%応える記事を書けるのは、実はそのニッチに特化した個人発信者の方が大手メディアより得意なのです。

### ニッチ特化の3つのメリット

**① 競合が少ない**
「旅行 おすすめ」は競合数十万ページ。「ひとり温泉 気まずくない」は競合数千ページ。上位表示のしやすさが段違いです。

**② 成約率が高い**
ニッチキーワードで来た読者は「具体的に悩んでいる人」です。「ひとり温泉 一人OK 旅館 関東」まで絞って検索している人は、すでに「ひとり温泉に行く」と決めており、あとは「どの宿に泊まるか」を探しているだけです。楽天トラベルへの誘導がそのまま予約につながりやすい状態です。

**③ 好きなことで書けるから続く**
旅行・温泉・ライブが好きなあなたにとって、ニッチに特化したジャンルは「得意分野」です。好きで詳しい領域の記事は、書いていて楽しく、読者の共感も得やすい。続けやすいのが最大の強みです。

# 第3章：5GPTsの役割と使い分け——分業させるから速くなる

「ChatGPTに記事を書かせているけど、なんかイマイチな文章しか出てこない」

## ■ GPT [1] 旅ターゲット設計くん

**役割:** 楽天トラベル向けに、誰に売るか・何を刺すかを設計するGPT

### なぜターゲット設計が必要か

記事を書く前にターゲットを明確にしないと、「誰にも刺さらない記事」になります。「旅行好きな人向け」では広すぎます。「30代・女性・ひとり温泉デビューしたいけど気まずさが不安」というレベルまで絞ることで、読んだ人が「これ、私のことだ！」と感じる記事になります。

### コピペ用入力プロンプト例

**プロンプト例1（ひとり温泉ニッチ）:**
```
私は楽天トラベルアフィリエイトで「ひとり温泉」ジャンルの記事を書きます。
以下を設計してください：
・メインターゲット（年齢・性別・状況・悩み）
・このターゲットが温泉に行けていない理由（3つ）
・このターゲットが宿を選ぶときに最も重視すること（3つ）
・このターゲットに刺さる記事タイトルの方向性（3パターン）
```

**プロンプト例2（ライブ遠征ニッチ）:**
```
私は楽天トラベルアフィリエイトで「ライブ遠征ホテル」ジャンルの記事を書きます。
以下を設計してください：
・メインターゲット（年齢・性別・遠征スタイル・悩み）
・このターゲットがホテル選びで困っていること（3つ）
・このターゲットが宿に求める条件TOP5
・このターゲットに響くアフィリエイト記事の訴求軸（3パターン）
```

**プロンプト例3（カスタマイズ用）:**
```
私は楽天トラベルアフィリエイトで「[ニッチジャンル]」の記事を書きます。
ターゲットは「[大まかなイメージ]」です。
このターゲットをさらに具体化して、ペルソナを1人設計してください。
・名前・年齢・職業・旅行スタイル
・最近悩んでいること
・理想の旅行のイメージ
・楽天トラベルで宿を探すときの検索ワード（3つ）
```

### ✅ POINT

旅ターゲット設計くんを使うコツは、**「大まかなジャンルと自分のイメージ」を渡すこと**です。白紙から「ターゲットを考えて」と頼むより、「ひとり温泉ジャンルで」「ライブ遠征者向けで」という方向性を示すと、精度の高いターゲット像が出てきます。

### 📝 ワーク

自分が攻めるニッチジャンルを選んで、上のプロンプト例1〜3のいずれかをGPTに入力してみてください。出てきたターゲット像を手元にメモしておきます（次の章の記事作成で使います）。

## ■ GPT [2] 楽天トラベル記事職人

**役割:** 旅行キーワードから、SEO記事構成・本文・CTAまで作るGPT

## ■ STEP 3: 記事×SNSの同時展開フローを確立する

**記事→SNS同時展開フローシート:**

```

① 記事を最終確認・整形（10分）
    ↓
② SNS投稿メーカーで投稿文を5本作成（5分）
    ↓
③ 記事を公開する
    ↓
④ X・Instagramに投稿文を1本ずつ投稿する（5分）
    ↓
⑤ 残り4本は翌日以降に1日1本ずつ予約投稿

【所要時間の目安】
記事公開 + SNS1本目の投稿: 約20分
残り4本: 4日間に分けて各5分
```

### ✅ POINT

**1記事から5日間投稿できる**のがこのフローの強みです。記事を1本書くたびに、5日分のSNS投稿素材が手に入ります。週1本記事を書けば、毎日投稿し続けることができます。
