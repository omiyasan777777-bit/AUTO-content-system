# 有料①｜AIで楽天商品レビュー記事を30分で量産する手順

## まず「型」を理解する

楽天アフィリエイトで稼ぐ記事には「型」があります。

この型に沿ってAIに指示を出すだけで、
読者が思わずリンクをクリックしたくなる記事が完成します。

**稼げるレビュー記事の5段構成：**

```
① 悩みの共感（読者の「あるある」を言語化）
② 商品紹介（スペック＋特徴）
③ 実際に使った感想・比較（リアリティ）
④ こんな人におすすめ（ターゲット明示）
⑤ CTA（楽天リンク＋ポイント訴求）
```

---

## 30分で記事を量産するSTEP

### STEP 1：商品選定（5分）

楽天アフィリエイトの管理画面 or 楽天市場で
**レビュー件数100件以上・評価4.0以上** の商品を選ぶ。

これが基準。レビューが多い＝需要が実証されている商品です。

---

### STEP 2：AIへの指示文（プロンプト）を入力（3分）

以下のプロンプトをそのままコピーして使ってください：

```
【楽天アフィリエイト記事生成プロンプト・基本版】

あなたはアフィリエイトライターのプロです。
以下の商品について、楽天アフィリエイト用のレビュー記事を書いてください。

商品名：[ここに商品名を入力]
ターゲット読者：[例：30代主婦、一人暮らしの男性など]
文字数：1,500〜2,000字

記事の構成：
1. 読者の悩みへの共感（200字）
2. 商品の概要・特徴（400字）
3. 使ってみてよかった点3つ（各150字）
4. 気になる点（正直に1つ）
5. こんな人におすすめ（3タイプ）
6. まとめ＋購入を後押しするCTA（楽天ポイント訴求含む）

口調：親しみやすく、でも信頼感のある語り口で。
専門用語は使わず、初めて商品を検討する人にも伝わる言葉で書いてください。
```

---

### STEP 3：AIの出力を確認・肉付け（10分）

AIが生成した文章を読んで、以下の3点だけ確認します：

- **実体験っぽいエピソードを1つ追加**（「先週使ってみたら〜」など）
- **数字を入れる**（「3回使って感じたのが〜」「2週間で〜」）
- **楽天リンクの前後に「ポイント◯倍！今すぐチェック↓」の文言を追加**

この3点だけで、AIの文章が「人間が書いたリアルなレビュー」に変わります。

---

### STEP 4：記事をブログ or noteに投稿（7分）

WordPressやnoteに貼り付け、
楽天アフィリエイトの商品リンクを適切な場所（記事中盤＋末尾）に設置。

タイトルは必ず以下の型を使う：

```
【[商品名]レビュー】[ターゲット]が[期間/回数]使った正直な感想
例：【バルミューダ扇風機レビュー】一人暮らし歴3年の私が夏中使った正直な感想
```

---

### STEP 5：1ヶ月の量産スケジュール

| 週 | 目標記事数 | 積み上げ合計 |
|----|----------|------------|
| 1週目 | 5記事 | 5記事 |
| 2週目 | 5記事 | 10記事 |
| 3週目 | 5記事 | 15記事 |
| 4週目 | 5記事 | 20記事 |

月20記事で「記事資産」が積み上がります。
半年後には120記事。
これが月1〜3万円の土台になります。

# 有料②｜クリック率が上がる商品紹介文のプロンプト設計法

## 「読んだけど買わなかった」の原因

記事を読んでもリンクをクリックしてもらえない。

その最大の原因は「紹介文が商品説明になっているから」です。

読者はスペックを知りたいのではなく、
**「自分の悩みが解決するかどうか」** を知りたい。

この視点をプロンプトに組み込むだけで、クリック率が劇的に変わります。

---

## クリック率2倍のプロンプト設計法

### 【型1】悩み解決型プロンプト

```
【クリック率重視・商品紹介文プロンプト】

商品名：[商品名]
読者の悩み：[具体的な悩みを1文で]

以下の順番で商品紹介文を200字で書いてください：

1. 読者の悩みをそのまま言語化する一文（「〜で悩んでいませんか？」）
2. この商品がその悩みを解決する理由を一言で
3. 具体的な使用シーンを絵が浮かぶように描写
4. 「今すぐ楽天でチェック↓（ポイント還元あり）」で締める

煽りすぎず、でもワクワク感が伝わるトーンで。
```

---

### 【型2】比較訴求型プロンプト

複数の似た商品を比較するシチュエーションに使います：

```
【比較訴求型・商品紹介文プロンプト】

商品A：[商品名A]
商品B：[商品名B]

以下の形式で比較紹介文を300字で書いてください：

・どんな人に商品Aが向いているか（具体的なシーン）
・どんな人に商品Bが向いているか（具体的なシーン）
・「迷ったらこちら」としてどちらかを推薦し、理由を一言で

断定口調で書き、「どちらも楽天でポイント還元中↓」で締める。
```

---

### 【型3】季節・タイミング訴求型プロンプト

楽天スーパーセールや季節イベントに合わせた文章：

```
【季節・タイミング訴求型プロンプト】

商品名：[商品名]
タイミング：[例：梅雨前、楽天スーパーセール、母の日前など]

「今買う理由」が明確に伝わる商品紹介文を200字で書いてください。

構成：
・今この時期に買うべき理由（タイミングの必然性）
・商品の特徴1つ（最も刺さるポイントのみ）
・「このタイミングを逃すともったいない」という軽い緊急性
・楽天リンクへのCTA

押し売り感ゼロで、読者の背中を自然に押すトーンで。
```

---

## クリック率を上げる「魔法の一言」リスト

以下のフレーズをCTAの直前に置くだけでクリック率が上がります：

```
✅「楽天ポイントで実質○○円引きになります」
✅「レビュー○○件超えの本物の人気商品です」
✅「在庫が少ないので気になる方はお早めに」
✅「まとめ買いするとポイント還元がお得です」
✅「楽天カード決済でさらにポイント2倍」
```

これらは**事実ベースの訴求**なので、
過度な煽りにならずに購買意欲を引き出せます。

# 有料③｜稼げるジャンル選定をAIに任せるリサーチフロー

## ジャンル選びを間違えると何をやっても稼げない

楽天アフィリエイトで稼げない人の9割が、
**競合が強すぎるジャンルに参入している** という問題を抱えています。

正しいジャンル選定の考え方：

```
稼げるジャンル = 需要が高い × 単価が高い × 競合が弱い
```

この3つが揃うジャンルを探すのが、最初にやるべきことです。

---

## AIを使ったジャンルリサーチフロー

### STEP 1：AIに「穴場ジャンル」を提案させる

```
【ジャンルリサーチプロンプト・Step1】

楽天アフィリエイトで稼ぎやすい「穴場ジャンル」を教えてください。

条件：
・商品単価が3,000円以上
・季節を問わず需要がある（もしくは複数シーズンで需要がある）
・大手メディアやプロブロガーが少ない（競合が弱い）
・楽天市場での取扱商品が豊富

10ジャンルをリストアップし、各ジャンルの「なぜ稼ぎやすいか」を1文で説明してください。
```

---

### STEP 2：選んだジャンルの深掘りリサーチ

```
【ジャンルリサーチプロンプト・Step2】

ジャンル：[Step1で選んだジャンル]

このジャンルについて以下を教えてください：

1. このジャンルで検索する人の「典型的な悩み3つ」
2. 楽天市場で人気が高い商品タイプ（3種類）
3. このジャンルで記事を書く際の「タイトルの型」3パターン
4. 競合記事と差別化できる切り口のアイデア2つ

初心者アフィリエイターが参入しやすい角度で答えてください。
```

---

### STEP 3：記事ネタを量産させる

```
【記事ネタ量産プロンプト】

ジャンル：[選んだジャンル]

このジャンルで書ける記事タイトルを20本提案してください。

条件：
・SEOを意識したタイトル（検索されそうなキーワードを含む）
・「〇〇おすすめ5選」「〇〇比較」「〇〇レビュー」などの形式を含める
・初心者〜上級者まで幅広い検索意図をカバーする
・楽天市場の商品にリンクしやすいテーマを優先する

各タイトルに「想定読者像」を1行で添えてください。
```

## SNS→アフィリエイトで大事な1つのルール

SNSで売り込みすぎると**フォロワーが離れます。**

黄金比率はこれ：

```
有益情報投稿 ：アフィリエイト誘導投稿 = 7：3
```

7割は「純粋に役立つ情報」を発信。
3割でさりげなく商品を紹介する。

この比率を守るだけで、フォロワーの信頼が積み上がり、
アフィリエイトリンクのクリック率が上がっていきます。

## 1万円達成者が語る「本当に大事だったこと」

実際に月1万円を達成した人たちに共通していたことを3つにまとめます：

**① 最初の30記事は「量>質」と割り切った**

**② 「稼げているジャンル」を見つけたらそこに集中した**
広く浅くより、1つのジャンルを深掘りした方が圧倒的に収益が上がった。

**③ 楽天スーパーセールのタイミングを逃さなかった**
楽天のセール期間は通常の3〜5倍クリックされる。
セール前に関連記事を量産しておくことで、一気に収益が跳ね上がった。
