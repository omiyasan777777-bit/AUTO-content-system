## プロンプト①：ChatGPTで「最適な楽天カード決済ルート」を設計する

### この章での課題

楽天の決済でポイントを最大化するには、
**どのカードを・どのサービスで・どの順番で使うか**の組み合わせが重要です。

でも、楽天のサービスは楽天カード・楽天Pay・楽天キャッシュなど複数あり、
「どれをどこで使えばいい？」と混乱している人が大半です。

放置すると、毎月の決済でポイントの取りこぼしが積み重なり、
1年間で数千〜数万円の損失になります。

でも、この章を読めば——

**ChatGPTに自分の状況を入力するだけで、最適な決済ルートを設計してもらえます。**
一度設定すれば、あとは日常的に使い続けるだけ。

[IMAGE: ChatGPTの画面にプロンプトを入力し、楽天の最適決済ルート図が生成されているイラスト。画面左にスマホ、右に矢印付きフロー図。ミニマルなUIデザイン]

---

### 具体的なやり方

**STEP 1：以下のプロンプトをそのままChatGPTにコピペする**

```
あなたは楽天経済圏のエキスパートです。
私の状況に合わせて、楽天ポイントを最大化する決済ルートを設計してください。

【私の情報】
- 楽天カード（通常）：あり / なし
- 楽天プレミアムカード：あり / なし
- 楽天銀行口座（引き落とし設定済み）：あり / なし
- 楽天証券口座：あり / なし
- 楽天モバイル契約：あり / なし
- 楽天Pay：使っている / 使っていない
- 月の楽天市場での買い物額：約___円
- 月のリアル店舗（コンビニ・スーパー等）での支払い額：約___円
- 楽天ガス・電気：契約あり / なし

上記をもとに以下を教えてください：
1. 楽天市場での最適な決済方法（カード種別・支払い方法）
2. リアル店舗での最適な支払い方法
3. 今すぐできるSPUアップ施策（コスト0円のものを優先）

```

**STEP 2：[ ] の中を自分の情報に書き換えて送信**

例えばこんな感じで返ってきます↓

```
【あなたの場合の最適決済ルート】

楽天市場：
→ 楽天カード + 楽天銀行引き落とし + 楽天Payで決済
→ 現在のSPU倍率：5.5倍
→ 月3万円の買い物で推定1,650ポイント獲得

→ 楽天Payを使用（楽天キャッシュにチャージして支払い）
→ 1.5%還元 → 月2万円で300ポイント獲得

今すぐできる追加施策（無料）：
→ 楽天証券でポイント投資設定（+0.5倍）
→ 楽天銀行で給与受け取り設定（+1倍）
→ 月間合計：約2,200ポイント → 現在より約700ポイントアップ
```

✅ POINT
> SPU（スーパーポイントアッププログラム）は、楽天の各サービスを使うほど楽天市場での買い物倍率が上がる仕組み。最大で通常の16.5倍になります。プロンプトを使えば、自分が今どこまで達成できているかも一覧できます。

**STEP 3：提案された施策を一つずつ実行する**

## プロンプト②：お買い物マラソン「最適買い回りプラン」を自動生成する

### この章での課題

お買い物マラソンは知っている人も多いですが、
**「どこで何を買えば効率が良いか」を事前に計画している人はほとんどいません。**

行き当たりばったりで買い回ると、
- ポイント倍率を最大化するための店舗数が足りなかった
- 「欲しくないものを無理に買った」で結局損した

そんな失敗をしてしまいます。

でも計画さえあれば、お買い物マラソンは**必要なものを買いながらポイントを最大化**できる最強イベントになります。

[IMAGE: お買い物マラソンのカレンダーと、購入計画リストが並んだイラスト。チェックボックス付きのリスト形式。各店舗のジャンル（日用品・食品・美容）のアイコン付き]

---

### 具体的なやり方

**STEP 1：マラソン前に以下のプロンプトをChatGPTに送る**

```
楽天お買い物マラソンの買い回り計画を作ってください。

【前提条件】
- 目標店舗数：___店舗（最大10店舗）
- 合計予算：___円
- マラソン期間：___日間

【普段の楽天での購入カテゴリー（あてはまるもの全部）】
□ 日用品（洗剤・トイレットペーパー等）
□ 食品・飲料
□ 美容・コスメ
□ ペット用品
□ 本・雑誌
□ 家電
□ 衣類・ファッション
□ ふるさと納税
□ サプリ・健康食品
□ スポーツ用品

【買いたいもの・近々必要なもの（自由記入）】
___

以下を出力してください：
1. 最適な買い回り計画（店舗×商品×価格の一覧）
2. ポイント倍率の試算（何店舗でいくらのポイントか）
3. 「消耗品を定期購入設定に変えてまとめる」おすすめ店舗
4. 「ふるさと納税」をマラソン中に組み込むべきか否かの判断
```

**STEP 2：返ってきた計画に沿って購入する**

ChatGPTは「まとめ買いすべき店」と「分散させるべき商品」を自動で仕分けしてくれます。

実際にプロンプトを使った結果の例：

```
【買い回り計画（予算3万円・8店舗目標）】

店舗①：ふるさと納税（10,000円）
  → ポイント倍率の恩恵が大きい。最低でも1店舗カウント

店舗②：日用品店（4,000円）
  → トイレットペーパー・洗剤を3ヶ月分まとめ買い

店舗③：食品・お取り寄せ（3,000円）
  → 普段の食費として消費できるので無駄ゼロ

店舗④：楽天ブックス（1,000円以上）
  → 本・Kindle・CD何でもOK。1,000円以上で1店舗カウント

店舗⑤：ペット用品（3,000円）
  → フードのまとめ買い推奨

店舗⑥：美容・コスメ（3,000円）
  → シャンプー・スキンケアを定期購入設定に切り替え

店舗⑦：サプリ（2,000円）
  → プロテインやビタミン剤を3ヶ月分

店舗⑧：スポーツ・アウトドア（2,000円）
  → 消耗品（ランニングソックス等）で調整

合計：28,000円 / 8店舗
推定ポイント倍率：8倍 → 約2,240ポイント獲得
```

✅ POINT
> お買い物マラソンは1,000円以上の注文が「1店舗」のカウント対象になります（一部例外あり）。1,000円ちょっとの消耗品を複数の店舗に分散させるのが効率的です。プロンプトでこの分散計画を自動生成してもらいましょう。

## プロンプト③：AIで楽天ふるさと納税を5分でシミュレーションする

### この章での課題

ふるさと納税は「節税になる」とわかっていても、
**「自分はいくらまで寄附できるのか」が計算できず、手が止まっている人が多い。**

控除上限額を超えた寄附は、節税効果なしの"ただの寄附"になってしまいます。

でも「計算が面倒だから後回し」にしていると、
年末になって「今年もふるさと納税やり忘れた…」という事態になりがちです。

**ChatGPTに聞けば、5分でシミュレーションが完了します。**

[IMAGE: 電卓とふるさと納税の返礼品（牛肉・カニ・フルーツ）が並ぶイラスト。計算式と矢印で「2,000円の自己負担→3万円分の返礼品」の関係がわかるデザイン。温かみのある配色]

---

### 具体的なやり方

**STEP 1：以下のプロンプトをChatGPTに送る**

```
楽天ふるさと納税のシミュレーションをしてください。

【私の情報】
- 給与所得（年収）：約___万円
- 家族構成：
  - 配偶者：あり（専業主婦/夫 or 共働き）/ なし
  - 子ども：___人（年齢：___, ___, ___）
  - 扶養親族（高齢の親など）：あり / なし
- 社会保険料控除などの特殊な控除：あり（内容：___）/ なし
- 住宅ローン控除：あり / なし

以下を教えてください：
1. おおよその「ふるさと納税の控除上限額（自己負担2,000円の目安）」
2. 楽天ふるさと納税でポイントを最大化するための寄附タイミング
3. お買い物マラソン中にふるさと納税をすべきかどうかの判断基準
4. 返礼品のおすすめカテゴリー（コスパ重視）
```

**STEP 2：返ってきた上限額を基準に寄附額を決める**

ChatGPTはあくまでも概算を出してくれます（正確な計算は税理士や総務省のシミュレーターで）。
でも「だいたいいくらまで大丈夫か」の目安を知るだけで、行動に移れます。

実際の出力例：

```
【シミュレーション結果（概算）】

年収500万円・独身の場合：
→ 控除上限額の目安：約61,000円
→ 自己負担2,000円で約59,000円分の寄附が節税対象

楽天でのポイント最大化タイミング：
→ お買い物マラソン中に寄附すると+ポイント倍率の恩恵あり
→ さらに「5と0のつく日」に寄附するとポイント+2倍

返礼品コスパランキング（筆者おすすめ）：
1. お米（返礼率30%前後・毎月消費できる）
2. 牛肉・豚肉（冷凍で長期保存可能）
3. 日本酒・ビール（嗜好品として楽しめる）
4. 日用品・ティッシュ・トイレットペーパー（生活コスト削減）

寄附のおすすめスケジュール：
→ 9〜11月：主要な寄附を完了
→ 12月中旬まで：残額を調整
```

✅ POINT
> 楽天ふるさと納税は「楽天スーパーSALE」「お買い物マラソン」期間中に寄附すると、通常の楽天市場のポイント倍率が乗ります。大きな金額を寄附するからこそ、ポイント倍率の差が数百〜数千円の差になります。

**STEP 3：返礼品を選んで寄附する**

上限額が把握できたら、あとは楽天ふるさと納税のサイトで返礼品を選ぶだけ。
「コスパ」「好み」「実用性」の3軸でChatGPTに選んでもらうこともできます。
