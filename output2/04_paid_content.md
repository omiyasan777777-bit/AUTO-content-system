## STEP2-2: 魂の宿った占い記事の生成手順

テーマが決まったら、いよいよmanusで生成します。

**フェーズ1: 「骨格」を生成する**

まず、記事の骨格（構成と主要メッセージ）をmanusに出力させます。

使うプロンプト：
```
[先ほどのジャンル別プロンプト] + 以下の指示を追加：

```

骨格を確認し、「ここを変えたい」という部分があればこの段階で修正します。
骨格の修正はコストが低く、後から直すより効率的です。

**フェーズ2: 「肉付け」をして本文生成する**

骨格を確認したら、本文の生成を指示します。

使うプロンプト：
```
「この構成に基づいて、先ほどの占い師スタイルで全文を生成してください。
特に[自分が強調したいポイント]については、より丁寧に描写してください。」
```

このフェーズで生成された本文が「叩き台」になります。

✅ POINT: フェーズ1で骨格確認をすることが非常に重要です。「なんか違う」という感覚をここで調整することで、フェーズ2の完成度が飛躍的に上がります。

## STEP2-3: AI生成を「あなたらしさ」に変える編集3ステップ

生成された文章に、最後の仕上げをします。

この3ステップを「魔法の仕上げ」と呼んでいます。

---

**✨ 編集STEP1: 自分の体験・実例を1つ挿入する**

AI生成の文章は「一般的な正解」を書きます。
でも読者が本当に求めているのは「リアルな実例」です。

どこかの段落に、あなた自身の経験や観察を1つ加えてください。

例：
- 「私がこのカードを引いた時、実際にこんな変化がありました」
- 「過去に蠍座の方をお見立てした際、このパターンが多かったです」
- 「この数字を持つ方に、よくこんな共通点があると感じています」

これだけで、記事に「人間味」が宿ります。

---

**✨ 編集STEP2: 読者の気持ちを代弁する一文を追加する**

AI生成の文章は「答え」を与えます。
でも感情的に刺さる文章は「読者の代わりに感情を言語化する」ものです。

各セクションの冒頭に、こんな一文を加えてみてください：
- 「こんな気持ち、ありませんか？」
- 「最近、こんな感覚を覚えることはないでしょうか」
- 「きっとあなたも感じているはずです。○○という感覚を」

この一文が「この記事は私のために書かれた」という感覚を生みます。

---

**✨ 編集STEP3: 最後に「あなたへのメッセージ」を1段落追加する**

記事の最後に、あなた自身の言葉でメッセージを書き加えます。

AIが生成した文章の後に：
```
[あなたの名前・ペンネーム]より

○○（今回のテーマ）について、私が最も伝えたかったことは、
[あなたの本音の言葉で1〜3文]

どうか、この言葉があなたの背中を押せますように。
```

たったこれだけで、記事全体が「あなた発のコンテンツ」になります。

## STEP2-4: スマホで読まれるnote記事フォーマット

編集まで終わったら、noteに合わせたフォーマットに整えます。

noteの読者の多くはスマートフォンで読んでいます。
スマホで読みやすい記事には、いくつかのルールがあります。

---

**📱 スマホ最適化チェックリスト**

**1段落の行数を4行以内に収める**
長い段落はスマホで「壁」に見えます。
4行を超えたら改行を入れてください。

**見出しを適切に使う**
h2（##）とh3（###）を使い分けて、記事に「ポイント」を作りましょう。
スマホで目次を見た時に「どんな記事か」がわかる見出しが理想です。

**箇条書きで整理する**
3つ以上の項目を並べる時は、必ず箇条書きにします。
文章の中に埋め込むより、圧倒的に読みやすくなります。

**装飾は最小限に**
強調したい言葉だけを**太字**にします。
全体を太字にすると、逆に読みにくくなります。

**有料と無料の「境界」を意識する**
有料部分の最初に「ここが一番重要なメッセージです」という
ハードルを感じさせる境界を作ると、購入への動機が高まります。

# 第4章：noteで売れる仕組みを作る

「記事を書いたけど、売れない」

これが、占い副業で挫折する人の最も多いパターンです。

コンテンツが良くても、売れない記事には理由があります。
逆に言えば、その理由を一つひとつ解消すれば、売れるようになります。

この章では、
「良いコンテンツを、確実に売れる仕組みに変える方法」を解説します。

## STEP3-1: 1,000〜3,000円で売れる価格設計法

最初に多くの人が悩むのが価格設定です。

「安すぎると安っぽく見える」
「高すぎると買ってもらえない」

どちらも正しい感覚です。では、どうすればいいか？

**答えは「コンテンツの量と価値」で決める、ではありません。**

**「読者が感じる価値」で決めるのです。**

---

**占い有料note の価格設計フレーム**

**980円（フロントエンド・入門層向け）**
- ターゲット: 占いnoteを初めて買う人
- 内容: 単発の月次運勢 / 簡単なジャンル別鑑定文
- 目的: まず買ってもらい、ファンになってもらう最初の一歩

**1,480円（スタンダード・コア層向け）**
- ターゲット: 占いに興味があり、少し踏み込んだ内容を求める人
- 内容: 特定のテーマの詳細鑑定 / 実践的なノウハウ
- 目的: ファンを「リピーター」に育てる

**1,980円（プレミアム・熱心な読者向け）**
- ターゲット: 占いを本気で活用したい、深い内容を求める人
- 内容: 総合的な年間運勢 / テーマを深掘りした完全ガイド
- 目的: 高単価コンテンツで収益効率を上げる

**最初の1本は980〜1,480円をおすすめします。**

高い商品ほど「実績がある販売者」から買いたいという心理があります。
最初は低価格で実績を作り、徐々に価格を上げていくのが正解です。

---

**✅ POINT: 価格を上げるタイミング**

以下のサインが来たら、価格を上げるタイミングです：
- スキが20を超え始めた
- 購入後のコメントが増えた
- 「続編が読みたい」という声がついた

価格を上げる時は、「今後〇〇円に値上げします」と事前告知すると、
「今のうちに買おう」という購買動機が生まれます。

## STEP3-2: 読者を「買いたい」に変える無料部分の書き方

noteの有料記事は「無料部分」と「有料部分」に分かれています。

**読者は無料部分を読んで、買うかどうかを決めます。**

つまり、コンテンツの品質がどれだけ高くても、
無料部分が魅力的でなければ、有料部分は永遠に読まれません。

ここでは「読者を買いたいに変える無料部分」の5ステップテンプレートを公開します。

---

**🔮 無料部分の5ステップテンプレート**

**STEP1: 読者の悩みを代弁する（共感）**

```
[ターゲットの悩みを1〜2文で代弁する]

こんな気持ち、ありませんか？
```

例：
```
「最近、何をやってもうまくいかない気がする」
「この流れ、いつまで続くんだろう」
```

---

**STEP2: 何の記事かを明示する（内容の予告）**

```
この記事は、[ターゲット]のための[内容]です。

この記事を読むと：
・[得られる情報1]
・[得られる情報2]
・[得られる情報3]
がわかります。
```

---

**STEP3: あなたの信頼性を示す（根拠）**

```
[あなたの自己紹介 / 占い歴 / 活動実績を1〜3文で]
```

最初は実績がなくても大丈夫です。
「占い歴○年」「タロット資格あり」「日々占いを実践している」
という事実を書くだけで、一定の信頼感が生まれます。

---

**STEP4: 有料部分の魅力をチラ見せする（期待感）**

```
有料部分では、さらに詳しく解説しています。
特に[最も魅力的なコンテンツ]については、
[どんな視点で / どれほど詳しく]扱っています。
```

「続きが気になる」という状態を作ることが目的です。

---

**STEP5: 購入を後押しする一文（CTA）**

```
[今の自分の状況に合っている人向けに、背中を押す一文]
```

例：
```
もし今、変化を感じたい気持ちがあるなら、
この記事がその一歩になれると思います。
```

## STEP3-3: フォロワーゼロでも売れる占いSNS投稿術

「フォロワーがいないと売れない」

**正確には「フォロワーが少なくても、正しい人に届けば売れます」。**

占い副業でSNSを使う目的は、フォロワーを増やすことではありません。
**「あなたの記事を買いそうな人」に届けることです。**

ここでは、フォロワーが少なくても効果的な3つのSNS活用法を紹介します。

## STEP4-1: 月3万円を確実に達成する量産スケジュール

まず、具体的な数字から逆算します。

**月3万円達成への逆算**

目標: 月3万円（1,980円×15冊）

では、15冊売るためには何が必要か？

noteで有料記事を継続的に売るためのデータとして、
「月に公開する記事数 × 各記事の平均販売数」が重要です。

初月〜3ヶ月目の現実的な目標：
- 月に2〜3本の新記事を公開する
- 1本あたり5〜10冊の販売を目指す
- 3ヶ月で計8〜10本の記事が蓄積 → 蓄積記事が毎月売れ続ける

**重要なのは「積み上げる」こと。**
最初の1ヶ月は月5冊しか売れなくても、
3ヶ月後には「先月公開した記事 + 今月公開した記事 + 先々月の記事」が
全部売れるようになります。

これが「ストック型収益」の力です。

---

**📅 月次・週次スケジュールテンプレート**

以下は「週2時間」で月3万円を目指すスケジュールです。

**毎週（必須作業）**

| 曜日 | 作業 | 時間 |
|------|------|------|
| 月曜日 | 来週公開する記事のテーマ選定 | 15分 |
| 水曜日 | manusで記事生成 + 編集3ステップ | 30分 |
| 金曜日 | noteフォーマット整え + 公開 | 15分 |
| 土曜日 | SNS投稿（X/Instagram更新） | 15分 |

**月1回（推奨作業）**

| タイミング | 作業 | 時間 |
|-----------|------|------|
| 月初 | 先月の売上分析 + テーマ選定（月分） | 30分 |
| 月中 | SNS反応確認 + フォロワーへの感謝ポスト | 15分 |
| 月末 | 翌月の記事計画立案 | 30分 |

合計: 週1時間15分〜2時間 + 月次作業1時間15分

これだけで、月3万円を目指せます。

## STEP4-2: 売れる記事を「再現」するPDCA術

月に数本の記事を公開していると、やがて気づきます。

「この記事は売れたのに、こっちは売れない」

この差に気づき、分析し、再現することが収益拡大の核心です。

---

**売上データの読み方**

noteの「クリエイターページ」から、以下のデータを確認できます：

- 各記事の閲覧数
- 各記事のスキ数
- 各記事の購入数（購入数÷閲覧数 = 購入率）

**重要なのは「購入率」です。**

閲覧数が少なくても購入率が高い記事は、
タイトルや内容がターゲットにドンピシャで刺さっています。

逆に閲覧数は多いのに購入率が低い記事は、
「タイトルで期待させているのに、無料部分で失望させている」
という可能性があります。

---

**売れた記事の「要素分解」チェックリスト**

売れた記事を振り返り、以下を確認します：

```
□ テーマは何だった？（恋愛運/仕事運/年間運勢/etc）
□ ターゲットは誰だった？（星座/年代/状況）
□ タイトルの構造は？（数字あり/逆説あり/etc）
□ 無料部分の長さは？（何字程度）
□ 価格は？
□ SNSで告知したか？どのプラットフォームで？
□ 公開した時期・タイミングは？（月初/イベント前後/etc）
```

売れた記事の共通パターンが見えてきたら、
それを「次の記事のテンプレート」として活用します。

これが「売れる記事の再現」です。

---

**✅ POINT: 季節・イベントを活用する**

占いコンテンツは「タイムリーなテーマ」と非常に相性が良いです。

以下のイベントに合わせて記事を準備すると、
需要が高まるタイミングで集客できます：

- 1月: 新年運勢、今年の全体運
- 2月: バレンタイン、恋愛運
- 3月: 年度末、転機・変化のテーマ
- 4月: 新年度、新しいスタート
- 7月: 夏の恋愛運、上半期の振り返り
- 10月: 下半期の運勢
- 12月: 今年の総括、来年への準備

イベント2週間前から準備・告知を始めると、ピーク時に売れやすくなります。
