# 【レビュー特典④】BAN回避マニュアル！Threads新規アカウントを安全に作る方法

---

## ✅ はじめに：なぜThreadsはBANされやすいのか？

Threadsは恋愛ジャンルの発信者が特に多く、同時にアカウントBANが頻繁に起きるSNSです。

**BANされやすい主な理由：**
- 恋愛・性的示唆のある表現がアルゴリズムに引っかかりやすい
- 新規アカウントは「bot」「スパム」と誤判定されやすい
- 外部リンク（noteリンク等）の貼り方が不適切だとBANフラグが立つ
- 大量フォロー・大量いいねを短時間でやると制限がかかる

BANはある日突然「このアカウントはご利用いただけません」と表示されて終わります。
事前に対策しておくことが重要です。

---

## ◆STEP1：新規アカウント作成前の準備

### ✅ Instagramアカウントを用意する

ThreadsはInstagramと連携して動きます。
**新しいInstagramアカウントを作成してから**Threadsアカウントを作るのが基本です。

```
【推奨手順】
1. 新しいメールアドレス or 電話番号を用意する
2. そのアカウントでInstagramに新規登録
3. Instagramのプロフィールを設定（アイコン・プロフィール文）
4. Instagramで3〜5日間、普通のアカウントとして行動する
5. その後Threadsに連携して作成する
```

> 💡 既存のInstagramからThreadsを作ると、フォロワーが一気に来て初速が出る場合がありますが、発信ジャンルが違う場合はリセット目的で新規作成がおすすめです。

---

### ✅ スマートフォンの環境を整える

複数のアカウントを運用する場合、同じデバイスから複数の新規アカウントを短期間で作成するとBANリスクが高まります。

```
【推奨環境】
・アカウントごとに別のメールアドレス or 電話番号を使用
・可能であれば別のWi-Fi環境（スマホのデータ通信）で作成
・複数アカウントを1つのスマホで管理する場合はアプリの切り替え機能を使う
```

---

## ◆STEP2：アカウント作成直後の行動ルール（最重要）

### ❌ 絶対にやってはいけないこと

| 行動 | 理由 |
|------|------|
| 作成直後に大量投稿（5投稿以上） | スパム判定されBANの原因になる |
| 作成直後に外部リンク（noteのURL）を投稿 | スパムフラグが立ちやすい |
| 短時間での大量フォロー・いいね | botと判定されシャドウバンまたはBAN |
| 作成初日に他アカウントをフォロー50件以上 | 凍結の引き金になる |
| 他アカウントのコメント欄でリンクを貼る | スパム報告を受けやすい |

---

### ✅ 作成直後の正しい行動（最初の1週間）

**【0〜12時間】完全に何もしない**
アカウントを作成したら、12〜15時間は投稿もいいねもしない。
アルゴリズムが新規アカウントを認識して落ち着くまで待つ。

**【1〜3日目】一般人のように動く**
```
・投稿は1日1〜2本まで
・リンクは一切貼らない
・発信ジャンルに関係する投稿をする（恋愛・感情・日常系）
・同ジャンルの投稿を1時間に3〜5いいね程度（1日合計30いいねまでを目安に）
```

**【4〜7日目】ジャンル認知を進める**
```
・同ジャンルアカウントのフォロー（1日5〜10件まで）
・投稿を1日2〜3本に増やす
・おすすめ欄に恋愛系コンテンツが出てきたらジャンル認知が進んでいる証拠
・まだリンクは貼らない
```

**【7日目以降】noteリンクを解禁**
```
・インプレッションが1投稿あたり100前後出るようになったら解禁
・リンクは必ずリプ欄に貼る（本文への直貼りはNG）
・1日のリンク付き投稿は1〜2本まで
```

## ◆STEP3：日常運用でBANを回避するルール

### ✅ 投稿のルール

```
・1日の投稿数：3〜5投稿が上限の目安
・投稿間隔：最低1時間以上空ける
・同じ内容の投稿を連続して出さない
・外部リンクは必ずリプ欄に貼る
・センシティブな単語（性的表現・過激な言葉）は避ける
```

---

### ✅ センシティブワードに注意する

恋愛ジャンルでBANの原因になりやすい表現の例：

```
【直接的すぎてNG】
・「体の関係」
・「セックス」「エッチ」
・「不倫の具体的な方法」
・「体目当て」「都合のいい女」（直接的すぎる表現）

【代わりに使う表現】
・「身体の距離」「大人の関係」「夜の時間」
・「婚外恋愛」「複雑な恋愛事情」
・「キープされている関係」
```

完全に避けるのではなく、**直接的な表現をマイルドに言い換える**ことでリスクを下げられます。

---

### ✅ フォロー・いいねのルール

```
・1時間あたりのいいね：5〜10件まで
・1日あたりのフォロー：20〜30件まで
・深夜（2〜5時）の大量操作は避ける
・自動化ツールは使わない（規約違反かつBANリスク大）
```

---

## ◆STEP4：BANされた時の復活手順

どれだけ気を付けていても、Threadsは突然BANされることがあります。
そのときのために、事前に準備しておくことが大切です。

### ✅ 事前にやっておくべきこと（BAN前の準備）

```
1. 投稿内容をスプレッドシートに記録しておく
   （投稿文・投稿日・インプ数・いいね数）

2. フォロワーリストのスクショを定期的に取っておく

3. noteのURLや固定投稿で使っている文章を保存しておく

4. プロフィール文・アイコン画像を手元に保存しておく
```

---

### ✅ BANされた後の対応

**【まず確認すること】**
```
□ アカウントが完全凍結か、一時的な制限か確認する
□ Instagramから「アカウントへのアクセスを確認」の通知が来ていないか確認
□ 身分証明書の提出を求められていないか確認（提出すれば解除される場合あり）
```

**【解除申請の方法】**
```
1. Threadsアプリの「ヘルプ」→「サポートに問い合わせる」から申請
2. 「私のアカウントは間違いで無効化されました」を選択
3. メールアドレスを入力して申請完了
4. 数日〜1週間で返答が来る（解除されない場合もある）
```

**【申請が通らない場合：新規アカウントで再スタート】**
```
1. 新しいメールアドレス・電話番号で新規Instagramを作成
2. STEP1〜2を守って新Threadsアカウントを作成
3. 保存していた投稿をAIでリライトして再投稿（文章を変えて別のコンテンツとして出す）
4. 過去の投稿データをもとに、最初から伸びやすい投稿を作れる
```

---

## ◆まとめ：BAN回避のゴールデンルール5箇条

```
① 新規アカウント作成後12〜15時間は何もしない

② 最初の1週間はリンクを貼らず、一般人のように動く

③ 投稿のリンクはすべてリプ欄に貼る（本文直貼りNG）

④ 1時間に5いいね・1日20フォローを超えない

⑤ 過去の投稿はすべてスプレッドシートに記録しておく
```

---

> ⚠️ **最後に一番大切なこと**
> BANは「もしも」ではなく「いつか来るもの」として備えてください。
> 事前に投稿を記録しておけば、BANされても翌日から新しいアカウントで再スタートできます。
> 「記録している人」と「していない人」では、BANされた後の回復速度がまったく違います。
