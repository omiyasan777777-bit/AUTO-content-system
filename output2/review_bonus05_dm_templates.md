# 【新レビュー特典⑤】売上に直結するDM返信テンプレート集30選

---

## ✅ このテンプレートの使い方

DMは「ただの相談対応」ではありません。
正しく使えば**①信頼構築 ②投稿ネタ収集 ③note購入への誘導**の3つを同時に実現できます。

テンプレートはそのまま使わず、**相手の状況に合わせて一言カスタマイズ**してから送ってください。
「テンプレっぽい」と思われた瞬間に信頼が崩れます。

## ◆カテゴリ①：最初の返信（第一印象を作る）

### テンプレート1：相談が来たとき（基本の返し）
```
メッセージありがとうございます！
読んでいて、気持ちがすごく伝わってきました。

少し詳しく聞かせてもらえると、もう少し具体的にお話できると思うので…
今の状況を教えてもらえますか？
（例：付き合ってどのくらいか、最後に連絡したのはいつか、など）
```

### テンプレート2：長文の相談が来たとき
```
長文を送ってくれてありがとうございます。
全部読みました。

正直、かなりしんどい状況だと思います。
でも、あなたが悪いとは全然思っていません。

少し整理して返信させてください。
1つだけ確認していいですか？
彼との最後のやりとり、どんな流れで終わりましたか？
```

### テンプレート3：感情が爆発した状態で相談が来たとき
```
まず、気持ちを打ち明けてくれてありがとう。

本当につらかったですよね。
その気持ち、ちゃんと受け取りました。

今すぐ答えを出そうとしなくていいです。
少し落ち着いたら、また話しましょう。
私はここにいます。
```

## ◆カテゴリ②：共感を伝える（信頼を作る）

### テンプレート4：「なんで連絡くれないの」系
```
既読スルーされてる間、何百回もスマホ確認してしまいますよね。

「嫌いになったのかな」「何か怒らせたかな」って
頭の中でぐるぐるして、全然他のことが考えられなくなる。

その気持ち、すごくわかります。
```

### テンプレート5：「もう終わりかも」系
```
「もう無理かもしれない」って思い始めたとき、
それでもまだ諦められないの、すごく苦しいですよね。

好きだから苦しいのに、その気持ちが重いって言われるのが
一番きつい矛盾だと思います。
```

### テンプレート6：「友達に相談できない」系
```
こういうこと、リアルの友達には話しにくいですよね。

「また同じ話？」って思われそうで言えなかったり、
「もう忘れなよ」って言われることがわかってて話せなかったり。

だからこそ、ここで話してくれてよかった。
ちゃんと聞きます。
```

## ◆カテゴリ③：アドバイスを伝える（価値を届ける）

### テンプレート7：追いLINEを止めたい人へ
```
追いLINEを送りたい気持ち、すごくわかります。
でも正直に言うと、今送ったらほぼ確実に逆効果になります。

彼が今感じているのは「嫌い」じゃなくて「重い」「息苦しい」の可能性が高いです。
追いかければ追いかけるほど、彼の「逃げたい」が強くなってしまう。

今一番必要なのは、「連絡を断つ」こと。
これ、しんどいのはわかってる。でも、これしかないんです。
```

### テンプレート8：冷却期間に入った人へ
```
冷却期間に入るの、勇気がいりますよね。

ただ待つだけじゃなくて、この期間に「自分を変える」ことが
復縁できるかどうかの分かれ道になります。

彼に「あれ、変わった？」と思わせるには
連絡しないことより、自分が変わることの方が重要です。

何から始めたらいいかわからなかったら、一緒に考えましょうか？
```

### テンプレート9：「どうせ無理」と思っている人へ
```
「もう無理かも」って思うの、当然だと思います。
あんなに傷ついた後に「大丈夫」なんて思えないですよね。

でもひとつだけ。
復縁できた人と、できなかった人の差は「気持ちの強さ」じゃないんです。

「正しい順番で動いたかどうか」だけです。
あなたはまだ動けます。
```

```
復縁した方法を、noteにまとめてあります。

私自身の体験談と、DMで100人以上の相談を受けてわかったことを
全部書いたので、今の状況に当てはまる部分があると思います。

よかったら読んでみてください→（リンク）
```

### テンプレート11：「もっと詳しく知りたい」と言われたとき
```
ありがとうございます！

正直、DMで全部話せるほどシンプルじゃないんですよね…
段階によってやることが変わるので。

それを全部まとめたのがこのnoteです→（リンク）

冷却期間の過ごし方から、再接触のタイミングまで
ステップごとに書いてあるので、読みながら進めてみてください。
```

### テンプレート12：価格を聞かれたとき
```
（価格）円です！

パート1日分より安くて、
私が3ヶ月以上かけて試行錯誤した全部が入ってます。

「知ってたら最初からこれをやった」って内容ばかりなので、

```

## ◆カテゴリ⑥：NG返し・炎上回避（やってはいけない返し方）

### ❌ NG例①：正論を返す
```
「その人はやめた方がいいですよ。脈なしだと思います。」
→ 最悪です。即ブロックされます。
```

### ✅ OK例①：同じことを共感ベースで伝える
```
「それだけ尽くしてきたんですね…。
その気持ち、ちゃんと受け取りました。
一つだけ聞いてもいいですか？
彼との最後のやりとりはどんな感じでしたか？」
```

---

### ❌ NG例②：すぐに売り込む
```
「私のnoteを読めば解決しますよ！リンクはこちらです！」
→ 信頼ゼロの状態での売り込みは逆効果です。
```

### ✅ OK例②：共感してから自然に誘導する
```
「同じ状況から復縁した方法、noteにまとめてあります。
押しつけたいわけじゃないんですが、
今話してくれた内容と近い部分が多くて。
よかったら参考にしてみてください。」
```

---

### ❌ NG例③：長文で返しすぎる
```
（500文字以上の長文返信）
→ 相手は読む気力がないことが多い。
  端的に共感して、1〜2の質問で返すのがベスト。
```

## ◆DM活用の3つの黄金ルール

```
① まず「共感」してから話す
　→ 相手が「この人わかってくれる」と感じるまで、アドバイスしない

② 返信は短く。1DMに1テーマで。
　→ 長文は読まれない。返信しやすい形で終わる

③ noteへの誘導は「押し付け」じゃなく「紹介」
　→「よかったら」「気になったら」のトーンを守る
```
