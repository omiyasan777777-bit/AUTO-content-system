## 悩みの深掘り

```
表面の悩み: 「副業を始めたいが、毎日疲れ果てて動けない」
  ↓ なぜそれが悩みなのか？
第2層: 「仕事や育児で消耗しきっていて、帰宅後に新しいことを始める気力が残っていない」
  ↓ それが解決しないとどうなる？
第3層: 「このまま何も変わらず、収入源が会社の給料だけという不安定な状態が続く。
         いつかリストラや病気で収入がゼロになるかもしれない恐怖」
  ↓ 本当はどうなりたい？
第4層: 「会社の給料以外にも"自分の力で稼ぐ収入"が少しでもある状態にしたい。
         誰かに頼らず、自分のペースで生きていける基盤を作りたい」
  ↓ それが叶ったらどんな感情？
第5層: 「毎月少額でも自分で稼いでいるという"自己効力感"と、
         万が一の時も怖くないという"安心感"を同時に手に入れたい」
```

→ セールスレターでは「疲労感」「不安・恐怖」「自己効力感・安心感」の全層に訴求する

## 価値の転換

```
■ 変換テンプレート

自分の経験: 日常生活の小さな工夫・失敗・気づきをnoteに書いてきた
  ↓
ターゲットの悩み: 「特別な実績や知識がないから発信できない」という思い込み
  ↓
提供する価値: 「地味な日常経験でも、切り取り方と見せ方を変えれば有料コンテンツになる」
              という思考転換と、それを実現する具体的なフレームワーク
  ↓
独自の切り口: 「頑張ること」を前提にした副業論を全否定し、
              「低エネルギーでも動ける最小設計」という視点からnote副業を再定義する
```

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 副業に興味がある20代後半〜40代の会社員・育児中の主婦
- 条件1: 毎日仕事・育児で消耗していて、副業に使えるエネルギーが限られている
- 条件2: 「自分には特別な実績も知識もない」と思い込み、発信に踏み出せていない
- 条件3: 努力論・根性論の副業コンテンツに疲れていて、「自分のペースで続けられる方法」を求めている
- ペルソナ名: **さとみさん（32歳・会社員・残業多め・副業歴ゼロ）**

### どこまで（ゴール）
- ゴール: noteで**初めての有料記事を公開し、月1万円の副収入を得る土台**を作る
- 達成判定基準: 有料note1本を公開 → 初売上が発生 → 月1万円ペース（5〜10本販売）

### どうやって（手段）
- 手段: 「日常経験の棚卸し → 最小構成コンテンツ化 → 資産型記事として積み上げる」3ステップ
- 独自メソッド名: **「省エネ積み上げメソッド」**
- 独自性: 「頑張る量」ではなく「設計の正しさ」で収益化できるという逆張り発想。
  1記事あたりの執筆負担を最小化しながら、記事が資産として積み上がる仕組みを最初から設計する。

## コンセプト二層構造

### コアコンセプト（実利的価値）
> **毎日疲れ果てた普通の会社員が、日常の経験だけを使って月1万円の副収入noteを作る方法**

### 抽象的コンセプト（世界観）
> **「省エネ設計図」** ─── 頑張らなくても積み上がる、静かな副収入の地図

- ワーディングの方向性:
  - 「積み上げる」「土台」「設計」「仕組み」「地図」などの安定・再現を示す言葉
  - 「省エネ」「小さく」「ゆっくり」「自分のペース」などの負担感ゼロを示す言葉
  - 「怠けていい」ではなく「設計が正しければ頑張らなくていい」というリフレーミング
- ビジュアルイメージ:
  - 夜の静かな部屋でスマホをぽちぽち操作している人
  - 1個ずつ積み上がっていくブロック、育っていく小さな植物
  - 「仕掛けておいたnoteが寝ている間に売れていた」という情景

## コンセプト文（完成形）

```
毎日残業・育児でヘトヘトで、
副業に回せる気力が1ミリも残っていない人へ。

「自分には特別な実績がない」
「発信できるほどの知識もない」
そう思い込んで、ずっと動けなかった人が、

─────────────────────────────

【省エネ積み上げメソッド】を使って

① 日常の小さな経験を有料noteのネタに変換する
② 1記事ずつ「資産型コンテンツ」として積み上げる
③ 仕組みを作ったらあとは記事が勝手に売れ続ける

という3ステップで、

毎月コツコツ作業しながら
「まず月1万円の副収入」の土台を完成させる
note副業設計ガイド。

─────────────────────────────

独自性: 「頑張る量」ではなく「設計の正しさ」が成果を決める。
       努力ではなく仕組みで動くnote副業の最小設計図。

ストーリー: 根性論の副業コンテンツを見るたびに
           「自分には無理だ」と感じ続けてきた。
           でも、日常の経験を正しく切り取って
           設計を整えるだけで、記事は資産になる。
           その事実に気づいたとき、副業への見方が180度変わった。
```

## 仮想ストーリー

```
【過去】
自分も、ターゲットと同じ状態だった。
毎日の仕事でエネルギーを使い果たして帰宅し、
「副業しなきゃ」と思いながら何もできない夜が続いていた。
SNSで「月10万稼いだ方法」を見るたびに、
「自分には無理だ」「自分に特別なスキルはない」と感じていた。

【転機】
ある日気づいた。
売れているnoteが全員「すごい実績の持ち主」ではなかった。
「自分と同じような普通の悩みを持つ人が、
それを丁寧に言語化して売っている」だけだった。
必要なのは特別な知識ではなく、「日常を切り取る視点」と「設計の正しさ」だった。

【変化】
日常経験の棚卸しから始めて、
最小構成で1記事を作り、資産として積み上げる──
この3ステップを繰り返しただけで、
疲れていても月1万円の副収入が自然と積み上がっていった。

【使命】
「頑張れない自分が悪い」ではなく、
「設計が間違っていただけ」だと気づいてほしい。
だから、同じように疲れ果てている人に向けて、
この「省エネ設計図」を届けたい。
```

## 商品設計への引き継ぎメモ

- このコンセプトから導かれる大まかなステップ案:
  1. 「なぜ頑張らなくていいのか」思考の転換
  2. 日常経験の棚卸し＆コンテンツ化フレームワーク
  3. 最小構成の有料note1本を作るステップ
  4. 記事を資産として積み上げる方法（シリーズ化・再利用）
  5. SEO・アフィリエイトで収益を複線化する補助線
  6. 月1万円から月3万円へスケールする設計

- 重点的に掘り下げるべきノウハウ:
  - 「日常経験→有料コンテンツへの変換フレームワーク」（最重要 / ここが独自の核）
  - 「頑張れない人の心理ブロック解除」（感情共感パート）
  - 「資産型記事の積み上げ設計」（継続性の担保）

- 省略可能な要素:
  - SEO詳細テクニック（補助線として簡潔に触れる程度）
  - アフィリエイトの詳しいやり方（複線化の概念のみ）
  - 高額転換・法人化など先の話

- 世界観に合ったワーディング例:
  - 「積み上げる」「土台を作る」「設計する」「仕組みを仕掛ける」
  - 「省エネ」「低負荷」「最小単位」「自分のペース」
  - 「経験を変換する」「切り取る」「言語化する」
  - 「資産になる」「眠っている間に売れる」「勝手に積み上がる」

- コンテンツマージ元3商品の取り入れ方:
  1. 競合1「頑張れない人向けnote副収入」← 感情設計の流れ・不安共感パートの構成
  2. 競合2「元社畜の設計図」← 「設計図」という視覚的概念の提示方法
  3. 競合3「猫丸シリーズ」← タイトル設計・複数記事での展開戦略
