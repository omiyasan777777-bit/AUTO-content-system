## 悩みの深掘り

```
表面の悩み:「海外のSubstackクリエイターは稼いでいるらしいけど、日本人の自分には無理では？」
  ↓ なぜそれが悩みなのか？
第2層:「英語情報ばかりで、日本語市場でそのまま再現できるか分からない。
        試して失敗する時間・エネルギーが怖い」
  ↓ それが解決しないとどうなる？
第3層:「結局アルゴリズム依存のSNSに戻り、疲弊しながら発信を続けるか、
        何も稼げないまま副業を諦めることになる」
  ↓ 本当はどうなりたい？
第4層:「会社の給料だけに依存しない。自分の文章・知識が"資産"として
        毎月継続収入を生んでほしい」
  ↓ それが叶ったらどんな感情？
第5層:「自分の言葉で生きていける実感。誰かに依存しない自由。
        発信が生活を守る"盾"になっている安心感」
```

→ セールスレターでは「再現できない不安（第2層）」「アルゴリズム地獄への絶望（第3層）」「継続収入という資産への渇望（第4層）」「自分の力で生きていく自由（第5層）」の全層に訴求する

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 副業収入を作りたい日本の会社員・フリーランス志望者
- 条件1: SNS発信をしている or 試みたが、アルゴリズムや継続疲弊で行き詰まっている
- 条件2: ニュースレター・サブスク型の「継続収入」に興味があるが、何から始めるか分からない
- 条件3: Substack未経験〜始めたばかり・フォロワー1,000人以下の初心者
- ペルソナ名: サブスタック初心者・さとみさん（32歳・会社員・週末にnote書いてる）

### どこまで（ゴール）
- ゴール: 90日以内に初の有料購読者を獲得し、月3万円の継続収入の土台を作る
- 達成判定基準: Substackで有料プランを設定 → 購読者1〜30人 → 月3万円の売上達成

### どうやって（手段）
- 手段: 海外Substack成功者5名のモデルを日本語に翻訳した「実行設計書」＋週次行動スケジュール（英語力不要）
- 独自メソッド名: **「海外モデル翻訳法」**（英語情報を日本語市場向けに再設計する5ステップ）
- 独自性: 日本語でのSubstackノウハウは「仕組み解説」が中心。本書は「実行ロードマップ」として、週ごとのTODOリスト付きで提供。海外成功事例の"なぜ売れたか"を分解して日本語市場に移植する

### 抽象的コンセプト（世界観）

- ワーディングの方向性:
  - 「再現」「移植」「翻訳」「設計図」「ロードマップ」を統一使用
  - 難しさより「コピーする感覚」「地図通りに歩く感覚」を強調

- ビジュアルイメージ: 海外地図に日本語の注釈が書き込まれた「翻訳地図」。または青写真（ブループリント）

## コンセプト文（完成形）

```
【ターゲット】
SNS発信に疲れた会社員・副業初心者で
・Substackに興味はあるがフォロワーゼロで何から始めるか分からない
・海外クリエイターが稼いでいるのは知っているが英語情報は読めない
・アルゴリズムに振り回されず自分の言葉で月3〜5万稼ぎたい
という人が、

【手段】
海外Substack成功者5名の収益モデルを日本語に翻訳した
「実行設計書＋90日ロードマップ」を使うことで、

【ゴール】
英語力・フォロワー数に関係なく、
90日以内に初の有料購読者を獲得し
月3万円の継続収入の土台を作ることができる

【独自性】
「何を書くか」ではなく「どう設計するか」にフォーカス。
海外モデルを日本語でそのまま再現できる週次TODOリスト付き

【ストーリー】
海外では「1,000人の真のファン」理論を実践し月100万以上を稼ぐ
Substackクリエイターが続々と生まれている。
しかし日本語でその"設計図"を解説したコンテンツは存在しない。
英語の壁が、日本人クリエイターの参入を阻んでいる。
この情報格差を埋め、日本語でも同じ成功を再現できることを証明するために作られた実践書。
```

## 仮想ストーリー

```
【過去】
SNSで発信を続けながら「フォロワーが増えない」「バズらないと収益が出ない」という
アルゴリズムの壁に何度もぶつかってきた。
投稿をやめたら全てリセット。資産が何も残らない感覚。
「自分の言葉には価値がないのかも」と諦めかけていた。

【転機】
ある日、海外のSubstackクリエイターが
「フォロワー300人で月50万円を稼いでいる」という記事を読んだ。
フォロワー数ではなく、"深さ"で稼ぐ。
アルゴリズムではなく、"直接届ける"。
そのモデルが日本にはまだほとんど存在しないことに気づいた。

【変化】
海外成功者のモデルを1つずつ分解し、日本語市場向けに翻訳・実装した結果、
フォロワーゼロのゼロ状態から90日で初の有料購読者を獲得できた。
100人・1,000人ではなく、「濃い30人」が毎月安定した収入をもたらしてくれる土台ができた。

【使命】
英語の壁のせいで、この稼ぎ方を知らない日本人クリエイターに
「設計図」を届けたい。
海外では当たり前になっているモデルを、日本語で再現できることを証明したい。
```
