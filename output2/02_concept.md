## 悩みの深掘り

```
表面の悩み: 「朝活を始めたいが何をすればいいかわからない」
  ↓ なぜそれが悩みなのか？
第2層: 「欧米のマインドフルネスアプリは続かない。日本人として軸になる朝の過ごし方がない」
  ↓ それが解決しないとどうなる？
第3層: 「海外の人と話すとき、日本人としての文化的な誇りや深みを語れない。ただの"普通の人"で終わる」
  ↓ 本当はどうなりたい？
第4層: 「禅という本物の日本文化を体で知り、グローバルな場でも"軸のある日本人"として存在したい」
  ↓ それが叶ったらどんな感情？
第5層: 「自分の生き方を語れる。文化を背骨にした自信が生まれる。外国人に誇れる朝がある」
```

→ **セールスレターの核**: 「朝活の悩み」から入り、最終的に「グローバルに軸のある日本人になりたい」という根本欲求を刺激する

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: グローバル志向の日本人 / 外国人に日本文化を発信したい人
- 条件1: 英語を使う環境にいる、または海外発信に興味がある
- 条件2: 朝活・瞑想・禅に関心はあるが、体系的に実践できていない
- 条件3: 「日本人として誇れる何か」を探しているが、具体的な手段がない
- ペルソナ名: **Zen Bridgeを渡りたいマサキ**（34歳・外資系勤務・朝活未経験・英語中級）

### どこまで（ゴール）
- ゴール: 「禅式モーニングルーティン30分」を21日間継続し、外国人に禅の朝を英語で説明・発信できるようになる
- 達成判定基準: 21日後に「My Zen Morning Routine」をSNSで英語投稿できる状態

### どうやって（手段）
- 手段: 禅寺の朝の構造（坐禅・清掃・食事作法）を現代人向けに7ステップに圧縮 + 英語発信テンプレート
- 独自メソッド名: **「禅の朝プロトコル」（Zen Morning Protocol）**
- 独自性: ①禅の本質的解説（宗教でなく哲学として）②30分に凝縮した再現可能なステップ③海外発信用英語フレーズ集の三位一体

## コンセプト二層構造

### コアコンセプト（実利的価値）
> 外国人に語れる朝を持っていない日本人が、禅式7ステップのモーニングプロトコルを21日間実践することで、英語でZen朝活を発信できるグローバルな自分になる

### 抽象的コンセプト（世界観）
> **「禅の朝の設計図」── 本物の静けさを、現代の朝に再インストールする**

- ワーディングの方向性: 「インストール」「プロトコル」「設計図」など工学×禅の対比で知的な新鮮さ。日本語では「余白」「整える」「静寂」、英語では「Zen protocol」「morning ritual」「authentic Japanese wisdom」
- ビジュアルイメージ: 夜明け前の禅寺の縁側。墨汁で書かれた7文字のステップ。青白い朝光

## コンセプト文（完成形）

```
英語を使う環境にいる30〜40代の日本人で、
・朝活や瞑想に関心はあるが何から始めればいいか分からない
・外国人と話すとき「日本人らしさ」を語れない
・CalmやHeadspaceは続かなかった

という人が、

禅寺の朝の構造を30分に凝縮した「禅の朝プロトコル7ステップ」と
海外発信用の英語フレーズテンプレートを使うことで、

21日間で「禅式モーニングルーティン」を習慣化し、
外国人にも誇れる本物の朝を持ち、
英語でZen朝活を発信できるグローバルな日本人になる

──独自性──
「禅を体験するだけ」でも「英語を学ぶだけ」でもない。
禅の朝を生き、それを世界に語る、二重の実践ガイド。

──ストーリー──
海外のマインドフルネスが広まる一方で、
本場・日本人こそが「禅の朝」を知らないまま過ごしている。
欧米に渡った禅が変形してCalmやHeadspaceになった。
だが本物の禅の朝は、もっと静かで、もっと具体的で、
もっと日本人の血に馴染むものだった。
```

## 商品設計への引き継ぎメモ

- **このコンセプトから導かれる大まかなステップ案**:
  1. 序章: 欧米で誤解されているZenと本物の禅の朝
  2. 第1章: 禅の朝プロトコル7ステップ（各10〜15分で解説）
  3. 第2章: 21日間継続プログラム（週ごとのレベルアップ設計）
  4. 第3章: 英語発信テンプレート集（SNS投稿・会話フレーズ）
  5. 付録: よくある挫折パターンと対処法

- **重点的に掘り下げるべきノウハウ**:
  - 禅の7ステップの具体的な手順（何秒座る、何を意識するか）
  - 英語フレーズの実用例（Instagram投稿文・外国人への説明トーク）
  - 禅の本質（宗教でなく哲学として）の平易な解説

- **省略可能な要素**:
  - 禅の歴史的詳細
  - 坐禅の宗派別の違い
  - 食事作法の厳格な戒律

- **世界観に合ったワーディング例**:
  - 「インストールする」「再起動する」「プロトコル」（工学系メタファー）
  - 「余白」「整える」「静寂」「ただそこにいる」（禅的表現）
  - 「Zen morning」「authentic ritual」「Japanese wisdom」（英語訴求）

- **コンテンツマージ元3商品の取り入れ方**:
  1. 「禅×マインドフルネス 21日間の旅」← 21日間プログラム形式の週次設計を参照
  2. 「現役僧侶が400日続けた〜」← 体験談の構成・権威性の見せ方を参照
  3. 「6分間のモーニング・ルーティーン」← 最小ステップの具体的な時間設定手法を参照
