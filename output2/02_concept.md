```

  ↓ なぜそれが悩みなのか？
第2層: 毎月の固定費・生活費に追われて旅行を「贅沢」と感じてしまう
  ↓ それが解決しないとどうなる？
第3層: 「また来年でいいか」を繰り返して、いつまでも行けないまま歳を取る
  ↓ 本当はどうなりたい？
第4層: 好きなときに好きな場所へ行ける生活を手に入れたい
  ↓ それが叶ったらどんな感情？

```

```

  ↓

  ↓
独自の切り口: 「節約術の羅列」ではなく「6月という最強シーズンを逆手に取る旅行設計」という時期特化×ライフスタイル設計
```

## コンセプト4要素

### 誰が（ターゲット）
- 大カテゴリ: 旅行好きの20〜35歳
- 条件1: 旅行に行きたい気持ちはあるが「お金がかかる」を理由に年1回以下になっている
- 条件2: 航空券・ホテルの最適な買い方・タイミングがわからない
- 条件3: 旅行を諦めてはいないが、行動に移せていない
- ペルソナ名: 「もったいない旅行者・ゆいさん（28歳・会社員）」

### どこまで（ゴール）
- ゴール: 旅行費を今より3〜5割削減しながら、年3回以上の旅行を実現する
- 達成判定基準: 本書のステップに沿って1回の旅行を計画・実行し、予算が従来比3割以上削減できたかで判定

### どうやって（手段）
- 手段: 6月（梅雨・オフシーズン）の特性を最大活用した「旅行設計ステップ」＋テンプレート
- 独自メソッド名: **「梅雨トク旅設計メソッド」**
- 独自性: 時期特化（6月）× ライフスタイル設計（年間旅行計画） × テンプレ付きで即実践可能

## 仮想ストーリー

```
【過去】
社会人になってから、毎年「今年こそ旅行に行こう」と思いながら、
気づいたら年末に「また行けなかった」とため息をついていた。
GWや夏休みはどこも混んで高いし、「まあいいか」を繰り返すうちに
旅行は「いつか」のままになっていた。

【転機】
ある年の6月、友人に誘われて渋々行った旅行が転機だった。
梅雨だから安い・空いている・でも景色は最高。
「なんで今まで6月を避けていたんだろう」という気づきがそこにあった。

【変化】
6月の活用法と旅行費の仕組みを徹底的に調べ、
「いつ予約するか」「どこを削るか」「どこにかけるか」を整理したことで、
年3回以上の旅行が当たり前になった。しかも1回あたりの費用は以前の6割以下。

【使命】
「お金がないから旅行は無理」と思っている人に伝えたい。
問題はお金ではなく、仕組みを知らないことと、シーズンの選び方だ。
この設計図があれば、今すぐ旅行を計画できる。
```

## 商品設計への引き継ぎメモ

- このコンセプトから導かれる大まかなステップ案:
  1. 6月旅行の真実（なぜ最強か）
  2. 旅行費の仕組みを理解する（航空券・ホテル・現地費用）
  3. 梅雨トク旅設計ステップ（具体的な手順）
  4. 年間旅行計画テンプレートの使い方
  5. 実践事例・旅先おすすめリスト

- 重点的に掘り下げるべきノウハウ:
  - 6月に安く・空いている旅先の具体的なリスト
  - 航空券の最安タイミング（何日前・何曜日）
  - ホテル予約の使い分け（OTA vs 公式）

- 省略可能な要素:
  - 海外旅行の複雑な手続き（初回はドメスティックに絞ってOK）
  - マイル上級者向けの複雑な積算テク

- 世界観に合ったワーディング例:
  - 「梅雨トク」「オフシーズン最強論」「旅行設計図」「逆張り旅」「混まない旅」

- コンテンツマージ元3商品の取り入れ方:
  1. 「旅費を半額にする裏技15選」← 節約テクニックの網羅構造を参考に章立てへ活用
  2. 「旅系インフルエンサー養成」← ストーリー×発信設計の導線設計を参考に
  3. 「旅行好き必見！お得に旅する裏ワザ」← フック設計・冒頭の引きを参考に
