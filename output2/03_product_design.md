## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 元彼との曖昧な関係を3ヶ月以内に「自分が望む着地点」に動かせる
- サブベネフィット1: 「なぜ沼るのか」が腑に落ち、感情の消耗が止まる
- サブベネフィット2: 相手の一言一句に振り回されなくなり、自分に時間とエネルギーが戻る
- サブベネフィット3: 距離感・連絡・感情表現を戦略的にコントロールできる
- 感情的ベネフィット: 「自分の恋愛を自分でコントロールできる」という自信と解放感

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: 沼脱出×着地点設計メソッド
- 他商品にない要素: 「沼る心理の仕組み（可変強化スケジュール）」を恋愛に応用した深い解説
- 自分だけが語れる理由: 同じ「友達以上恋人未満」の沼を経験し、脱出した当事者として語れる

■ 要素3: 実績・証拠
- 自分の実績: 自身が同様の関係から脱出し、関係の主導権を取り戻した実体験
- 購入者の実績: 友人・知人複数名が同じ状況から関係の着地点を見つけたサポート実績
- 具体的な数字: 沼る期間6ヶ月→3ヶ月で着地点設計に動けた体験

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: 曖昧な関係は放置するほど「引っ込みがつかない」期間が伸び、自分の消耗が深まるから
- 限定要素: ローンチ価格4,980円（値上げ予定）
- 見逃した場合の損失: 6ヶ月・1年と消耗し続け、気づいた時には「あの時間を取り戻せない」という後悔
```

### オファー構成

```
■ メイン商品
- コンテンツ本体: 15,000〜20,000文字・全6章
- 形式: テキスト（note）

■ 特典（3個）
- 特典1: 「関係の現在地チェックシート」← 目的: 自分の状況を客観視する ← 単体想定価値: 980円
- 特典2: 「着地点別アクションガイド（復縁・前進・終わらせる3パターン）」← 目的: 即実践できる行動を提供 ← 単体想定価値: 1,480円
- 特典3: 「返信コントロール術——連絡の頻度・タイミング・内容の設計テンプレ」← 目的: 具体的な連絡行動をサポート ← 単体想定価値: 980円
※ 特典合計想定価値 3,440円 > メイン価格4,980円には満たないが、コンテンツ本体の深さで価格を正当化

■ 価格設計
- 本体価格: 5,980円（通常）
- ローンチ価格: 4,980円（先着50名 or 1ヶ月間）
- 値上げ計画: 4,980円 → 5,980円 → 6,980円
- 価格正当化ロジック:
  - 比較対象1: 恋愛相談カウンセリング（60分） = 8,000〜15,000円 → それと比べて1回分以下
  - 比較対象2: 復縁占い（1回） = 5,000〜10,000円 → 占いより具体的なメソッド
  - 1日あたり換算: 16円/日（1年間使い続けた場合）
  - 得られる成果の金銭的価値: 「また付き合えた」という事実は数値化不可。3ヶ月の消耗時間の節約

■ バックエンド導線
- 本商品の先に用意する上位商品/サービス: 個別相談DM（30分/8,000円）、継続サポートLINE（月額制）
- 本商品内でのバックエンド導線: まとめ章で「より深くサポートを受けたい方へ」として案内
```

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「なぜ自分はこれを書いたか（自己開示）」+ 「この教材を読むと何が変わるか（全体像）」
- ノウハウ章で漫画・図解があると効果的な箇所:
  1. 可変強化スケジュールの仕組み図（第2章）→ [DIAGRAM]
  2. 「キープ/本命/迷い中」の判定チャート（第3章）→ [DIAGRAM]
  3. 「関係の現在地マップ」全体像（第4章）→ [DIAGRAM or ILLUSTRATION]
  4. 「頑張るほど遠くなる」追いかける恋愛の悪循環イラスト（第5章）→ [MANGA or ILLUSTRATION]
- 読者がつまずきやすいポイント: 「着地点を決める」章——「どれを選べばいいかわからない」という迷い
- ワーク / テンプレートが必要な箇所: 第4章（沼パターン診断）、第5章（関係の現在地チェックシート）、第6章（返信テンプレ）
- 世界観に合った表現の具体例: 「舵を切る」「着地点を定める」「主導権を取り戻す」「霧が晴れる」
- コンテンツ融合法の実行方針:
  - 商品A（男性心理解説）: 第3章「彼の心理」の男性パターン解説の切り口を参考に
  - 商品B（5つの方法）: 各章末のアクションリスト形式を参考に具体性を上げる
  - 商品C（復縁心理学）: 感情設計（希望→恐怖→希望の流れ）をストーリーラインに活用
  - 自分の独自要素: 可変強化スケジュール解説、着地点の3択設計、当事者体験談
