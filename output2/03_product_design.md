#### 大ステップ1: 「理解の罠」の構造を知る

##### 小ステップ1-2: 「理解の罠」3パターンを知る
```
課題: 「理解している＝正しい対応をしている」と誤解しており、消耗の原因が見えていない
解決策: 理解力が逆効果になる3パターンを具体的に示すことで、自分の行動を客観視させる

何をするか:
「共感免除型」（怒りを感じても共感で自分を納める）、「待機補助型」（彼の決断を待ちながら全て整える）、
「解釈代行型」（彼の気持ちを先読みして代わりに処理する）の3パターンを事例付きで解説する。

なぜやるか:
自分の消耗の「原因」が特定できないと、何を変えればいいかわからない。
パターン名がつくことで「これが自分の罠だ」と認識でき、行動変容のスタートラインに立てる。

コツ/注意点:
「罠にハマっていた自分は間違いだった」と自己否定させない。
「優しさの使い方の問題であって、優しさ自体は才能」というフレームを維持する。

成果物: 自分が3パターンのうちどれに当てはまるか認識できる状態

使うツール/リソース: 3パターン解説テキスト＋事例エピソード2〜3個
```

##### 小ステップ3-2: 環境リセットSTEP2「待つ」から「問う」に変える
```
課題: 彼の反応を待つことが習慣化し、自分の意思表示が「迷惑になる」と感じている
解決策: 「問い」という形式で自分の意思を表現する方法を学ぶ

何をするか:
「あなたはどうしたい？」「私はこうしたいんだけど、あなたはどう思う？」など、
"彼に考えさせる問いかけ"のテンプレートを提供する。
一方的な主張ではなく「問い」にすることで、彼の回避反応を軽減しながら自分の意思を示せる。

なぜやるか:
「待つ」行動は「どうなっても受け入れます」というメッセージを相手に与える。
「問う」行動は「あなたにも考えてほしい」という対等なメッセージになる。この違いが関係の構造を変える。

コツ/注意点:
問いかけ後の沈黙に耐えること。すぐに自分から答えを埋めない。
問いかけは1回のやりとりで1つまで（詰問にならないよう）。

成果物: 3つの「環境リセット問いかけテンプレート」

使うツール/リソース: 問いかけ例文10パターン（特典LINE実例集と連動）
```

## 有料部分作成への引き継ぎメモ

- **導入章で特に強調すべきポイント**:
  「理解できるのに前に進めない」という状況の描写を徹底的にリアルに書く。読者が「これは私のことだ」と確信するまで共感を積む。その上で「実は構造がある」と問題の本質へ誘導する。

- **ノウハウ章で効果的な箇所**:
  - 第1章: 「理解の罠3パターン」は図解的なテキスト表現（箇条書き・囲み）で視覚的に整理
  - 第3章: 「環境リセット3ステップ」は番号付きで段階感を出す

- **読者がつまずきやすいポイント**:
  - 「環境リセット＝自分が我慢しなくなること」と誤解するかもしれない → 「尊重しながら自分の意思を出す」ことを丁寧に説明
  - 「判断軸を使って決断すること」への罪悪感 → 決断を促すのではなく「選択肢が明確になる」という表現に留める

- **ワーク / テンプレートが必要な箇所**:
  - 1-3: 3パターン診断チェックシート
  - 3-1〜3-3: 各ステップの実践テンプレート（特典ワークシートと連動）
  - 4-2: 「続ける・変える・離れる」判断ワーク

- **世界観に合った表現の具体例**:
  - ✓「環境をリセットする」「構造に気づく」「設計し直す」「地図を持つ」
  - ✓「消耗」「理解の罠」「選ぶ側に立つ」「主体的に」
  - ✗「攻略する」「テクニック」「彼を動かす」「別れなさい」
