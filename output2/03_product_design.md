## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 旅費を今より3〜5割削減しながら、年3回以上の旅行を当たり前にする
- サブベネフィット1: 「梅雨だから安い・空いている・楽しい」の三拍子を手に入れる
- サブベネフィット2: 航空券・ホテルをいつ・どこで予約すればいいか迷わなくなる
- サブベネフィット3: 年間旅行計画テンプレートで「また来年でいいか」を卒業できる
- 感情的ベネフィット: 「自分は旅行できる人間だ」という自信と人生の充足感

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: 梅雨トク旅設計メソッド
- 他商品にない要素: 6月・梅雨シーズン特化 × ライフスタイル設計 × 年間旅行計画テンプレート付き
- 自分だけが語れる理由: 「また来年でいいか」を繰り返していたが6月旅行との出会いで年3回以上の旅行生活に変わった実体験

■ 要素3: 実績・証拠
- 自分の実績: 6月活用で1回あたりの旅行費を従来比40%削減・年3回以上旅行を継続
- 購入者の実績: （モニター段階：初回購入者の声を収集予定）
- 具体的な数字: 旅費40%削減・年3回旅行・繁忙期比ホテル代平均30〜50%安

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: 6月は目前。今手に入れれば今年の6月旅行から即実践できる
- 限定要素: ローンチ価格は先着30名 or 7日間限定で通常より500円引き
- 見逃した場合の損失: 今年の6月を見逃せば次のチャンスは1年後。繁忙期の高い旅行費を払い続けることになる
```

### オファー構成

```
■ メイン商品
- コンテンツ本体: 約15,000〜20,000字・6章構成
- 形式: テキスト（note有料記事）

■ 特典（3個）
- 特典1: 年間旅行計画テンプレート（Googleスプレッドシート）
  ← 目的: 読んだ後すぐ計画を立てられる即実践ツール ← 単体想定価値: 980円
- 特典2: 6月おすすめ旅先リスト15選（国内・穴場厳選）
  ← 目的: 旅先選びに迷わないための具体的リスト ← 単体想定価値: 500円
- 特典3: 旅行費チェックリスト（予約前・準備・当日・帰宅後）
  ← 目的: 「やり忘れ」「損」をなくす実践ツール ← 単体想定価値: 500円
※ 特典合計想定価値: 1,980円 > 本体価格1,480円

■ 価格設計
- 本体価格: 1,480円（通常価格）
- ローンチ価格: 980円（先着30名 or 7日間限定）
- 値上げ計画: 980円 → 1,480円 → 1,980円（最終価格）
- 価格正当化ロジック:
  - 比較対象1: 旅行雑誌・ガイドブック = 1,000〜1,500円 → 読むだけ vs 即実践できる設計図
  - 比較対象2: 繁忙期1泊で増える宿泊費差額 = 平均5,000〜15,000円 → たった1回の旅行で元が取れる
  - 1日あたり換算: 約4円（1年間365日で割った場合）
  - 得られる成果の金銭的価値: 年3回旅行・1回あたり旅費3割削減 = 年間節約額1万〜3万円相当

■ バックエンド導線
- 本商品の先に用意する上位商品: 旅行×発信（Threads・note）で収益化するノウハウ / 個別旅行計画サポート
- 本商品内でのバックエンド導線の仕込み方: 最終章で「旅行生活を発信・収益化したい人向け」として次のステップを案内
```

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント:
  - 「繁忙期に旅行しているから損している」という共感と気づきから入る
  - 「設計図さえあれば誰でも再現できる」という希望で締める

- 読者がつまずきやすいポイント:
  - 「本当に梅雨でも楽しめるの？」という懐疑心 → データ・実例で払拭が必須
  - 「自分には関係ない」と思われないようターゲットの悩みを冒頭で明確に言語化

- ワーク / テンプレートが必要な箇所:
  - 第3章: 旅行費シミュレーション表
  - 第4章: 旅行計画テンプレート（日数別）
  - 第5章: 年間旅行カレンダー・積み立て設計シート

- 世界観に合った表現の具体例:
  - 「旅行は才能ではなく設計だ」
  - 「梅雨を逆張りする旅人になる」
  - 「繁忙期に払っていたお金を、旅の回数に変える」

- コンテンツ融合法の実行方針:
  - 商品A（旅費を半額にする裏技15選）: 節約テクニックの具体例・数字を第3章に組み込む
  - 商品B（旅系インフルエンサー養成）: ストーリー構造・感情設計を導入・まとめ章に活用
  - 商品C（旅行好き必見！お得に旅する裏ワザ）: 冒頭フック設計・読者の引き込み方を参考にする
  - 自分の独自要素: 6月特化データ・梅雨旅先15選・年間旅行設計の考え方を全編に統一
