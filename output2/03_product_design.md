## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 感情が爆発する深夜でも「今夜は送らない」を自分で選択できる
- サブベネフィット1: 衝動的な行動で関係をさらに悪化させるリスクがなくなる
- サブベネフィット2: 「何かしたい」感情に神社参拝という意味ある行動を与えられる
- サブベネフィット3: 自分の感情を言語化する習慣がつき、メンタルが安定する
- 感情的ベネフィット: 「焦って壊してしまう自分」から「落ち着いて選択できる自分」への変容

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: 聖地参拝メソッド（清め→祈り→奉納→待機→感謝の5フェーズ）
- 他商品にない要素: 「今夜のリアルタイム処方箋」として設計。読んだその夜に使える
- 自分だけが語れる理由: 深夜の衝動と戦い、神社参拝が転機になった実体験

■ 要素3: 実績・証拠
- 自分の実績: 深夜の衝動LINE送信をゼロにし、自分から距離を保つことで関係が好転した体験
- 購入者の実績: 「読んだその夜、送るのを止められた」「神社に行く意味がわかった」
- 具体的な数字: 衝動的なLINE送信3回分の「止める力」

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: 今夜もあの画面を開いてしまう前に、処方箋を手元に置いておく
- 限定要素: 初版価格980円（価格改定予定）
- 見逃した場合の損失: また衝動的に連絡して、関係修復の可能性を自ら削ってしまう
```

### オファー構成

```
■ メイン商品
- コンテンツ本体: 約18,000文字・6章構成（導入+4ノウハウ章+まとめ）
- 形式: テキスト（note有料記事）

■ 特典（2個）
- 特典1: 「今夜連絡したくなったら開く、感情チェックシート」← 衝動時の即使用 ← 単体想定価値: 500円
- 特典2: 「復縁神社パワースポット10選マップ（関東・関西）」← 実際に参拝するときの利便性 ← 単体想定価値: 300円

■ 価格設計
- 本体価格: 1,480円
- ローンチ価格: 980円（先着50名）
- 値上げ計画: 980円 → 1,480円 → 1,980円
- 価格正当化ロジック:
  - 比較対象1: 恋愛相談LINE占い = 3,000〜10,000円 → それと比べて圧倒的にコスパが高い
  - 比較対象2: 恋愛カウンセリング = 10,000円〜/回 → 一生使えるコンテンツが980円
  - 1日あたり換算: 約26円（37日間で割った場合）
  - 得られる成果の金銭的価値: 衝動LINEで関係を壊さずに済んだ価値は計り知れない

■ バックエンド導線
- 本商品の先に用意する上位商品/サービス: 「復縁を現実にする60日プログラム」個別サポート付き（29,800円）
- 本商品内でのバックエンド導線の仕込み方: まとめ章で「次のステップ」として自然に案内
```

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「この本は今夜のためにある」という即時性の強調
- 読者がつまずきやすいポイント: 感情を「我慢」することと「整える」ことの違い
- ワーク / テンプレートが必要な箇所: 感情チェックシート（1-1）、トリガーマップ（1-2）、今夜の行動リスト（4-1）
- 世界観に合った表現の具体例: 「清める」「参道を歩く」「願いを奉納する」「灯籠の光」「夜の静けさ」
- コンテンツ融合法の実行方針:
  - 90日ロードマップ: ステップ設計の明確さ・段階的なプロセス感
  - 復縁率78%教科書: 失敗パターンの先出し・問題提起の構成
  - 禁断LINE術: 「今夜すぐ使える」即効性の演出・具体的なシーン描写
  - 自分の独自要素: 神社メタファー×感情整理の融合・深夜に読まれる設計
