## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 1日10分の確認だけで楽天ROOMが自動で稼ぎ続ける仕組みの完成
- サブベネフィット1: 商品選定に迷わない（AIが毎回ベスト商品を選ぶ）
- サブベネフィット2: 紹介文作成が不要（AIが人間らしい文章を生成）
- サブベネフィット3: 挫折しない（1日10分なので続く）
- 感情的ベネフィット: 「寝ている間に稼いでいる」安心感・副業が軌道に乗った自信

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: AI店長メソッド
- 他商品にない要素: AIを「道具」でなく「経営判断者（店長）」として設計するフレームワーク
                   特定ツール非依存・利用規約安全設計・1日10分PDCAルーティン
- 自分だけが語れる理由: 「ツールを入れただけでは稼げない」という市場の本質的問題を、
                       AIの役割設定という発想転換で解決したアプローチ

■ 要素3: 実績・証拠
- 自分の実績: AI店長設定後3ヶ月以内に月3万円超え達成
- 購入者の実績: 「毎日投稿をやめてから逆に収益が上がった」「1日10分になって続くようになった」
- 具体的な数字: 月3万円 / 1日10分 / 3ヶ月 / 作業時間90%削減

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: AI活用の早期参入者が有利。市場が成熟する前に仕組みを作った人が長期的に稼ぎ続ける
- 限定要素: ローンチ価格（期間限定）
- 見逃した場合の損失: 毎日投稿の消耗が続く・ツール任せで稼げない状態が続く
```

### オファー構成

```
■ メイン商品
- コンテンツ本体: 約17,000字 / 6章構成（導入1章+ノウハウ4章+まとめ1章）
- 形式: テキスト（note有料記事）

■ サポート設計
- 質問対応: あり（DM）
- 対応期間: 購入後30日間
- 対応回数: 3回まで
- コミュニティ: なし
- 添削・フィードバック: なし（ガイドラインを見れば自己完結できる設計）

■ 特典（3個）
- 特典1: AI店長セットアップ用プロンプト集（コピペOK）← 目的: すぐ使い始められる ← 単体想定価値: 3,000円
- 特典2: 1日10分PDCAチェックシート（毎日の確認ルーティン表）← 目的: 継続の仕組み化 ← 単体想定価値: 1,000円
- 特典3: 売れやすいジャンル選定ガイド（楽天ROOM × 季節 × カテゴリ早見表）← 目的: 初動を速くする ← 単体想定価値: 1,500円
※ 特典合計想定価値: 5,500円 > 本体価格 1,980円

■ 価格設計
- 本体価格: 1,980円
- ローンチ価格: 1,480円（リリース後7日間 / 先着50名）
- 値上げ計画: 1,480円 → 1,980円 → 2,980円（3ヶ月後）
- 価格正当化ロジック:
  - 比較対象1: 自動化ツール月額費用 = 3,000〜5,000円/月 → それより安く永続的な仕組みが手に入る
  - 比較対象2: 副業コンサル = 50,000円〜 → 1/25以下の価格で同等の設計図が手に入る
  - 1日あたり換算: 約5.4円/日（365日で割ると）
  - 得られる成果の金銭的価値: 月3万円 × 12ヶ月 = 年36万円の副収入

■ バックエンド導線
- 本商品の先に用意する上位商品: AI副業コンサル（個別相談30分 / 10,000円）
- 本商品内でのバックエンド導線: まとめ章で「より高度な設定・収益最大化はコンサルで」と案内
```

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「ツールだけでは稼げない理由」→「AI店長という発想転換」の流れ
- ノウハウ章で効果的な箇所: AI店長設定プロンプトはコードブロックで提示（コピペOK）
- 読者がつまずきやすいポイント: AIへの指示の具体性（曖昧な指示→機械的な結果の悪循環）
- ワーク / テンプレートが必要な箇所: 各章末に1つずつ（プロンプトテンプレ or チェックシート）
- 世界観に合った表現の具体例: 「採用する」「任せる」「委任する」「報告を受ける」
