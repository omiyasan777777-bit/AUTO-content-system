## PART A: オファー設計

### 訴求4要素

```
■ 要素1: ベネフィット（得られる成果）
- メインベネフィット: 「恋愛すると自分が自分じゃなくなる」から卒業できる
- サブベネフィット1: 返信待ちの不安に振り回されなくなる
- サブベネフィット2: 「断る」「確認する」勇気が持てるようになる
- サブベネフィット3: 繊細さを弱点でなく武器として使えるようになる
- 感情的ベネフィット: 好きな人の前でも「自分らしくいられる」安心感・自信

■ 要素2: 独自性（なぜこの商品なのか）
- 独自メソッド名: HSP恋愛クエスト
- 他商品にない要素: 恋愛の悩みを「モンスター」として可視化するRPG世界観
- 自分だけが語れる理由: 恋愛テクニックゼロ。「相手の攻略」ではなく「自分の心の育成」という真逆の設計

■ 要素3: 実績・証拠
- 自分の実績: HSP気質で恋愛のたびに消耗してきた当事者として5つのスキルを体系化
- 具体的な数字: 5つのスキル（待つ・断る・確認する・休む・離れる）で構成された体系的メソッド

■ 要素4: 緊急性・限定性
- なぜ今買うべきか: 繊細さを「欠点」と思い続けるほど、恋愛から遠ざかっていく
- 限定要素: 先行価格あり（後述）
- 見逃した場合の損失: このまま「恋愛するたびにHPを削られる」パターンを繰り返す
```

### オファー構成

```
■ メイン商品
- コンテンツ本体: 約20,000字・6章構成
- 形式: テキスト（RPG世界観で装飾）

■ 特典（3個）
- 特典1: 「あなたを削る恋愛モンスター診断シート」← 目的: 自己分析促進 ← 単体想定価値: 980円
- 特典2: 「シーン別・自己防衛フレーズ集（返信待ち・断り方・確認の仕方）」← 目的: 即使える行動ツール ← 単体想定価値: 1,480円
- 特典3: 「繊細さを武器に変える！HSPの『強み活用』チェックリスト」← 目的: 自己肯定感の強化 ← 単体想定価値: 780円
※ 特典合計: 3,240円（本体価格を上回る設計）

■ 価格設計
- 本体価格: 2,480円
- ローンチ価格: 1,980円（先着50名）
- 値上げ計画: 1,980円 → 2,480円 → 2,980円（最終価格）
- 価格正当化ロジック:
  - 比較対象1: 恋愛カウンセリング1回 = 5,000〜15,000円 → それと比べて圧倒的にコスパ◎
  - 比較対象2: 恋愛コーチング = 月3〜10万円 → それと比べて何度でも読み返せる
  - 1日あたり換算: 約66円（1ヶ月で割った場合）
  - 得られる成果の金銭的価値: 「消耗しない恋愛」で失う時間・エネルギー・自己肯定感を換算すれば計り知れない

■ バックエンド導線
- 本商品の先に用意する上位商品/サービス: 個別DM相談・1on1コーチング（HSP恋愛特化）
- 本商品内でのバックエンド導線の仕込み方: まとめ章に「さらに深めたい方へ」として誘導
```

## PART C: 世界観設計

### ワーディング

```
■ トーン
- 全体の雰囲気: 親しみやすい・かわいい・でも中身は本質的
- 一人称: 私
- 読者の呼び方: あなた / 繊細さん

■ キーワード
- 繰り返し使うワード: HP、モンスター、スキル、クエスト、装備、レベルアップ、冒険、攻略、クリア、繊細さ、自己防衛
- 使わないワード: テクニック、駆け引き、攻略（相手の攻略の意味では使わない）、依存
- 章タイトルの統一パターン: 「ステージN：〇〇——△△」

■ メタファー
- 全体のメタファー: 「恋愛RPG」「冒険書」「クエスト」
- このメタファーから派生する表現:
  - 「HPが削られる」→ 感情的な消耗
  - 「スキルを装備する」→ 新しい対処スキルを習得する
  - 「モンスター」→ 不安・察しすぎ・依存などの感情パターン
  - 「クリアする」→ 悩みを乗り越える
  - 「レベルアップ」→ 自己成長
  - 「セーブポイント」→ 振り返り・ワークの機会
```

### ビジュアルイメージ

```
■ 本文内の装飾ルール
- 見出しの装飾: 🗡️ ⚔️ 💊 🛡️ ✨ など（RPGっぽい絵文字）
- 強調の方法: **太字**
- 区切りのスタイル: ---
- モンスター登場: 【⚠️ モンスター出現：〇〇】という形式
- スキル習得: 【✨ スキル習得：〇〇】という形式
- セーブポイント: 【💾 セーブポイント】でワーク区切り
```

## 有料部分作成への引き継ぎメモ

- 導入章で特に強調すべきポイント: 「相手を攻略するのではなく自分の心を育てる」という逆転の発想。RPGの世界観説明
- ノウハウ章で具体的なシーンが必要な箇所: 返信待ち・既読スルー・断るシーン・「どう思ってるの？」を聞くシーン
- 読者がつまずきやすいポイント: 「断ること」への罪悪感、「離れること」への自己嫌悪
- ワーク / テンプレートが必要な箇所: 1-1（モンスター診断）、2-2（待つ練習）、3-2（確認の練習）、5-3（エンディング設定）
- 世界観に合った表現の具体例: 「今日もHPを削られてしまったあなたへ」「そのモンスター、攻略法があります」
- コンテンツ融合法の実行方針:
  - 競合1（共感文体）: 導入章・各章の痛みの代弁部分に活用
  - 競合2（具体的悩みシーン）: 返信待ち・既読スルー等のシーン描写に活用
  - 競合3（数字・チェックリスト）: 各章末のセーブポイント・チェックリストに活用
  - 自分の独自要素: RPG世界観・モンスター図鑑・スキル習得システム全体
