## PART A: オファー設計

### 訴求4要素

**■ 要素1: ベネフィット**
- メインベネフィット: 月1回2時間の作業で30日分のThreads投稿を自動スケジュール、毎日投稿が永続的に仕組み化される
- サブベネフィット1: 毎日「今日何を投稿しよう」と悩むストレスがなくなる
- サブベネフィット2: Claude Codeへの指示テンプレートを手に入れ、投稿ネタ切れが永遠に起きなくなる
- サブベネフィット3: コードが一行も書けなくても、エンジニアと同等の自動化ワークフローを持てる
- 感情的ベネフィット: 寝ている間もThreadsが動き続ける安心感と「仕組みで動く自分の資産」を持つ自信

**■ 要素2: 独自性**
- 独自メソッド名: 「月イチ2時間システム」
- 他商品にない要素: GAS・Zapier・Makeを一切使わずClaude Code × Bufferだけで完結するワークフロー全公開
- 自分だけが語れる理由: 元コードゼロ人間が実際に構築・運用しているシステムを、非エンジニア目線で完全公開

**■ 要素3: 実績・証拠**
- 自分の実績: 月1回2時間の作業で30日分の投稿を自動スケジュール実現
- 具体的な数字: 月2時間 → 30日分投稿、約2〜3週間でワークフロー構築完了

**■ 要素4: 緊急性・限定性**
- なぜ今買うべきか: AI×SNS自動化は2026年最注目テーマ。今のうちに仕組みを持つ人と持たない人の差は今後どんどん広がる
- 限定要素: ローンチ価格（先着・期間限定）
- 見逃した場合の損失: 毎日「今日の投稿どうしよう」というストレスを抱え続け、結局Threadsを諦めることになる

### オファー構成

**■ メイン商品**
- コンテンツ本体: 約15,000〜18,000字・5章構成
- 形式: テキスト（Claude Codeプロンプト全文・Buffer設定スクリーンショット付き）

**■ 特典**
- 特典1: 「Claude Code×Threads投稿生成プロンプトテンプレート集（10パターン）」← すぐ使える ← 単体想定価値: 980円
- 特典2: 「Buffer APIセットアップ チェックリスト」← 初期設定ミスゼロ ← 単体想定価値: 500円
- 特典3: 「30日分投稿カレンダーテンプレート（Notion/Google Sheets）」← 運用管理 ← 単体想定価値: 500円

**■ 価格設計**
- ローンチ価格: **980円**（最初の2週間）
- 通常価格: **1,480円**
- 最終価格: **1,980円**
- 価格正当化ロジック:
  - 比較1: SNS代行サービス = 月3〜10万円 → それと比べて圧倒的コスパ
  - 比較2: AI活用コンサル = 1回1〜3万円 → 同等の知識が980円で手に入る
  - 1日あたり換算: 約27円（ローンチ価格を1ヶ月で割った場合）
  - 得られる成果: 毎月のSNS投稿時間を30時間→2時間に短縮（=28時間分の時間創出）

**■ バックエンド導線**
- 上位商品候補: 「Claude Code×Buffer×Threads 完全構築サポート（個別相談）」
- 本商品内での仕込み: 最終章に「さらに深く・個別に最適化したい方へ」として導線を設置

## 有料部分作成への引き継ぎメモ

- **導入章で特に強調すべきポイント**: 「毎日投稿が続かなかった理由はあなたのせいではなく、仕組みがなかったから」というフレーミング
- **ノウハウ章で図解があると効果的な箇所**: 3ツールの役割分担図、ワークフロー全体図、Buffer設定スクリーンショット、プロンプト→投稿生成フロー
- **読者がつまずきやすいポイント**: Claude Codeのコスト設定、Buffer×Threads連携のOAuth認証、プロンプトの品質が低い時の対処
- **ワーク / テンプレートが必要な箇所**: キャラ設定プロンプトテンプレート、投稿型5パターンテンプレート、月イチ作業チェックリスト
- **世界観に合った表現例**: 「仕組みを設計する」「マシンを稼働させる」「月イチメンテナンス」

  - 商品1（GASラボ）→ 丁寧なスクリーンショット解説スタイルを採用（Bufferの設定画面を全部載せる）

  - 商品3（ai_jissennkai）→ プロンプト全文公開のスタイルで差別化（隠さない）
  - 自分の独自要素: 「コードゼロの非エンジニア目線」「キャラ設定プロンプトでAI感を消す技術」「note×Threadsの収益導線」
