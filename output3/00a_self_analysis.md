## 1. ビフォーアフター整理

### 変化1: 毎日投稿の苦痛からの解放
- Before: Threadsで毎日投稿しようとするも、ネタ切れ・時間不足で三日坊主になる
- きっかけ: Claude Code × Buffer の組み合わせでワークフローを構築できることを発見した
- 乗り越えた壁: Claude Codeの使い方を覚える学習コスト、Buffer APIの設定、プロンプト設計の試行錯誤
- やったこと: Claude Codeで投稿文を一括生成 → Buffer APIでスケジュール自動登録のワークフロー構築
- After: 月1回2時間の作業で30日分の投稿を自動スケジュール。毎日投稿が完全自動化
- 期間: 約1〜2ヶ月

### 変化2: コード知識ゼロからの自動化実装
- Before: 「自動化したい」と思っても、プログラミングの壁で諦めていた
- きっかけ: Claude Codeに自然言語で指示するだけでコードが書けることを知った
- 乗り越えた壁: 「AI任せにして本当に動くのか」という不信感、エラーが出た時の対処法
- やったこと: Claude Codeへの指示の型を作り、エラー対処フローも確立
- After: コードが一行も書けなくても、複雑な自動化ワークフローを自分で構築・管理できる
- 期間: 約2〜3週間

## 2. ストーリー素材

### 挫折・失敗
- Threadsで「毎日投稿するぞ」と決意するも2週間で枯渇した経験
- Buffer単体で使ってみたが「投稿文を考える時間」は削減できず挫折
- 最初のClaude Codeへの指示が曖昧すぎて、使えない投稿文ばかり生成された

### 怒り・不満
- 「SNS自動化」と言いながらツール費用が月数万円かかる高額サービスへの不満
- 「毎日投稿しなければフォロワーは増えない」というプレッシャーへの反発
- コード書けない人が置いてけぼりになるAI活用情報への怒り

### 感動・転機
- Claude Codeに「Threadsの投稿を30本作って」と指示したら、5分で30本のドラフトが生成された瞬間
- Buffer APIと連携して初めて「自動投稿」が成功した時の感動
- 「これ、コード書けない人でも絶対できる」と確信した瞬間

### 恐怖・不安
- AIが生成した投稿が「無個性になるんじゃないか」という不安
- 自動投稿でアカウント凍結されるリスクへの恐怖
- 技術的なことがわからないまま進めることへの不安

### 誇り・達成感
- 月1回2時間の作業で30日分の投稿を完全自動化できた
- コードが書けない自分でも「エンジニア的な自動化」を実現できた
- 構築したワークフローを仲間に教えたら「これ革命じゃん」と言われた

### 意外な発見・逆転
- AIで生成した投稿の方が、手書きより反応率が高かった（設計次第）
- 「毎日考える」より「月1回まとめて考える」方が投稿の質が上がった
- コードが書けないことが強みになった——「コード知識ゼロの目線」で解説できる

## 7. セルフリサーチ（12項目）

1. **達成した具体的成果**: 月1回2時間の作業で30日分のThreads投稿を自動スケジュール、毎日投稿の完全自動化
2. **最も重要だった行動**: Claude Codeへの指示テンプレートを作り込むこと（プロンプト設計が全体品質を決める）
3. **初心者がやるべき3つのこと**: ①Claude Codeで10投稿を試験生成する ②Bufferアカウント作成＋基本設定 ③Claude Code→Buffer連携のシンプルなフローを1回完走する
4. **多くの人が陥る間違い**: 「まず毎日手書きで投稿してから自動化」と思って先延ばしにする。最初から自動化を組んだ方が早い
5. **自分だけが知っている裏技**: Claude Codeに「自分のキャラクター設定」を最初に読み込ませると、AI感のない投稿が生成される
6. **最も感情が動いた瞬間**: 初めて自動投稿が成功して、寝てる間にThreadsに投稿されていた時
7. **ゼロからやり直すなら**: いきなりフル自動化を目指さず、まず「Claude Codeで10投稿生成→手動でBufferに登録」から始める
8. **読者に守ってほしいこと**: プロンプトに「自分の言葉・体験・価値観」を必ず入れること。AIだけに任せると無個性になる
9. **3ヶ月後の読者の状態**: 毎日「今日の投稿どうしよう」というストレスがなくなり、Threadsの運用が習慣化している
10. **8割解決する核心**: Claude Codeへの「キャラ設定プロンプト」と「投稿テンプレート」の2つを作ること
11. **競合の足りない部分**: 「自動化ツールの紹介」で終わっていて、「実際に動くワークフローの構築手順」がない
12. **「買ってよかった」の瞬間**: 初めて自動投稿が成功した時、または「1ヶ月分の投稿が2時間で終わった」と実感した時
