# Phase 2: コンセプト設計

生成日: 2026-03-25

---

## コアコンセプト（実利的価値）

**「別れた後に迷走した女が、遠回りの経験から気づいた復縁の本音」**

---

## 抽象的コンセプト（世界観）

**「遠回りは、あなたの本気の証明だった」**

失敗したから弱いんじゃない。
占いに頼って、他の男と会って、長文LINEを送って——
それだけ必死に「元カレでいい」と答えを探し続けた、その遠回りこそが
「あの人じゃないとダメ」という本物の気持ちの証明だ。

---

## コンセプト5要素

| 要素 | 内容 |
|------|------|
| **ターゲット** | 別れてから後悔・迷走している20〜30代女性。他の男と関係を持ったり、感情的になって元カレに迷惑をかけてしまった経験がある人 |
| **ゴール** | 「やっぱり元カレがいい」という確信を整理し、復縁に向けた具体的な行動の第一歩を踏み出せる |
| **プロセス** | 迷走体験の正直な棚卸し → 「寂しさ」と「本物の好き」を分ける → 元カレ心理の理解 → やっていいこと・いけないことの整理 → 今日からできる行動計画 |
| **独自性** | 「成功した人の手順書」ではなく「迷走した当事者の失敗と気づき」。テクニックより先に「自分の感情の整理」から始める設計 |
| **ストーリー** | 別れを受け入れられず他の男と関係を持った→空虚感→「やっぱり元カレだ」という確信→感情の整理→行動へのシフト |

---

## 掛け合わせ公式

```
「遠回りした当事者」× 「本音・失敗談ベース」× 「感情整理→行動」
= 「迷走を経て辿り着いた、正直な復縁の地図」
```

---

## ポジショニング

| 軸 | 競合 | 本作 |
|----|------|------|
| 語り口 | 成功者・専門家 | 迷走した当事者 |
| 切り口 | テクニック・手順書 | 感情整理→行動 |
| 姿勢 | 煽り・高圧的 | 正直・等身大 |
| 扱う時期 | 冷却期間後・LINE再開後 | 別れた直後〜迷走期 |
| 内容 | 外側（彼を動かす方法） | 内側（自分を立て直す）→外側 |

## コアメッセージ（繰り返すキーワード）

> **「遠回りは、あなたの本気の証明だった」**

このメッセージをタイトル・冒頭・各章・クロージングで7回以上繰り返す。

---

## 読者の感情フロー設計

1. **共感**: 「この人、私のことわかってる」（失敗談・本音ベースの冒頭）
2. **承認**: 「遠回りしたことを責めなくていい」（ネガティブ体験の再定義）
3. **整理**: 「寂しさと本物の好きは違う」（感情の仕分け）
4. **理解**: 「元カレはこう考えていた」（男性心理の解説）
5. **納得**: 「だから私のやり方は逆効果だったんだ」（失敗の原因理解）
6. **希望**: 「これをすれば、まだチャンスはある」（行動の可能性）
7. **行動**: 「今日からこれをやる」（具体的な第一歩）

## ターゲットペルソナ詳細

**名前**: さくら（27歳 / 会社員）
**状況**: 2年付き合った彼氏と3ヶ月前に別れた。別れた直後は「前に進まなきゃ」と思い、友達に紹介された男性と会い、関係を持った。でも翌朝、布団の中で「全然違う」と思って泣いた。それ以来、元カレのことしか考えられない。
**行動**: 元カレのSNSを毎日チェック。占いに3万円使った。友達に愚痴を言いすぎて「もういい加減にしなよ」と言われた。
**感情**: 自分のことが嫌い。遠回りしたことへの後悔と罪悪感。でも「やっぱり彼しかいない」という確信が消えない。
**検索行動**: 「復縁 可能性」「元カレ 忘れられない」「他の人と付き合えない 元カレ好き」
