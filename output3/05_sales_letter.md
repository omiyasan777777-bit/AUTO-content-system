## なぜ今、Claude Code × Buffer なのか

Claude Codeは、コードを書かずに自然言語（日本語）で指示するだけで動くAIです。

「Threadsの投稿を30本作って」

これだけで、5分後に30本のドラフトが画面に並びます。

Bufferは、SNS投稿のスケジュール管理ツールです。
「3月1日の7時にこの投稿を出す」と設定するだけで、あとは自動で投稿されます。

この2つをつなぐだけで、「毎日投稿の自動化」が完成します。

GASもZapierもMakeも不要。
Vercelも、GitHubも、APIキーも不要。

**Claude Code と Buffer だけ。それだけです。**
