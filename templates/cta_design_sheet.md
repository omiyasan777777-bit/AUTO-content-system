# CTA設計シート

> 「行動の指示 ＋ 得られるもの ＋ 条件 ＋ 緊急性」の4要素で完成するCTA文テンプレート集。
> 読んだ人が「今すぐ動きたい」と思う文章を設計する。

---

## CTA 4要素の定義

| 要素 | 役割 | 問い |
|---|---|---|
| **① 行動の指示** | 何をすればいいか明示する | 「何をしてほしいか？」 |
| **② 得られるもの** | 行動した結果の報酬を示す | 「やると何が手に入るか？」 |
| **③ 条件** | 対象者を絞り込む | 「誰が・どんな状況で動くか？」 |
| **④ 緊急性** | 今すぐ動く理由を作る | 「なぜ今やらないといけないか？」 |

---

## 基本フォーマット

```
【① 行動の指示】今すぐ{具体的なアクション}してください。

【② 得られるもの】{行動した結果}が手に入ります。

【③ 条件】{対象者の状況}の方は特に効果的です。

【④ 緊急性】{今すぐやらない場合のリスク or 今だけの理由}。
```

## CTA文テンプレート集

### パターン A：シンプル直球型（最もスタンダード）

```
今すぐ {具体的なアクション} してください。

{行動した先に広がる世界・得られる結果} を手に入れられます。

{対象者の状況} に当てはまる方は、このタイミングを逃さないでください。
{今しかできない理由 or 放置した場合の損失} 。
```

**作例:**
> 今すぐプロフィールを書き直してください。
> たった1行変えるだけで、フォロワーが「この人だ」と感じる第一印象が作れます。
> 「フォロワーが増えない」と悩んでいる方は、このタイミングを逃さないでください。
> 認知が広がるゴールデンタイムは、ジャンルを絞った最初の3ヶ月です。

---

### パターン B：共感→解決→行動型（信頼構築向け）

```
{読者の悩みや状況を1文で共感}、ということはありませんか？

それは {原因の一言診断} が原因です。

{具体的なアクション} をするだけで、{得られる変化} になります。

{条件：「〜の方は」で絞り込む}、まず {最初の一歩の指示} から始めてください。

{緊急性：期限 or 今やらない理由}。
```

**作例:**
> 「投稿してもスルーされる」ということはありませんか？
> それはフックが弱く、最初の1〜2行で読者が離脱しているのが原因です。
> 冒頭の1文だけ変えるだけで、最後まで読まれる投稿になります。
> 「いいねが0〜2のまま」という方は、まずこのシートのワークAだけやってください。
> 発信のゴールデンタイムは今のフォロワー数帯です。伸びしろが一番大きい今、動いてください。

---

### パターン C：ベネフィット先出し型（欲望に訴える）

```
{得られるもの・理想の未来} が、{条件：〜するだけで / 〜な方なら} 手に入ります。

必要なのは、{具体的なアクション} だけです。

{緊急性：今やる理由 or やらない場合の損失}。

{最後の一言：背中を押す一文}
```

**作例:**
> 「売れるコンテンツのタイトル」が、この公式を使うだけで作れます。
> 必要なのは、今から15分と紙とペンだけです。
> ライバルが気づく前に、あなたのジャンルで先行者優位を取ってください。
> 動いた人だけが、3ヶ月後に「あのとき始めてよかった」と言えます。

---

### パターン D：ストーリー締め型（感情に訴える）

```
{読者が到達できる理想の未来を短いストーリーで描く（2〜3文）}

そのために必要な最初の一歩は、{具体的なアクション} です。

{条件：〜の方は、今日中に〜してください}。

{緊急性を感じさせる一言で締める}
```

**作例:**
> 3ヶ月後、あなたが「この投稿がバズった」とスクショを撮っている姿を想像してください。
> そこへの最初の一歩は、今日この章のワークに答えることです。
> 「自分の言葉で発信できていない」と感じている方は、今日中にワークを書いて投稿してください。
> 行動した1日目が、すべての始まりです。

---

### パターン E：限定・希少性型（緊急性を最大化）

```
これを読んでいる{対象者}だけに伝えます。

{希少性のある理由：今だけ / 〇〇人限定 / このタイミングでしか使えない}。

今すぐ {具体的なアクション} してください。

{行動した結果得られるもの}。

{やらない場合の損失・機会コストを一言で}。
```

**作例:**
> これを読んでいる方だけに伝えます。
> 今のSNSアルゴリズムは「新規参入者」が有利な時期です。参入が遅れるほど埋もれやすくなります。
> 今すぐ「ジャンル×ターゲット×発信軸」の3つを決めてください。
> この3つが決まった瞬間、あなたの発信は「量産コンテンツ」から「誰かのための発信」に変わります。
> 1ヶ月後に「あのとき決めておけば」と後悔しないために、今日動いてください。

## 4要素 設計ワークシート

CTAを一から設計するときに使う。

```
【あなたのCTA設計シート】

■ ① 行動の指示
読者に「何を」してほしいか？
→ 動詞で始める / 具体的に / ハードルを低く

例：プロフィールを書き直す / ワークの穴埋めをする / スクリーンショットを撮る / コメントに投稿する

あなたのCTA:「今すぐ {　　　　} してください」

---

■ ② 得られるもの
行動した結果、読者は何を手に入れるか？
→ 感情・状態の変化 / 具体的な成果 / 数字があれば尚よい

例：3ヶ月でフォロワーが3倍 / 「この人から買いたい」と思われる / 投稿に自信が持てる

あなたの設計:「{　　　　} が手に入ります」

---

■ ③ 条件
誰に向けたCTAか？どんな状況の人が動くか？
→ 「〜な方は」「〜で悩んでいる方は」で絞る

例：フォロワーが100人以下の方 / 投稿しても反応がない方 / 副業を始めたばかりの方

あなたの設計:「{　　　　} の方は」

---

■ ④ 緊急性
なぜ今すぐやらないといけないか？
→ 機会コスト / 期限 / 放置の損失 / 競合に先を越される

例：ゴールデンタイムは今だけ / 毎日遅れるほど差が広がる / 来月には競合が増える

あなたの設計:「{　　　　} だから、今すぐ動いてください」

---

■ 完成したCTA文（4要素を組み合わせて）

今すぐ {①行動} してください。
{②得られるもの} が手に入ります。
{③条件} の方は、このタイミングを逃さないでください。
{④緊急性} 。
```

## 用途別 CTA早見表

| 使う場面 | おすすめパターン | 重視する要素 |
|---|---|---|
| note 有料部分の締め | パターン D（ストーリー） | ②得られるもの・④緊急性 |
| セールスレターの購入促進 | パターン E（限定・希少性） | ④緊急性・①行動指示 |
| 各章末ワーク誘導 | パターン A（シンプル直球） | ①行動指示・③条件 |
| SNS投稿の締め | パターン B（共感→解決） | ①行動指示・②得られるもの |
| フォロー・LINE登録促進 | パターン C（ベネフィット先出し） | ②得られるもの・④緊急性 |

## CTA作成ルール

1. **動詞で始める** ── 「今すぐ〜してください」が最強。迷わせない
2. **ハードルを下げる** ── 「5分でいい」「1つだけやる」「見るだけでOK」
3. **結果を具体的に** ── 「成長できます」より「3ヶ月でフォロワーが3倍になります」
4. **緊急性は嘘をつかない** ── 根拠のない「今だけ！」はNG。必ず理由を添える
5. **長くても4文まで** ── CTAが長いと読まれない。4文以内に収める
6. **1CTAに1アクション** ── 「フォローして、いいねして、コメントして」は分散してNG
