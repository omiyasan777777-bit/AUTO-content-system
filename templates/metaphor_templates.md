# 比喩テンプレート集

有料コンテンツ・SNS投稿・セールスレターで使える比喩の作り方テンプレートです。
抽象的なノウハウを日常の失敗・体験に結びつけることで、読者の「わかった！」を最速で引き出します。

ジャンル不問で使えます。`[ ]` の部分を自分のテーマに置き換えるだけで完成します。

---

## 基本公式

```
[現状・概念] は [日常の失敗・体験] と同じだ。
なぜなら [共通する理由] からだ。
```

---

## テンプレート一覧

### 公式①「日常の失敗型」── 最も使いやすい基本型

```
[ノウハウ・状況] は、[日常で誰もが経験する失敗] と同じです。

[日常の失敗] のとき、あなたはどうしますか？
{失敗の場面を1〜2文で具体的に描写}

[ノウハウ・状況] でも、まったく同じことが起きています。
{共通する構造を1〜2文で説明}

だから [解決策・正しい行動] が必要なんです。
```

**作例:**
> フォロワーが増えないのは、**居酒屋でいきなり名刺を渡す**のと同じです。
> まだ顔も知らない相手に「これ持ってって」と渡しても怖いですよね。
> SNSも同じで、「まず存在を認知させること」が先なんです。

---

### 公式②「料理・食べ物型」── 共感度が高く万人に刺さる

```
[概念] は、[料理の工程 or 食べ物の状態] に似ています。

[料理の具体的な場面を描写]
{なぜそうなるかを1文で}

発信（or ビジネス）でも、まったく同じことが言えます。
{共通点を1〜2文で説明}
```

**作例:**
> コンテンツに「軸がない」状態は、**味付けのバラバラな料理を毎日出すお店**と同じです。
> 今日はカレー屋、明日はラーメン屋、来週はフレンチ——これじゃリピーターはつきませんよね。
> 発信も同じで、「この人といえば○○」という一貫性が信頼をつくります。

---

### 公式③「乗り物・旅行型」── プロセス・ステップの説明に最適

```
[現在地] から [ゴール] まで行くのに、[間違った手段] で行こうとしているようなものです。

{乗り物・経路の具体例を1〜2文で}

最終目的地は同じでも、手段を間違えると [問題・損失] が起きます。
[正しい手段] を選べば、[ゴール] までの最短ルートになります。
```

**作例:**
> 「とにかくフォロワーを買う」のは、**東京から大阪に行くのに北に向かう**ようなものです。
> 移動はしているのに、目的地から離れていく。
> エンゲージメントが死んだフォロワーを増やしても、売上には繋がりません。

---

### 公式④「スポーツ・練習型」── 努力・継続・スキルの話に最適

```
[行動 or 習慣] は、[スポーツの練習メニュー] と同じです。

[スポーツの場面を具体的に1〜2文で描写]
{なぜその練習が必要かを1文で}

発信（or ビジネス）でも同じで、
[行動 or 習慣] を続けることで [成果] が手に入ります。
```

**作例:**
> 毎日投稿は、**素振り千本**と同じです。
> バッターボックスに入る前に、素振りで体に染み込ませる。
> 一球ごとの打席（バズ狙い）より、日々の素振り（毎日投稿）の方が結果的に打率が上がります。

---

### 公式⑤「医療・健康型」── 問題放置の恐怖を伝えるのに最適

```
[問題を放置した状態] は、[体の不調を無視し続ける状態] と同じです。

{体の不調の具体例を1〜2文で}
最初は軽い違和感でも、放置すれば [深刻な結果] になりますよね。

[ビジネス上の問題] も同じで、早めに [解決策] を取らないと
[最悪のシナリオ] になりかねません。
```

**作例:**
> 「なんとなく投稿がバズらない」状態を放置するのは、**胃痛を市販薬でごまかし続ける**のと同じです。
> 表面の痛みは消えても、原因が残っていれば悪化する一方。
> コンセプト設計という根本治療をしない限り、また同じ「バズらない」に戻ります。

---

### 公式⑥「子育て・育成型」── 長期的な取り組み・育てる感覚を伝えるのに最適

```
[プロセス] は、[子育ての一場面] に似ています。

{子育ての具体的な場面を1〜2文で}
{なぜそれが必要かを1文で}

[ビジネス・発信] でも同じで、
[プロセス] をすっ飛ばして [結果だけ求めること] はできません。
```

**作例:**
> ファン化は、**子どもの成長**と同じです。
> 生まれた翌日に「もう走れ」と言う親はいませんよね。
> フォロワーも、認知 → 共感 → 信頼 → 購買 という順番でしか育ちません。

---

### 公式⑦「天気・自然型」── 不確実性・タイミングの話に最適

```
[状況] は、[天気・自然現象] のようなものです。

{天気・自然の具体例を1〜2文で}

[ビジネス・発信] の世界でも、
[状況] に抵抗するより [状況に合わせた行動] をとる方が賢いやり方です。
```

**作例:**
> アルゴリズムの変動は、**天気**のようなものです。
> 雨が降っても太陽を呼び戻すことはできません。
> でも傘を持っていれば、雨の日でも前に進めます。
> どんなアルゴリズムでも生き残れる「コアファン」を育てることが傘になります。

## 比喩を作る手順（実践用）

```
STEP 1: 伝えたい「抽象的な概念・ノウハウ」を1文で書く
→ 例：「コンセプトがないと発信がブレる」

STEP 2: その概念と「構造が同じ日常の失敗」を探す
→ 問い：「これと同じ構造の、日常のあるある失敗は何？」
→ 例：「メニューが毎日変わる飲食店はリピーターがつかない」

STEP 3: 公式に当てはめる
→「コンセプトがないのは、メニューが毎日変わる飲食店と同じだ」

STEP 4: 読者の共感ポイントを描写する（20〜50字）
→「今日はパスタ、明日は焼肉、来週はタイ料理——これじゃ「また行きたい」とはならない」

STEP 5: ビジネスの話に戻す接続文を入れる
→「発信でも同じで、軸が一貫していない人にはファンがつきません」
```

## 日常ネタ素材リスト

比喩に使いやすい「日常の失敗・体験」のストック。

| カテゴリ | ネタ例 |
|---|---|
| 食事 | 冷めたラーメン / 塩を入れ忘れた料理 / 消費期限切れ / 分量を間違えたレシピ |
| 乗り物 | 反対方向の電車に乗る / ガス欠 / カーナビの誤案内 / 自転車のパンク |
| 体・健康 | 虫歯を放置する / 睡眠不足で仕事する / 筋肉痛なのに追い込む |
| 勉強 | 答えを先に見る / テスト前夜だけ詰め込む / ノートをきれいにするだけ |
| 家事 | 洗濯物を取り込み忘れる / 鍋を火にかけたまま忘れる / 段取りを間違える |
| スポーツ | 素振りをせず試合に出る / ルールを知らずにゲームする / ウォームアップなしで全力 |
| 人間関係 | 初対面でいきなり営業する / 話を聞かずに自分だけ話す / 謝らずに言い訳する |
| 買い物 | セール品を衝動買いする / 必要なものをリストなしで買いに行く |

## 使用上の注意

- **日常感が命**: 読者が「あるある！」と思える場面を選ぶ。マニアックすぎるネタはNG
- **1比喩=1メッセージ**: 1つの比喩で複数のことを伝えようとしない
- **長くなりすぎない**: 比喩の描写は3〜5文まで。長すぎると本題が霞む
- **落とし方を忘れない**: 必ず「だから○○が必要」という本題への接続で終わる
